# Ant Design Solid

Ant Design-inspired components for SolidJS.

## Stack

- SolidJS
- Vite 8
- pnpm 11
- TypeScript
- Token-driven CSS-in-JS

## Development

```bash
pnpm install
pnpm dev
pnpm build
pnpm test
pnpm typecheck
pnpm lint
```

## Packages

- `@ant-design-solid/core`
- `@ant-design-solid/theme`
- `@ant-design-solid/cssinjs`
- `@ant-design-solid/icons`

The docs app is kept private in this monorepo as `@ant-design-solid/docs`.

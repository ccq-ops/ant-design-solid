# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

By participating, you are expected to uphold a respectful and constructive environment for contributors and users.

Unacceptable behavior includes harassment, discriminatory language or imagery, personal attacks, sustained disruption, and publishing private information without permission.

Project maintainers may remove, edit, or reject contributions and comments that do not align with this standard.

Report conduct concerns through the repository maintainers or GitHub's reporting tools.

The full Contributor Covenant text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

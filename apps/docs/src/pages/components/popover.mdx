# Popover

Popover displays rich floating content around a trigger element.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Button, Popover, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Popover title="Title" content="Helpful content">
      <Button>Hover me</Button>
    </Popover>
  )
}

export default Demo1
```

### Trigger

```tsx
import { createSignal } from 'solid-js'
import { Button, Popover, Space } from '@ant-design-solid/core'

const Demo2 = function () {
  return (
    <Space wrap>
      <Popover trigger="click" title="Click" content="Clicked content">
        <Button>Click</Button>
      </Popover>
      <Popover trigger="focus" title="Focus" content="Focused content">
        <Button>Focus</Button>
      </Popover>
    </Space>
  )
}

export default Demo2
```

### Controlled

```tsx
import { createSignal } from 'solid-js'
import { Button, Popover, Space } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Space>
      <Popover open={open()} onOpenChange={setOpen} content="Controlled content">
        <Button>Controlled</Button>
      </Popover>
      <Button onClick={() => setOpen((next) => !next)}>{open() ? 'Close' : 'Open'}</Button>
    </Space>
  )
}

export default Demo4
```

## API

| Property            | Description                                           | Type                                         | Default   |
| ------------------- | ----------------------------------------------------- | -------------------------------------------- | --------- |
| `title`             | Popover title content.                                | `JSX.Element`                                | `-`       |
| `content`           | Popover body content.                                 | `JSX.Element`                                | `-`       |
| `placement`         | Overlay placement relative to the trigger.            | `PopoverPlacement`                           | `-`       |
| `trigger`           | Interaction that opens the overlay.                   | `'hover' \| 'click' \| 'focus'`              | `'hover'` |
| `open`              | Controlled overlay open state.                        | `boolean`                                    | `-`       |
| `defaultOpen`       | Initial overlay open state for uncontrolled usage.    | `boolean`                                    | `false`   |
| `onOpenChange`      | Called when overlay open state changes.               | `(open: boolean) => void`                    | `-`       |
| `mouseEnterDelay`   | Delay before opening on hover.                        | `number`                                     | `-`       |
| `mouseLeaveDelay`   | Delay before closing after hover leaves.              | `number`                                     | `-`       |
| `overlayClass`      | Additional class for the overlay.                     | `string`                                     | `-`       |
| `overlayStyle`      | Inline style for the overlay.                         | `JSX.CSSProperties`                          | `-`       |
| `zIndex`            | Overrides overlay z-index.                            | `number`                                     | `-`       |
| `getPopupContainer` | Returns the element used to mount the overlay portal. | `(triggerNode?: HTMLElement) => HTMLElement` | `-`       |

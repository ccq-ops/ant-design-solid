# ConfigProvider

### Theme override

```tsx
import { Button, ConfigProvider } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <ConfigProvider theme={{ token: { colorPrimary: '#722ed1' } }}>
      <Button type="primary">Custom Theme</Button>
    </ConfigProvider>
  )
}

export default Demo1
```

### Component size

```tsx
import { Button, ConfigProvider } from '@ant-design-solid/core'

const Demo2 = function () {
  return (
    <ConfigProvider componentSize="large">
      <Button>Large by config</Button>
    </ConfigProvider>
  )
}

export default Demo2
```

## API

| Property            | Description                                         | Type                                         | Default    |
| ------------------- | --------------------------------------------------- | -------------------------------------------- | ---------- |
| `prefixCls`         | Global CSS class prefix for components.             | `string`                                     | `'ads'`    |
| `componentSize`     | Default component size.                             | `'small' \| 'middle' \| 'large'`             | `'middle'` |
| `direction`         | Text and layout direction.                          | `'ltr' \| 'rtl'`                             | `'ltr'`    |
| `theme`             | Theme token configuration.                          | `ThemeConfig`                                | `-`        |
| `getPopupContainer` | Global popup portal container resolver.             | `(triggerNode?: HTMLElement) => HTMLElement` | `-`        |
| `children`          | Application or subtree receiving the configuration. | `JSX.Element`                                | `-`        |

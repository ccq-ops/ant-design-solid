# List

List displays a group of related items with optional headers, actions, and metadata.

### Data source

```tsx
import { Avatar, Button, List, Space } from '@ant-design-solid/core'

const tasks = ['Create component API', 'Write tests', 'Document examples']

const Demo1 = function () {
  return (
    <List
      dataSource={tasks}
      renderItem={(item, index) => <List.Item>{`${index + 1}. ${item}`}</List.Item>}
    />
  )
}

export default Demo1
```

### Bordered with header and footer

```tsx
import { Avatar, Button, List, Space } from '@ant-design-solid/core'

const tasks = ['Create component API', 'Write tests', 'Document examples']

const Demo2 = function () {
  return (
    <List
      bordered
      header={<strong>Project checklist</strong>}
      footer="3 items"
      dataSource={tasks}
      renderItem={(item) => <List.Item>{item}</List.Item>}
    />
  )
}

export default Demo2
```

### Meta and actions

```tsx
import { Avatar, Button, List, Space } from '@ant-design-solid/core'

const users = [
  {
    name: 'Ada Lovelace',
    description: 'Mathematician and first computer programmer.',
    avatar: 'A',
  },
  {
    name: 'Grace Hopper',
    description: 'Computer scientist and compiler pioneer.',
    avatar: 'G',
  },
]

const Demo3 = function () {
  return (
    <List
      dataSource={users}
      renderItem={(user) => (
        <List.Item
          actions={[
            <a href={`#edit-${user.avatar}`}>Edit</a>,
            <a href={`#view-${user.avatar}`}>View</a>,
          ]}
          extra={<Button size="small">Follow</Button>}
        >
          <List.Item.Meta
            avatar={<Avatar>{user.avatar}</Avatar>}
            title={<a href={`#${user.avatar}`}>{user.name}</a>}
            description={user.description}
          />
        </List.Item>
      )}
    />
  )
}

export default Demo3
```

### Loading and empty

```tsx
import { Avatar, Button, List, Space } from '@ant-design-solid/core'

const tasks = ['Create component API', 'Write tests', 'Document examples']

const Demo4 = function () {
  return (
    <Space direction="vertical" class="w-full">
      <List
        loading
        dataSource={tasks}
        renderItem={(item) => <List.Item>{item}</List.Item>}
        bordered
      />
      <List dataSource={[]} emptyText="Nothing found" bordered />
    </Space>
  )
}

export default Demo4
```

## API

### List

| Property     | Description                                             | Type                                      | Default     |
| ------------ | ------------------------------------------------------- | ----------------------------------------- | ----------- |
| `dataSource` | Array of data items rendered by renderItem.             | `T[]`                                     | `-`         |
| `renderItem` | Render function for each data item.                     | `(item: T, index: number) => JSX.Element` | `-`         |
| `header`     | Header content rendered above the list items.           | `JSX.Element`                             | `-`         |
| `footer`     | Footer content rendered below the list items.           | `JSX.Element`                             | `-`         |
| `bordered`   | Whether to render the list border.                      | `boolean`                                 | `false`     |
| `split`      | Whether to render separators between items.             | `boolean`                                 | `true`      |
| `size`       | List size.                                              | `'default' \| 'large' \| 'small'`         | `'default'` |
| `loading`    | Shows loading placeholder instead of items.             | `boolean`                                 | `false`     |
| `emptyText`  | Empty state content when there are no items.            | `JSX.Element`                             | `'No Data'` |
| `children`   | Manual list items used when dataSource is not provided. | `JSX.Element`                             | `-`         |

### List.Item

| Property   | Description                                    | Type            | Default |
| ---------- | ---------------------------------------------- | --------------- | ------- |
| `actions`  | Action items rendered after the main content.  | `JSX.Element[]` | `-`     |
| `extra`    | Extra content rendered at the end of the item. | `JSX.Element`   | `-`     |
| `children` | Main list item content.                        | `JSX.Element`   | `-`     |

### List.Item.Meta

| Property      | Description                              | Type          | Default |
| ------------- | ---------------------------------------- | ------------- | ------- |
| `avatar`      | Avatar content rendered before metadata. | `JSX.Element` | `-`     |
| `title`       | Metadata title.                          | `JSX.Element` | `-`     |
| `description` | Metadata description.                    | `JSX.Element` | `-`     |
| `children`    | Additional metadata content.             | `JSX.Element` | `-`     |

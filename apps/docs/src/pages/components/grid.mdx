# Grid

Grid uses rows and columns to build layouts on a 24-column system.

### 24 columns

```tsx
import { Col, Row } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Row>
      <Col span={12}>
        <div class="bg-blue-50 p-4">span 12</div>
      </Col>
      <Col span={12}>
        <div class="bg-blue-200 p-4">span 12</div>
      </Col>
    </Row>
  )
}

export default Demo1
```

### Gutter

```tsx
import { Col, Row } from '@ant-design-solid/core'

const blockStyle = 'rounded bg-blue-50 p-4 text-center text-blue-700'

const strongBlockStyle = 'rounded bg-blue-200 p-4 text-center text-blue-900'

const Demo2 = function () {
  return (
    <Row gutter={[16, 24]}>
      <Col span={8}>
        <div class={blockStyle}>A</div>
      </Col>
      <Col span={8}>
        <div class={strongBlockStyle}>B</div>
      </Col>
      <Col span={8}>
        <div class={blockStyle}>C</div>
      </Col>
      <Col span={8}>
        <div class={strongBlockStyle}>D</div>
      </Col>
    </Row>
  )
}

export default Demo2
```

### Offset and order

```tsx
import { Col, Row } from '@ant-design-solid/core'

const blockStyle = 'rounded bg-blue-50 p-4 text-center text-blue-700'

const strongBlockStyle = 'rounded bg-blue-200 p-4 text-center text-blue-900'

const Demo3 = function () {
  return (
    <Row>
      <Col span={6} order={2}>
        <div class={blockStyle}>order 2</div>
      </Col>
      <Col span={6} offset={6} order={1}>
        <div class={strongBlockStyle}>offset 6 / order 1</div>
      </Col>
    </Row>
  )
}

export default Demo3
```

### Flex alignment

```tsx
import { Col, Row } from '@ant-design-solid/core'

const blockStyle = 'rounded bg-blue-50 p-4 text-center text-blue-700'

const Demo4 = function () {
  return (
    <Row justify="space-between" align="middle">
      <Col span={6}>
        <div class={blockStyle}>Short</div>
      </Col>
      <Col span={6}>
        <div class="rounded bg-blue-200 p-8 text-center text-blue-900">Tall</div>
      </Col>
      <Col span={6}>
        <div class={blockStyle}>Middle aligned</div>
      </Col>
    </Row>
  )
}

export default Demo4
```

## API

### Row

| Property  | Description                                                         | Type                                                                                  | Default |
| --------- | ------------------------------------------------------------------- | ------------------------------------------------------------------------------------- | ------- |
| `gutter`  | Horizontal gutter or horizontal and vertical gutter pair in pixels. | `number \| [number, number]`                                                          | `0`     |
| `align`   | Vertical alignment of columns inside the row.                       | `'top' \| 'middle' \| 'bottom' \| 'stretch'`                                          | `-`     |
| `justify` | Horizontal distribution of columns inside the row.                  | `'start' \| 'end' \| 'center' \| 'space-around' \| 'space-between' \| 'space-evenly'` | `-`     |
| `wrap`    | Allows columns to wrap when there is not enough horizontal space.   | `boolean`                                                                             | `true`  |

### Col

| Property | Description                                                       | Type     | Default |
| -------- | ----------------------------------------------------------------- | -------- | ------- |
| `span`   | Column width as a number of units in the 24-column grid.          | `number` | `-`     |
| `offset` | Number of grid units to move the column to the right with margin. | `number` | `-`     |
| `order`  | CSS flex order for the column.                                    | `number` | `-`     |
| `push`   | Moves the column right by the specified grid units.               | `number` | `-`     |
| `pull`   | Moves the column left by the specified grid units.                | `number` | `-`     |

# Transfer

Move items between source and target lists.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Transfer } from '@ant-design-solid/core'

const dataSource = [
  { key: '1', title: 'Alpha', description: 'First item' },
  { key: '2', title: 'Beta', description: 'Second item' },
  { key: '3', title: 'Gamma', description: 'Disabled item', disabled: true },
]

const Demo1 = function () {
  return <Transfer dataSource={dataSource} />
}

export default Demo1
```

### Controlled

```tsx
import { createSignal } from 'solid-js'
import { Transfer } from '@ant-design-solid/core'

const dataSource = [
  { key: '1', title: 'Alpha', description: 'First item' },
  { key: '2', title: 'Beta', description: 'Second item' },
  { key: '3', title: 'Gamma', description: 'Disabled item', disabled: true },
]

const Demo4 = function () {
  const [targetKeys, setTargetKeys] = createSignal(['2'])

  return (
    <Transfer
      dataSource={dataSource}
      targetKeys={targetKeys()}
      onChange={(next) => setTargetKeys(next)}
    />
  )
}

export default Demo4
```

## API

### Transfer

| Property              | Description                                             | Type                                                  | Default |
| --------------------- | ------------------------------------------------------- | ----------------------------------------------------- | ------- |
| `dataSource`          | Items that can be moved between lists.                  | `TransferItem[]`                                      | `-`     |
| `targetKeys`          | Controlled keys currently in the target list.           | `string[]`                                            | `-`     |
| `defaultTargetKeys`   | Initial keys in the target list for uncontrolled usage. | `string[]`                                            | `-`     |
| `selectedKeys`        | Controlled selected item keys across both lists.        | `string[]`                                            | `-`     |
| `defaultSelectedKeys` | Initial selected item keys for uncontrolled usage.      | `string[]`                                            | `-`     |
| `disabled`            | Disables selection and transfer operations.             | `boolean`                                             | `false` |
| `showSearch`          | Shows search inputs for filtering both lists.           | `boolean`                                             | `false` |
| `titles`              | Source and target list titles.                          | `[JSX.Element, JSX.Element]`                          | `-`     |
| `operations`          | Labels for move-to-target and move-to-source buttons.   | `[JSX.Element, JSX.Element]`                          | `-`     |
| `filterOption`        | Custom item filter used by search.                      | `(inputValue: string, item: TransferItem) => boolean` | `-`     |
| `prefixCls`           | Custom CSS class prefix.                                | `string`                                              | `-`     |
| `onChange`            | Called after items move between lists.                  | `(targetKeys, direction, moveKeys) => void`           | `-`     |
| `onSelectChange`      | Called when source or target selected keys change.      | `(sourceSelectedKeys, targetSelectedKeys) => void`    | `-`     |

### TransferItem

| Property      | Description                              | Type          | Default |
| ------------- | ---------------------------------------- | ------------- | ------- |
| `key`         | Stable item key.                         | `string`      | `-`     |
| `title`       | Item title content.                      | `JSX.Element` | `-`     |
| `description` | Optional item description.               | `JSX.Element` | `-`     |
| `disabled`    | Disables selecting and moving this item. | `boolean`     | `-`     |

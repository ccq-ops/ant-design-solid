# Statistic

Statistic displays a formatted numerical value with optional title and adornments.

### Basic

```tsx
import { Card, Space, Statistic } from '@ant-design-solid/core'

const Demo1 = function () {
  return <Statistic title="Active Users" value={112893} />
}

export default Demo1
```

### Prefix, suffix and precision

```tsx
import { Card, Space, Statistic } from '@ant-design-solid/core'

const Demo2 = function () {
  return (
    <Statistic title="Account Balance" value={1128.693} precision={2} prefix="$" suffix="USD" />
  )
}

export default Demo2
```

### Loading

```tsx
import { Card, Space, Statistic } from '@ant-design-solid/core'

const Demo3 = function () {
  return <Statistic title="Revenue" value={5680} loading />
}

export default Demo3
```

### In card

```tsx
import { Card, Space, Statistic } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Space wrap align="start">
      <Card class="w-60">
        <Statistic title="Feedback" value={93} suffix="%" valueStyle={{ color: '#3f8600' }} />
      </Card>
      <Card class="w-60">
        <Statistic title="Views" value={8846} prefix="👁" />
      </Card>
    </Space>
  )
}

export default Demo4
```

## API

| Property     | Description                                                    | Type                          | Default |
| ------------ | -------------------------------------------------------------- | ----------------------------- | ------- |
| `title`      | Statistic title.                                               | `JSX.Element`                 | `-`     |
| `value`      | Value content. Numeric values can be formatted with precision. | `string \| number \| null`    | `-`     |
| `precision`  | Decimal precision for numeric values.                          | `number`                      | `-`     |
| `prefix`     | Content rendered before the value.                             | `JSX.Element`                 | `-`     |
| `suffix`     | Content rendered after the value.                              | `JSX.Element`                 | `-`     |
| `loading`    | Shows a loading placeholder instead of the value.              | `boolean`                     | `false` |
| `valueStyle` | Inline style applied to the value element.                     | `JSX.CSSProperties \| string` | `-`     |

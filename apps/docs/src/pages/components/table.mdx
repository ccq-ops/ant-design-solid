# Table

### Basic

```tsx
import { Badge, Table, Tag } from '@ant-design-solid/core'

const data = [
  { key: '1', name: 'Ada Lovelace', age: 32, status: 'active' },
  { key: '2', name: 'Grace Hopper', age: 36, status: 'idle' },
]

const columns = [
  { title: 'Name', dataIndex: 'name' },
  { title: 'Age', dataIndex: 'age', align: 'right' },
  {
    title: 'Status',
    dataIndex: 'status',
    render: (value) =>
      value === 'active' ? (
        <Tag color="success">Active</Tag>
      ) : (
        <Badge status="default" text="Idle" />
      ),
  },
]

const Demo1 = function () {
  return <Table columns={columns} dataSource={data} />
}

export default Demo1
```

## API

### Table

| Property     | Description                               | Type                                                                    | Default                 |
| ------------ | ----------------------------------------- | ----------------------------------------------------------------------- | ----------------------- |
| `columns`    | Table column definitions.                 | `TableColumn<T>[]`                                                      | `-`                     |
| `dataSource` | Data records rendered as rows.            | `T[]`                                                                   | `-`                     |
| `rowKey`     | Record field or function used as row key. | `keyof T \| ((record: T, index: number) => string \| number)`           | `record.key \|\| index` |
| `loading`    | Shows the loading indicator.              | `boolean`                                                               | `false`                 |
| `emptyText`  | Empty state content.                      | `JSX.Element`                                                           | `'No data'`             |
| `size`       | Table size.                               | `'small' \| 'middle' \| 'large'`                                        | `'middle'`              |
| `bordered`   | Whether to render bordered table styling. | `boolean`                                                               | `false`                 |
| `showHeader` | Whether to render the header row.         | `boolean`                                                               | `true`                  |
| `onRow`      | Returns extra props for each table row.   | `(record: T, index: number) => JSX.HTMLAttributes<HTMLTableRowElement>` | `-`                     |

### TableColumn

| Property    | Description                        | Type                                                        | Default |
| ----------- | ---------------------------------- | ----------------------------------------------------------- | ------- |
| `title`     | Column header content.             | `JSX.Element`                                               | `-`     |
| `dataIndex` | Record field read for this column. | `keyof T \| string`                                         | `-`     |
| `key`       | Column key.                        | `string`                                                    | `-`     |
| `render`    | Custom cell renderer.              | `(value: unknown, record: T, index: number) => JSX.Element` | `-`     |
| `width`     | Column width.                      | `number \| string`                                          | `-`     |
| `align`     | Cell text alignment.               | `'left' \| 'center' \| 'right'`                             | `-`     |
| `class`     | Additional cell class.             | `string`                                                    | `-`     |
| `classList` | Conditional cell classes.          | `Record<string, boolean \| undefined>`                      | `-`     |

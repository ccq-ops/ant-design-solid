# Drawer

Drawer renders a slide-out panel for contextual tasks without leaving the page. It supports antd-style placement, sizing, semantic styling, nested push behavior, loading, custom containers, focus handling, and resizing.

### Basic

```tsx
import { Button, Drawer, Space } from '@ant-design-solid/core'
import { For, createSignal } from 'solid-js'

const Demo1 = function () {
  return (
<Button type="primary" onClick={() => setBasicOpen(true)}>
          Open drawer
        </Button>
        <Drawer open={basicOpen()} title="Basic drawer" onClose={() => setBasicOpen(false)}>
          Content displayed inside the drawer body.
        </Drawer>
  )
}

export default Demo1
```

### Placement

```tsx
import { Button, Drawer, Space } from '@ant-design-solid/core'
import { For, createSignal } from 'solid-js'

const Demo2 = function () {
  return (
<Space>
          <For each={placements}>
            {(item) => (
              <Button
                onClick={() => {
                  setPlacement(item)
                  setPlacementOpen(true)
                }}
              >
                {item}
              </Button>
            )}
          </For>
        </Space>
        <Drawer
          open={placementOpen()}
          title={placement() + ' drawer'}
          placement={placement()}
          onClose={() => setPlacementOpen(false)}
        >
          Placement controls which side of the viewport the drawer is anchored to.
        </Drawer>
  )
}

export default Demo2
```

### Nested drawer push

```tsx
import { Button, Drawer, Space } from '@ant-design-solid/core'
import { For, createSignal } from 'solid-js'

const Demo11 = function () {
  return (
<Button type="primary" onClick={() => setParentOpen(true)}>
          Open parent drawer
        </Button>
        <Drawer
          open={parentOpen()}
          title="Parent drawer"
          push={{ distance: 120 }}
          onClose={() => {
            setParentOpen(false)
            setChildOpen(false)
          }}
        >
          <p>Open a nested drawer to push this parent drawer.</p>
          <Button type="primary" onClick={() => setChildOpen(true)}>
            Open child drawer
          </Button>
          <Drawer open={childOpen()} title="Child drawer" onClose={() => setChildOpen(false)}>
            Opening the child pushes the parent drawer by the configured distance.
          </Drawer>
        </Drawer>
  )
}

export default Demo11
```

## API

| Property          | Description                                                                                          | Type                                                                                                                       | Default             |
| ----------------- | ---------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | ------------------- |
| `open`            | Controls whether the drawer is visible.                                                              | `boolean`                                                                                                                  | `false`             |
| `title`           | Drawer title content.                                                                                | `JSX.Element`                                                                                                              | `-`                 |
| `placement`       | Side of the viewport the drawer slides from.                                                         | `'left' \| 'right' \| 'top' \| 'bottom'`                                                                                   | `'right'`           |
| `size`            | Preset or custom drawer size. Takes precedence over width and height when provided.                  | `'default' \| 'large' \| number \| string`                                                                                 | `'default'`         |
| `width`           | Drawer width for left and right placements. Kept for antd compatibility; prefer size for new code.   | `number \| string`                                                                                                         | `378`               |
| `height`          | Drawer height for top and bottom placements. Kept for antd compatibility; prefer size for new code.  | `number \| string`                                                                                                         | `378`               |
| `closable`        | Shows the close button. Object form customizes icon, disabled state, and start/end position.         | `boolean \| { closeIcon?: JSX.Element; disabled?: boolean; placement?: 'start' \| 'end' }`                                 | `true`              |
| `mask`            | Shows the background mask. Object form configures whether the mask is enabled, blurred, or closable. | `boolean \| { enabled?: boolean; blur?: boolean; closable?: boolean }`                                                     | `true`              |
| `maskClosable`    | Allows clicking the mask to close the drawer. Prefer mask.closable in new code.                      | `boolean`                                                                                                                  | `true`              |
| `keyboard`        | Allows closing with Escape.                                                                          | `boolean`                                                                                                                  | `true`              |
| `destroyOnClose`  | Unmounts children after close. Prefer destroyOnHidden in new code.                                   | `boolean`                                                                                                                  | `false`             |
| `destroyOnHidden` | Unmounts children after the drawer is hidden.                                                        | `boolean`                                                                                                                  | `false`             |
| `forceRender`     | Pre-renders Drawer content even when closed.                                                         | `boolean`                                                                                                                  | `false`             |
| `extra`           | Extra header action content.                                                                         | `JSX.Element`                                                                                                              | `-`                 |
| `footer`          | Footer content.                                                                                      | `JSX.Element`                                                                                                              | `-`                 |
| `zIndex`          | Overrides drawer z-index.                                                                            | `number`                                                                                                                   | `-`                 |
| `getContainer`    | Portal mount container. Pass false to render inline instead of portaling to document.body.           | `HTMLElement \| () => HTMLElement \| string \| false`                                                                      | `document.body`     |
| `rootClassName`   | Class name for the root element that contains the mask and drawer panel.                             | `string`                                                                                                                   | `-`                 |
| `rootStyle`       | Inline style for the root element that contains the mask and drawer panel.                           | `JSX.CSSProperties`                                                                                                        | `-`                 |
| `classNames`      | Semantic class names for mask, wrapper, content, header, body, footer, and section.                  | `Partial<Record<DrawerSemanticDOM, string>> \| (info) => Partial<Record<DrawerSemanticDOM, string>>`                       | `-`                 |
| `styles`          | Semantic inline styles for mask, wrapper, content, header, body, footer, and section.                | `Partial<Record<DrawerSemanticDOM, JSX.CSSProperties>> \| (info) => Partial<Record<DrawerSemanticDOM, JSX.CSSProperties>>` | `-`                 |
| `loading`         | Shows a skeleton loading state in the drawer body instead of children.                               | `boolean`                                                                                                                  | `false`             |
| `push`            | Pushes the parent drawer when a nested drawer opens. Pass an object to customize distance.           | `boolean \| { distance: number \| string }`                                                                                | `{ distance: 180 }` |
| `drawerRender`    | Custom render wrapper for the drawer panel node.                                                     | `(node: JSX.Element) => JSX.Element`                                                                                       | `-`                 |
| `focusable`       | Focus trap and return-focus configuration.                                                           | `{ trap?: boolean; focusTriggerAfterClose?: boolean }`                                                                     | `-`                 |
| `resizable`       | Enables drag resizing and optional resize lifecycle callbacks.                                       | `boolean \| { onResizeStart?: () => void; onResize?: (size: number) => void; onResizeEnd?: () => void }`                   | `-`                 |
| `maxSize`         | Maximum width or height while resizing.                                                              | `number`                                                                                                                   | `-`                 |
| `onClose`         | Called when the drawer requests close from the close button, mask, or keyboard.                      | `(event?: MouseEvent \| KeyboardEvent) => void`                                                                            | `-`                 |
| `afterOpenChange` | Called after open state changes.                                                                     | `(open: boolean) => void`                                                                                                  | `-`                 |
| `children`        | Drawer body content.                                                                                 | `JSX.Element`                                                                                                              | `-`                 |
| `class`           | Solid class for the drawer panel.                                                                    | `string`                                                                                                                   | `-`                 |
| `className`       | React-style alias for the drawer panel class.                                                        | `string`                                                                                                                   | `-`                 |
| `classList`       | Solid classList for conditional drawer panel classes.                                                | `Record<string, boolean \| undefined>`                                                                                     | `-`                 |
| `style`           | Inline style for the drawer panel.                                                                   | `JSX.CSSProperties`                                                                                                        | `-`                 |
| `aria-label`      | Accessible label for the dialog.                                                                     | `string`                                                                                                                   | `-`                 |
| `aria-labelledby` | ID of an external element that labels the dialog.                                                    | `string`                                                                                                                   | `-`                 |

## Semantic DOM

Use these keys with `classNames` and `styles` to customize stable semantic sections of the drawer.

| Property  | Description                                                        | Type                | Default |
| --------- | ------------------------------------------------------------------ | ------------------- | ------- |
| `mask`    | Background mask element.                                           | `DrawerSemanticDOM` | `-`     |
| `wrapper` | Drawer panel dialog element.                                       | `DrawerSemanticDOM` | `-`     |
| `content` | Content container inside the panel.                                | `DrawerSemanticDOM` | `-`     |
| `header`  | Header area that contains title, extra, and close.                 | `DrawerSemanticDOM` | `-`     |
| `body`    | Scrollable body content area.                                      | `DrawerSemanticDOM` | `-`     |
| `footer`  | Footer action area.                                                | `DrawerSemanticDOM` | `-`     |
| `section` | Alias applied to the drawer panel for antd-style semantic styling. | `DrawerSemanticDOM` | `-`     |

## ClosableConfig

| Property    | Description                                             | Type               | Default |
| ----------- | ------------------------------------------------------- | ------------------ | ------- |
| `closeIcon` | Custom close icon content.                              | `JSX.Element`      | `-`     |
| `disabled`  | Disables the close button.                              | `boolean`          | `false` |
| `placement` | Places the close button before or after the title area. | `'start' \| 'end'` | `'end'` |

## MaskConfig

| Property   | Description                               | Type      | Default |
| ---------- | ----------------------------------------- | --------- | ------- |
| `enabled`  | Whether to render the mask.               | `boolean` | `true`  |
| `blur`     | Adds backdrop blur to the mask.           | `boolean` | `false` |
| `closable` | Whether clicking the mask requests close. | `boolean` | `true`  |

## FocusableConfig

| Property                 | Description                                                  | Type      | Default |
| ------------------------ | ------------------------------------------------------------ | --------- | ------- |
| `trap`                   | Traps Tab focus inside the drawer.                           | `boolean` | `true`  |
| `focusTriggerAfterClose` | Returns focus to the previously focused trigger after close. | `boolean` | `false` |

## ResizableConfig

| Property        | Description                                  | Type                     | Default |
| --------------- | -------------------------------------------- | ------------------------ | ------- |
| `onResizeStart` | Called when dragging starts.                 | `() => void`             | `-`     |
| `onResize`      | Called while dragging with the current size. | `(size: number) => void` | `-`     |
| `onResizeEnd`   | Called when dragging ends.                   | `() => void`             | `-`     |

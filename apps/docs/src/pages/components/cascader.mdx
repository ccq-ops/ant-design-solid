# Cascader

Select from a nested set of related options with one menu column per level.

### Basic

```tsx
import { Form, Cascader, Space } from '@ant-design-solid/core'

const options = [
  {
    label: 'Zhejiang',
    value: 'zhejiang',
    children: [
      {
        label: 'Hangzhou',
        value: 'hangzhou',
        children: [{ label: 'West Lake', value: 'west-lake' }],
      },
      { label: 'Ningbo', value: 'ningbo' },
    ],
  },
  {
    label: 'Jiangsu',
    value: 'jiangsu',
    children: [{ label: 'Nanjing', value: 'nanjing' }],
  },
]

const Demo1 = function () {
  return <Cascader placeholder="Select area" options={options} />
}

export default Demo1
```

## API

### Cascader

| Property               | Description                                                                          | Type                                                                             | Default                |
| ---------------------- | ------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------- | ---------------------- |
| `options`              | Nested option tree rendered by the cascader.                                         | `CascaderOption[]`                                                               | `-`                    |
| `value`                | Controlled selected path values.                                                     | `OptionValue[] \| OptionValue[][]`                                               | `-`                    |
| `defaultValue`         | Initial selected path values for uncontrolled usage.                                 | `OptionValue[] \| OptionValue[][]`                                               | `-`                    |
| `open`                 | Controlled popup open state.                                                         | `boolean`                                                                        | `-`                    |
| `defaultOpen`          | Initial popup open state.                                                            | `boolean`                                                                        | `false`                |
| `placeholder`          | Placeholder shown when no path is selected.                                          | `JSX.Element`                                                                    | `-`                    |
| `disabled`             | Disables opening and selection.                                                      | `boolean`                                                                        | `false`                |
| `allowClear`           | Shows a clear button when a value is selected.                                       | `boolean`                                                                        | `false`                |
| `changeOnSelect`       | Allows selecting intermediate parent options.                                        | `boolean`                                                                        | `false`                |
| `expandTrigger`        | Interaction used to expand child columns.                                            | `'click' \| 'hover'`                                                             | `'click'`              |
| `displayRender`        | Customizes selected path display.                                                    | `(labels: JSX.Element[], selectedOptions: CascaderOption[]) => JSX.Element`      | `-`                    |
| `prefixCls`            | Custom CSS class prefix.                                                             | `string`                                                                         | `-`                    |
| `zIndex`               | Overrides popup z-index.                                                             | `number`                                                                         | `-`                    |
| `getPopupContainer`    | Returns the element used to mount the popup portal.                                  | `(triggerNode?: HTMLElement) => HTMLElement`                                     | `-`                    |
| `showSearch`           | Enables path search or configures filtering, sorting, rendering, and search control. | `boolean \| CascaderShowSearch`                                                  | `false`                |
| `searchValue`          | Controlled search text.                                                              | `string`                                                                         | `-`                    |
| `onSearch`             | Called when search input changes.                                                    | `(search: string) => void`                                                       | `-`                    |
| `loadData`             | Loads child options lazily for options with isLeaf set to false.                     | `(selectedOptions: CascaderOption[]) => void \| Promise<void>`                   | `-`                    |
| `loadingIcon`          | Custom loading indicator for lazy options.                                           | `JSX.Element`                                                                    | `-`                    |
| `multiple`             | Enables multiple path selection.                                                     | `boolean`                                                                        | `false`                |
| `showCheckedStrategy`  | Controls whether parent or child paths are shown in multiple mode.                   | `Cascader.SHOW_PARENT \| Cascader.SHOW_CHILD`                                    | `Cascader.SHOW_PARENT` |
| `tagRender`            | Custom multiple tag renderer.                                                        | `(label: JSX.Element, onClose: () => void, value: OptionValue[]) => JSX.Element` | `-`                    |
| `removeIcon`           | Custom remove icon for multiple tags.                                                | `JSX.Element`                                                                    | `-`                    |
| `maxTagCount`          | Maximum number of visible tags.                                                      | `number \| 'responsive'`                                                         | `-`                    |
| `maxTagPlaceholder`    | Placeholder rendered for omitted tags.                                               | `JSX.Element \| ((omittedValues: CascaderSelectedPath[]) => JSX.Element)`        | `-`                    |
| `maxTagTextLength`     | Maximum tag label text length.                                                       | `number`                                                                         | `-`                    |
| `autoClearSearchValue` | Clears search text after selecting in multiple search mode.                          | `boolean`                                                                        | `true`                 |
| `size`                 | Selector size.                                                                       | `'large' \| 'middle' \| 'small'`                                                 | `'middle'`             |
| `status`               | Validation status style.                                                             | `'error' \| 'warning'`                                                           | `-`                    |
| `variant`              | Selector visual variant.                                                             | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`                         | `'outlined'`           |
| `prefix`               | Custom prefix content inside the selector.                                           | `JSX.Element`                                                                    | `-`                    |
| `onChange`             | Called with selected values and selected option path.                                | `(value: OptionValue[], selectedOptions: CascaderOption[]) => void`              | `-`                    |
| `onOpenChange`         | Called when popup open state changes.                                                | `(open: boolean) => void`                                                        | `-`                    |

### CascaderOption

| Property   | Description                                                               | Type                          | Default |
| ---------- | ------------------------------------------------------------------------- | ----------------------------- | ------- |
| `label`    | Option display label.                                                     | `JSX.Element`                 | `-`     |
| `value`    | Option value.                                                             | `string \| number \| boolean` | `-`     |
| `disabled` | Disables this option.                                                     | `boolean`                     | `-`     |
| `children` | Child options rendered in the next column.                                | `CascaderOption[]`            | `-`     |
| `isLeaf`   | Marks a lazy option as leaf or non-leaf when children are not loaded yet. | `boolean`                     | `-`     |
| `loading`  | Marks this option as loading.                                             | `boolean`                     | `-`     |

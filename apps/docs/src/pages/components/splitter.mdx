# Splitter

Splitter separates content into resizable panels.

### Basic

```tsx
import { Splitter } from '@ant-design-solid/core'

const panelStyle = {
  display: 'flex',
  'align-items': 'center',
  'justify-content': 'center',
  height: '100%',
  background: 'var(--docs-surface-subtle)',
}

const Demo1 = function () {
  return (
    <Splitter style={{ height: '160px' }}>
      <Splitter.Panel defaultSize="40%" style={panelStyle}>
        Left
      </Splitter.Panel>
      <Splitter.Panel style={panelStyle}>Right</Splitter.Panel>
    </Splitter>
  )
}

export default Demo1
```

### Min and max

```tsx
import { Splitter } from '@ant-design-solid/core'

const panelStyle = {
  display: 'flex',
  'align-items': 'center',
  'justify-content': 'center',
  height: '100%',
  background: 'var(--docs-surface-subtle)',
}

const Demo3 = function () {
  return (
    <Splitter style={{ height: '160px' }}>
      <Splitter.Panel defaultSize={160} min={100} max={240} style={panelStyle}>
        100px - 240px
      </Splitter.Panel>
      <Splitter.Panel style={panelStyle}>Flexible</Splitter.Panel>
    </Splitter>
  )
}

export default Demo3
```

## API

### Splitter

| Property        | Description                               | Type                              | Default        |
| --------------- | ----------------------------------------- | --------------------------------- | -------------- |
| `layout`        | Panel layout direction.                   | `'horizontal' \| 'vertical'`      | `'horizontal'` |
| `onResizeStart` | Called when dragging a resize bar starts. | `(sizes: SplitterSize[]) => void` | `-`            |
| `onResize`      | Called while panel sizes change.          | `(sizes: SplitterSize[]) => void` | `-`            |
| `onResizeEnd`   | Called when dragging a resize bar ends.   | `(sizes: SplitterSize[]) => void` | `-`            |

### Splitter.Panel

| Property      | Description                                         | Type                            | Default |
| ------------- | --------------------------------------------------- | ------------------------------- | ------- |
| `size`        | Controlled panel size.                              | `number \| string \| undefined` | `-`     |
| `defaultSize` | Initial panel size for uncontrolled usage.          | `number \| string \| undefined` | `-`     |
| `min`         | Minimum panel size.                                 | `number \| string \| undefined` | `-`     |
| `max`         | Maximum panel size.                                 | `number \| string \| undefined` | `-`     |
| `resizable`   | Whether adjacent resize bars can resize this panel. | `boolean`                       | `true`  |

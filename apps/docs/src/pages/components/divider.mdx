# Divider

### Basic

```tsx
import { Divider, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <div>
      Text
      <Divider />
      Text
    </div>
  )
}

export default Demo1
```

### With text

```tsx
import { Divider, Space } from '@ant-design-solid/core'

const Demo2 = function () {
  return (
    <div>
      <Divider orientation="left">Left</Divider>
      <Divider>Center</Divider>
      <Divider orientation="right">Right</Divider>
    </div>
  )
}

export default Demo2
```

### Variants

```tsx
import { Divider, Space } from '@ant-design-solid/core'

const Demo3 = function () {
  return (
    <div>
      <Divider dashed />
      <Divider plain>Plain Text</Divider>
    </div>
  )
}

export default Demo3
```

### Vertical

```tsx
import { Divider, Space } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Space>
      <span>A</span>
      <Divider type="vertical" />
      <span>B</span>
      <Divider type="vertical" dashed />
      <span>C</span>
    </Space>
  )
}

export default Demo4
```

## API

| Property      | Description                                             | Type                            | Default        |
| ------------- | ------------------------------------------------------- | ------------------------------- | -------------- |
| `type`        | Divider direction.                                      | `'horizontal' \| 'vertical'`    | `'horizontal'` |
| `orientation` | Text orientation for horizontal dividers with children. | `'left' \| 'center' \| 'right'` | `'center'`     |
| `dashed`      | Uses a dashed divider line.                             | `boolean`                       | `false`        |
| `plain`       | Uses plain text styling for divider content.            | `boolean`                       | `false`        |
| `children`    | Text or content rendered inside a horizontal divider.   | `JSX.Element`                   | `-`            |

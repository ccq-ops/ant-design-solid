# Skeleton

Skeleton displays placeholder blocks while content is loading.

### Basic

```tsx
import { Card, Skeleton, Space } from '@ant-design-solid/core'
import { createSignal } from 'solid-js'

const Demo1 = function () {
  return <Skeleton />
}

export default Demo1
```

### Custom rows

```tsx
import { Card, Skeleton, Space } from '@ant-design-solid/core'
import { createSignal } from 'solid-js'

const Demo4 = function () {
  return (
    <Skeleton
      title={{ width: '50%' }}
      paragraph={{ rows: 4, width: ['100%', '90%', '70%', '40%'] }}
    />
  )
}

export default Demo4
```

### Loaded content

```tsx
import { Card, Skeleton, Space } from '@ant-design-solid/core'
import { createSignal } from 'solid-js'

const Demo5 = function () {
  const [loaded, setLoaded] = createSignal(false)

  return (
    <Space direction="vertical" class="w-full">
      <button type="button" onClick={() => setLoaded((value) => !value)}>
        Toggle loaded
      </button>
      <Skeleton loading={!loaded()} active>
        <Card title="Loaded">Content is ready.</Card>
      </Skeleton>
    </Space>
  )
}

export default Demo5
```

## API

### Skeleton

| Property    | Description                                         | Type                                | Default |
| ----------- | --------------------------------------------------- | ----------------------------------- | ------- |
| `active`    | Shows animated loading effect.                      | `boolean`                           | `false` |
| `loading`   | Controls whether skeleton or children are rendered. | `boolean`                           | `true`  |
| `avatar`    | Shows avatar placeholder or configures it.          | `boolean \| SkeletonAvatarProps`    | `-`     |
| `title`     | Shows title placeholder or configures it.           | `boolean \| SkeletonTitleProps`     | `-`     |
| `paragraph` | Shows paragraph placeholders or configures them.    | `boolean \| SkeletonParagraphProps` | `-`     |
| `round`     | Uses rounded placeholder corners.                   | `boolean`                           | `false` |
| `children`  | Content rendered when loading is false.             | `JSX.Element`                       | `-`     |

### Skeleton avatar

| Property | Description               | Type                                        | Default    |
| -------- | ------------------------- | ------------------------------------------- | ---------- |
| `size`   | Avatar placeholder size.  | `'small' \| 'default' \| 'large' \| number` | `-`        |
| `shape`  | Avatar placeholder shape. | `'circle' \| 'square'`                      | `'circle'` |

### Skeleton title

| Property | Description              | Type               | Default |
| -------- | ------------------------ | ------------------ | ------- |
| `width`  | Title placeholder width. | `number \| string` | `-`     |

### Skeleton paragraph

| Property | Description                    | Type                               | Default |
| -------- | ------------------------------ | ---------------------------------- | ------- |
| `rows`   | Number of paragraph rows.      | `number`                           | `-`     |
| `width`  | Paragraph row width or widths. | `SkeletonWidth \| SkeletonWidth[]` | `-`     |

# BorderBeam

Add an animated beam around the border of a single HTML element.

### Basic

```tsx
import { BorderBeam, Card, Space } from '@ant-design-solid/core'

const panelStyle = {
  position: 'relative',
  width: '300px',
  padding: '24px',
  border: '1px solid var(--docs-border)',
  'border-radius': '12px',
  background: 'var(--docs-surface-solid)',
  'box-shadow': 'var(--docs-card-shadow)',
}

const Demo1 = function () {
  return (
    <BorderBeam>
      <div style={panelStyle}>
        <h3>AI Assistant</h3>
        <p>BorderBeam decorates the host element without adding a visible wrapper.</p>
      </div>
    </BorderBeam>
  )
}

export default Demo1
```

### Gradient and outset

```tsx
import { BorderBeam, Card, Space } from '@ant-design-solid/core'

const panelStyle = {
  position: 'relative',
  width: '300px',
  padding: '24px',
  border: '1px solid var(--docs-border)',
  'border-radius': '12px',
  background: 'var(--docs-surface-solid)',
  'box-shadow': 'var(--docs-card-shadow)',
}

const Demo3 = function () {
  return (
    <BorderBeam
      outset={4}
      color={[
        { color: '#1677ff', percent: 0 },
        { color: '#36cfc9', percent: 55 },
        { color: '#95de64', percent: 100 },
      ]}
    >
      <div style={panelStyle}>
        <h3>Gradient beam</h3>
        <p>Use gradient stops and an outset to draw the beam outside the element border.</p>
      </div>
    </BorderBeam>
  )
}

export default Demo3
```

### With Card

```tsx
import { BorderBeam, Card, Space } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Space wrap>
      <BorderBeam color="#722ed1" outset="2px">
        <Card title="Insight" style={{ width: '300px', position: 'relative' }}>
          BorderBeam can decorate existing components when the child resolves to an HTML element.
        </Card>
      </BorderBeam>
    </Space>
  )
}

export default Demo4
```

## API

The child must resolve to a single `HTMLElement`; text and SVG children are rendered without the beam.

| Property    | Description                                                | Type                                   | Default |
| ----------- | ---------------------------------------------------------- | -------------------------------------- | ------- |
| `prefixCls` | Custom CSS class prefix.                                   | `string`                               | `-`     |
| `class`     | Additional class applied to the generated beam element.    | `string`                               | `-`     |
| `classList` | Conditional classes applied to the generated beam element. | `Record<string, boolean \| undefined>` | `-`     |
| `style`     | Inline style merged into the generated beam element.       | `JSX.CSSProperties \| string`          | `-`     |
| `children`  | Single host element to decorate.                           | `JSX.Element`                          | `-`     |
| `color`     | CSS color string or gradient stop array for the beam.      | `BorderBeamColor`                      | `-`     |
| `outset`    | Distance the beam extends outside the host border.         | `number \| string`                     | `-`     |

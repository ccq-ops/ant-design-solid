# Carousel

Display a group of content in a carousel with dots, arrows, and autoplay.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Button, Carousel, Space } from '@ant-design-solid/core'

function Slide(props) {
  return <div style={slideStyle}>{props.children}</div>
}

const Demo1 = function () {
  return (
    <Carousel>
      <Slide>1</Slide>
      <Slide>2</Slide>
      <Slide>3</Slide>
    </Carousel>
  )
}

export default Demo1
```

## API

### Carousel

| Property        | Description                                  | Type                                          | Default     |
| --------------- | -------------------------------------------- | --------------------------------------------- | ----------- |
| `autoplay`      | Automatically advances slides.               | `boolean`                                     | `false`     |
| `autoplaySpeed` | Autoplay interval in milliseconds.           | `number`                                      | `3000`      |
| `dots`          | Whether to show dots or dot configuration.   | `boolean \| { className?: string }`           | `true`      |
| `dotPosition`   | Dot navigation position.                     | `'top' \| 'bottom' \| 'left' \| 'right'`      | `'bottom'`  |
| `arrows`        | Whether to show previous and next arrows.    | `boolean`                                     | `false`     |
| `effect`        | Transition effect.                           | `'scrollx' \| 'fade'`                         | `'scrollx'` |
| `fade`          | Alias that enables fade effect.              | `boolean`                                     | `false`     |
| `easing`        | CSS transition easing.                       | `string`                                      | `'ease'`    |
| `speed`         | Transition duration in milliseconds.         | `number`                                      | `500`       |
| `infinite`      | Whether navigation wraps around at the ends. | `boolean`                                     | `true`      |
| `initialSlide`  | Initial active slide index.                  | `number`                                      | `0`         |
| `beforeChange`  | Called before active slide changes.          | `(current: number, next: number) => void`     | `-`         |
| `afterChange`   | Called after active slide changes.           | `(current: number) => void`                   | `-`         |
| `prefixCls`     | Custom CSS class prefix.                     | `string`                                      | `-`         |
| `ref`           | Receives imperative carousel methods.        | `CarouselRef \| ((ref: CarouselRef) => void)` | `-`         |

### CarouselRef

| Property | Description                        | Type                                             | Default |
| -------- | ---------------------------------- | ------------------------------------------------ | ------- |
| `goTo`   | Switches to the given slide index. | `(slide: number, dontAnimate?: boolean) => void` | `-`     |
| `next`   | Switches to the next slide.        | `() => void`                                     | `-`     |
| `prev`   | Switches to the previous slide.    | `() => void`                                     | `-`     |

# Switch

Switch toggles a boolean state on or off.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Form, Space, Switch } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Space>
      <Switch checkedChildren="On" unCheckedChildren="Off" />
      <Switch defaultChecked checkedChildren="On" unCheckedChildren="Off" />
    </Space>
  )
}

export default Demo1
```

### Controlled

## API

| Property            | Description                                                | Type                                            | Default    |
| ------------------- | ---------------------------------------------------------- | ----------------------------------------------- | ---------- |
| `checked`           | Controlled checked state.                                  | `boolean`                                       | `-`        |
| `defaultChecked`    | Initial checked state for uncontrolled usage.              | `boolean`                                       | `false`    |
| `disabled`          | Disables user interaction.                                 | `boolean`                                       | `false`    |
| `loading`           | Displays loading state and disables interaction.           | `boolean`                                       | `false`    |
| `size`              | Switch size.                                               | `'small' \| 'middle'`                           | `'middle'` |
| `checkedChildren`   | Content displayed when checked.                            | `JSX.Element`                                   | `-`        |
| `unCheckedChildren` | Content displayed when unchecked.                          | `JSX.Element`                                   | `-`        |
| `prefixCls`         | Custom CSS class prefix.                                   | `string`                                        | `-`        |
| `onChange`          | Called with the next checked state after user interaction. | `(checked: boolean, event: MouseEvent) => void` | `-`        |

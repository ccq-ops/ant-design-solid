# Select

Select lets users choose one value from a dropdown list.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Form, Select, Space } from '@ant-design-solid/core'

const options = [
  { label: 'Apple', value: 'apple' },
  { label: 'Orange', value: 'orange' },
  { label: 'Pear', value: 'pear' },
]

const Demo1 = function () {
  return <Select class="w-60" placeholder="Select fruit" options={options} />
}

export default Demo1
```

### Controlled

```tsx
import { createSignal } from 'solid-js'
import { Form, Select, Space } from '@ant-design-solid/core'

const options = [
  { label: 'Apple', value: 'apple' },
  { label: 'Orange', value: 'orange' },
  { label: 'Pear', value: 'pear' },
]

const Demo2 = function () {
  const [value, setValue] = createSignal('orange')

  return (
    <Space direction="vertical">
      <Select class="w-60" value={value()} onChange={(next) => setValue(next)} options={options} />
      <span>Selected: {String(value() ?? 'none')}</span>
    </Space>
  )
}

export default Demo2
```

## API

| Property                | Description                                                                                              | Type                                                                        | Default                |
| ----------------------- | -------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- | ---------------------- |
| `value`                 | Controlled selected value. Arrays are used by `multiple` and `tags` modes.                               | `RawValue \| RawValue[] \| LabeledValue \| LabeledValue[]`                  | `-`                    |
| `defaultValue`          | Initial selected value for uncontrolled usage.                                                           | `RawValue \| RawValue[] \| LabeledValue \| LabeledValue[]`                  | `-`                    |
| `mode`                  | Select mode.                                                                                             | `'multiple' \| 'tags'`                                                      | `-`                    |
| `labelInValue`          | Emits `{ value, label, key }` objects instead of raw values.                                             | `boolean`                                                                   | `false`                |
| `open`                  | Controlled dropdown open state.                                                                          | `boolean`                                                                   | `-`                    |
| `defaultOpen`           | Initial dropdown open state.                                                                             | `boolean`                                                                   | `false`                |
| `options`               | Select options. Primitive options are normalized to label/value pairs and grouped options are supported. | `SelectOptionInput[]`                                                       | `-`                    |
| `fieldNames`            | Custom option field names.                                                                               | `{ label?: string; value?: string; options?: string; groupLabel?: string }` | `-`                    |
| `placeholder`           | Placeholder displayed when no value is selected.                                                         | `JSX.Element`                                                               | `-`                    |
| `disabled`              | Disables opening and value changes.                                                                      | `boolean`                                                                   | `false`                |
| `allowClear`            | Shows a clear button and can customize its icon.                                                         | `boolean \| { clearIcon?: JSX.Element }`                                    | `false`                |
| `showSearch`            | Enables search, or configures search behavior.                                                           | `boolean \| SelectSearchConfig`                                             | `multiple/tags: true`  |
| `searchValue`           | Controlled search text.                                                                                  | `string`                                                                    | `-`                    |
| `filterOption`          | Filters options by search text.                                                                          | `boolean \| (inputValue, option) => boolean`                                | `true`                 |
| `filterSort`            | Sorts filtered options.                                                                                  | `(optionA, optionB, info) => number`                                        | `-`                    |
| `optionFilterProp`      | Option field or fields used for default filtering.                                                       | `string \| string[]`                                                        | `label` with `options` |
| `optionLabelProp`       | Option field used as selected display label.                                                             | `string`                                                                    | `label`                |
| `optionRender`          | Custom dropdown option renderer.                                                                         | `(option, info) => JSX.Element`                                             | `-`                    |
| `labelRender`           | Custom selected label renderer.                                                                          | `(props: LabeledValue) => JSX.Element`                                      | `-`                    |
| `tagRender`             | Custom multiple/tag item renderer.                                                                       | `(props: SelectTagRenderProps) => JSX.Element`                              | `-`                    |
| `popupRender`           | Custom popup renderer.                                                                                   | `(originNode) => JSX.Element`                                               | `-`                    |
| `notFoundContent`       | Content shown when no option matches.                                                                    | `JSX.Element`                                                               | `'Not Found'`          |
| `placement`             | Popup placement.                                                                                         | `'bottomLeft' \| 'bottomRight' \| 'topLeft' \| 'topRight'`                  | `'bottomLeft'`         |
| `popupMatchSelectWidth` | Matches popup width to selector, or sets explicit width.                                                 | `boolean \| number`                                                         | `true`                 |
| `size`                  | Selector size.                                                                                           | `'large' \| 'middle' \| 'small'`                                            | `middle`               |
| `status`                | Validation status style.                                                                                 | `'error' \| 'warning'`                                                      | `-`                    |
| `variant`               | Selector visual variant.                                                                                 | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`                    | `'outlined'`           |
| `prefix`                | Custom prefix content.                                                                                   | `JSX.Element`                                                               | `-`                    |
| `suffixIcon`            | Custom suffix icon. Set to `null` to hide.                                                               | `JSX.Element \| null`                                                       | `DownOutlined`         |
| `loading`               | Shows loading state.                                                                                     | `boolean`                                                                   | `false`                |
| `maxCount`              | Maximum number of selected values.                                                                       | `number`                                                                    | `-`                    |
| `maxTagCount`           | Maximum visible selected tags.                                                                           | `number \| 'responsive'`                                                    | `-`                    |
| `maxTagPlaceholder`     | Placeholder for omitted selected tags.                                                                   | `JSX.Element \| (omittedValues) => JSX.Element`                             | `+ N ...`              |
| `maxTagTextLength`      | Maximum displayed tag text length.                                                                       | `number`                                                                    | `-`                    |
| `tokenSeparators`       | Separators that create tags in `tags` mode.                                                              | `string[]`                                                                  | `-`                    |
| `classNames`            | Semantic class names.                                                                                    | `Record<string, string>`                                                    | `-`                    |
| `styles`                | Semantic inline styles.                                                                                  | `Record<string, JSX.CSSProperties>`                                         | `-`                    |
| `zIndex`                | Overrides dropdown z-index.                                                                              | `number`                                                                    | `-`                    |
| `getPopupContainer`     | Returns the element used to mount the dropdown portal.                                                   | `(triggerNode?: HTMLElement) => HTMLElement`                                | `-`                    |
| `onChange`              | Called when selected value changes.                                                                      | `(value, option) => void`                                                   | `-`                    |
| `onSelect`              | Called when an option is selected.                                                                       | `(value, option) => void`                                                   | `-`                    |
| `onDeselect`            | Called when an option is deselected.                                                                     | `(value, option) => void`                                                   | `-`                    |
| `onClear`               | Called when clear button is clicked.                                                                     | `() => void`                                                                | `-`                    |
| `onSearch`              | Called when search text changes.                                                                         | `(value: string) => void`                                                   | `-`                    |
| `onOpenChange`          | Called when dropdown open state changes.                                                                 | `(open: boolean) => void`                                                   | `-`                    |
| `onPopupScroll`         | Called when dropdown scrolls.                                                                            | `(event) => void`                                                           | `-`                    |

Deprecated antd aliases such as `dropdownRender`, `dropdownClassName`, `dropdownStyle`, `dropdownMatchSelectWidth`, `bordered`, and `onDropdownVisibleChange` are accepted for compatibility.

### Search config

`showSearch` can be `true` or an object with the same search options. Top-level search props override values from the object form.

| Property               | Description                                       | Type                                                               | Default                |
| ---------------------- | ------------------------------------------------- | ------------------------------------------------------------------ | ---------------------- |
| `autoClearSearchValue` | Clears search text after selecting.               | `boolean`                                                          | `true`                 |
| `filterOption`         | Filters options by search text.                   | `boolean \| (inputValue: string, option: SelectOption) => boolean` | `true`                 |
| `filterSort`           | Sorts search results.                             | `(optionA, optionB, info: { searchValue: string }) => number`      | `-`                    |
| `optionFilterProp`     | Option field or fields used by default filtering. | `string \| string[]`                                               | `label` with `options` |
| `searchValue`          | Controlled search text.                           | `string`                                                           | `-`                    |
| `onSearch`             | Called when search text changes.                  | `(value: string) => void`                                          | `-`                    |
| `searchIcon`           | Search icon for compatibility.                    | `JSX.Element`                                                      | `-`                    |

### Option

`options` is the recommended data API. `Select.Option` and `Select.OptGroup` are also available for JSX usage.

| Property              | Description                     | Type                          | Default |
| --------------------- | ------------------------------- | ----------------------------- | ------- |
| `label`               | Display label.                  | `JSX.Element`                 | `value` |
| `value`               | Option value.                   | `string \| number \| boolean` | `-`     |
| `disabled`            | Disables selecting this option. | `boolean`                     | `false` |
| `class` / `className` | Extra option class.             | `string`                      | `-`     |
| `style`               | Option inline style.            | `JSX.CSSProperties`           | `-`     |
| `title`               | Option title attribute.         | `string`                      | `-`     |

### OptGroup

| Property              | Description                             | Type                  | Default |
| --------------------- | --------------------------------------- | --------------------- | ------- |
| `label`               | Group label.                            | `JSX.Element`         | `-`     |
| `options`             | Group children when using the data API. | `SelectOptionInput[]` | `-`     |
| `class` / `className` | Extra group class.                      | `string`              | `-`     |
| `title`               | Group title attribute.                  | `string`              | `-`     |

### Ref

| Name      | Description           |
| --------- | --------------------- |
| `focus()` | Focuses the selector. |
| `blur()`  | Blurs the selector.   |

### Semantic keys

`classNames` and `styles` accept the semantic keys currently used by the component: `root`, `selector`, `item`, `popup`, and `popup.root`.

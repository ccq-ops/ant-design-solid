# Result

Show operation results and follow-up actions.

### Success

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Result
      status="success"
      title="Successfully submitted"
      subTitle="Your request has been saved."
    />
  )
}

export default Demo1
```

### Error

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo2 = function () {
  return <Result status="error" title="Submission failed" subTitle="Please check and try again." />
}

export default Demo2
```

### 404

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo3 = function () {
  return <Result status="404" title="404" subTitle="Page not found" />
}

export default Demo3
```

### Extra actions

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Result
      status="info"
      title="Ready"
      extra={
        <Space>
          <Button type="primary">Continue</Button>
          <Button>Back</Button>
        </Space>
      }
    />
  )
}

export default Demo4
```

### Custom icon

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo5 = function () {
  return <Result icon="🚀" title="Launched" />
}

export default Demo5
```

## API

| Property    | Description                                     | Type                                                                     | Default  |
| ----------- | ----------------------------------------------- | ------------------------------------------------------------------------ | -------- |
| `status`    | Preset result status and built-in icon style.   | `'success' \| 'error' \| 'info' \| 'warning' \| '404' \| '403' \| '500'` | `'info'` |
| `title`     | Main result title.                              | `JSX.Element`                                                            | `-`      |
| `subTitle`  | Secondary result description.                   | `JSX.Element`                                                            | `-`      |
| `icon`      | Custom icon content.                            | `JSX.Element`                                                            | `-`      |
| `extra`     | Action area rendered below the subtitle.        | `JSX.Element`                                                            | `-`      |
| `children`  | Additional content rendered in the result body. | `JSX.Element`                                                            | `-`      |
| `prefixCls` | Custom CSS class prefix.                        | `string`                                                                 | `-`      |

# Result

Show operation results and follow-up actions.

### Success

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Result
      status="success"
      title="Successfully submitted"
      subTitle="Your request has been saved."
    />
  )
}

export default Demo1
```

### Extra actions

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Result
      status="info"
      title="Ready"
      extra={
        <Space>
          <Button type="primary">Continue</Button>
          <Button>Back</Button>
        </Space>
      }
    />
  )
}

export default Demo4
```

### Semantic styles

```tsx
import { Button, Result, Space } from '@ant-design-solid/core'

const Demo6 = function () {
  return (
    <Result
      status="success"
      title="Styled result"
      subTitle="Customize each semantic slot with classNames and styles."
      extra={<Button type="primary">Done</Button>}
      class="custom-result"
      classNames={{
        icon: 'custom-result-icon',
        title: 'custom-result-title',
        subTitle: 'custom-result-subtitle',
        extra: 'custom-result-extra',
        body: 'custom-result-body',
      }}
      styles={{
        root: { padding: '32px 16px' },
        title: { color: '#1677ff' },
        body: { textAlign: 'center' },
      }}
    >
      Fine tune the root, icon, title, subTitle, extra, and body slots.
    </Result>
  )
}

export default Demo6
```

## API

| Property     | Description                                     | Type                                                                                                    | Default  |
| ------------ | ----------------------------------------------- | ------------------------------------------------------------------------------------------------------- | -------- |
| `status`     | Preset result status and built-in icon style.   | `'success' \| 'error' \| 'info' \| 'warning' \| 403 \| 404 \| 500 \| '403' \| '404' \| '500'`           | `'info'` |
| `title`      | Main result title.                              | `JSX.Element`                                                                                           | `-`      |
| `subTitle`   | Secondary result description.                   | `JSX.Element`                                                                                           | `-`      |
| `icon`       | Custom icon content.                            | `JSX.Element`                                                                                           | `-`      |
| `extra`      | Action area rendered below the subtitle.        | `JSX.Element`                                                                                           | `-`      |
| `children`   | Additional content rendered in the result body. | `JSX.Element`                                                                                           | `-`      |
| `class`      | Custom root class.                              | `string`                                                                                                | `-`      |
| `classNames` | Customize class for semantic slots.             | `Record<SemanticDOM, string> \| (info: { props }) => Record<SemanticDOM, string>`                       | `-`      |
| `styles`     | Customize inline style for semantic slots.      | `Record<SemanticDOM, JSX.CSSProperties> \| (info: { props }) => Record<SemanticDOM, JSX.CSSProperties>` | `-`      |
| `prefixCls`  | Custom CSS class prefix.                        | `string`                                                                                                | `-`      |

## Semantic DOM

| Name       | Description                   |
| ---------- | ----------------------------- |
| `root`     | Root result container.        |
| `icon`     | Built-in or custom icon area. |
| `title`    | Title wrapper.                |
| `subTitle` | Subtitle wrapper.             |
| `extra`    | Action area wrapper.          |
| `body`     | Additional content wrapper.   |

## Static Members

| Property                     | Description                   |
| ---------------------------- | ----------------------------- |
| `Result.PRESENTED_IMAGE_403` | Built-in 403 exception image. |
| `Result.PRESENTED_IMAGE_404` | Built-in 404 exception image. |
| `Result.PRESENTED_IMAGE_500` | Built-in 500 exception image. |

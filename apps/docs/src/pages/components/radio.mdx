# Radio

Radio lets users choose one option from a mutually exclusive set.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Form, Radio, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Space>
      <Radio defaultChecked>A</Radio>
      <Radio>B</Radio>
    </Space>
  )
}

export default Demo1
```

### Group

```tsx
import { createSignal } from 'solid-js'
import { Form, Radio, Space } from '@ant-design-solid/core'

const options = [
  { label: 'A', value: 'a' },
  { label: 'B', value: 'b' },
  { label: 'Disabled', value: 'disabled', disabled: true },
]

const Demo2 = function () {
  return (
    <Space direction="vertical">
      <Radio.Group options={options} defaultValue="a" />
      <Radio.Group optionType="button" options={options} defaultValue="b" />
    </Space>
  )
}

export default Demo2
```

## API

### Radio

| Property         | Description                                      | Type                                        | Default |
| ---------------- | ------------------------------------------------ | ------------------------------------------- | ------- |
| `checked`        | Controlled checked state.                        | `boolean`                                   | `-`     |
| `defaultChecked` | Initial checked state for uncontrolled usage.    | `boolean`                                   | `false` |
| `disabled`       | Disables the radio.                              | `boolean`                                   | `false` |
| `value`          | Radio option value.                              | `string \| number \| boolean`               | `-`     |
| `prefixCls`      | Custom CSS class prefix.                         | `string`                                    | `-`     |
| `children`       | Radio label content.                             | `JSX.Element`                               | `-`     |
| `onChange`       | Called when the native radio change event fires. | `JSX.EventHandler<HTMLInputElement, Event>` | `-`     |

### Radio.Group

| Property       | Description                                                | Type                                  | Default     |
| -------------- | ---------------------------------------------------------- | ------------------------------------- | ----------- |
| `value`        | Controlled selected value.                                 | `string \| number \| boolean`         | `-`         |
| `defaultValue` | Initial selected value for uncontrolled usage.             | `string \| number \| boolean`         | `-`         |
| `options`      | Options rendered as radios.                                | `OptionInput[]`                       | `-`         |
| `disabled`     | Disables all radios in the group.                          | `boolean`                             | `false`     |
| `optionType`   | Renders options as standard radios or button-style radios. | `'default' \| 'button'`               | `'default'` |
| `prefixCls`    | Custom radio CSS class prefix.                             | `string`                              | `-`         |
| `children`     | Additional custom radio content.                           | `JSX.Element`                         | `-`         |
| `onChange`     | Called with the next selected value.                       | `(checkedValue: OptionValue) => void` | `-`         |

# Timeline

Display chronological events with labels, pending state, and custom dots.

### Basic

```tsx
import { Timeline } from '@ant-design-solid/core'

const items = [
  { label: '2026-06-01', children: 'Create project', color: 'green' },
  { label: '2026-06-02', children: 'Add core components' },
  { label: '2026-06-03', children: 'Polish documentation', color: 'red' },
]

const Demo1 = function () {
  return <Timeline items={items} />
}

export default Demo1
```

## API

### Timeline

| Property     | Description                                                        | Type                               | Default  |
| ------------ | ------------------------------------------------------------------ | ---------------------------------- | -------- |
| `items`      | Timeline item data.                                                | `TimelineItem[]`                   | `-`      |
| `mode`       | Timeline item placement mode.                                      | `'left' \| 'right' \| 'alternate'` | `'left'` |
| `pending`    | Adds a pending item at the end. Boolean true renders Loading text. | `JSX.Element \| boolean`           | `false`  |
| `pendingDot` | Custom dot for the pending item.                                   | `JSX.Element`                      | `-`      |
| `reverse`    | Reverses the item order before rendering pending.                  | `boolean`                          | `false`  |
| `prefixCls`  | Custom CSS class prefix.                                           | `string`                           | `-`      |

### TimelineItem

| Property   | Description                                            | Type                                             | Default  |
| ---------- | ------------------------------------------------------ | ------------------------------------------------ | -------- |
| `label`    | Label content rendered beside the timeline item.       | `JSX.Element`                                    | `-`      |
| `children` | Main timeline item content.                            | `JSX.Element`                                    | `-`      |
| `color`    | Preset semantic color or custom CSS color for the dot. | `'blue' \| 'red' \| 'green' \| 'gray' \| string` | `'blue'` |
| `dot`      | Custom dot content.                                    | `JSX.Element`                                    | `-`      |
| `position` | Overrides item position for this item.                 | `'left' \| 'right'`                              | `-`      |

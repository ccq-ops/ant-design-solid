# Collapse

Collapse displays content areas that can be expanded or collapsed.

### Basic

```tsx
import { Collapse, Space } from '@ant-design-solid/core'

const basicItems = [
  { key: '1', label: 'This is panel header 1', children: <p>{text}</p> },
  { key: '2', label: 'This is panel header 2', children: <p>{text}</p> },
  { key: '3', label: 'This is panel header 3', children: <p>{text}</p> },
]

const Demo1 = function () {
  return <Collapse items={basicItems} />
}

export default Demo1
```

### Ghost and borderless

```tsx
import { Collapse, Space } from '@ant-design-solid/core'

const basicItems = [
  { key: '1', label: 'This is panel header 1', children: <p>{text}</p> },
  { key: '2', label: 'This is panel header 2', children: <p>{text}</p> },
  { key: '3', label: 'This is panel header 3', children: <p>{text}</p> },
]

const Demo4 = function () {
  return (
    <Space direction="vertical" class="w-full">
      <Collapse ghost defaultActiveKey="1" items={basicItems.slice(0, 2)} />
      <Collapse bordered={false} defaultActiveKey="2" items={basicItems.slice(0, 2)} />
    </Space>
  )
}

export default Demo4
```

### Collapsible area

```tsx
import { Collapse, Space } from '@ant-design-solid/core'

const text = `A dog is a type of domesticated animal. Known for its loyalty and faithfulness, it can be found as a welcome guest in many households across the world.`

const Demo5 = function () {
  return (
    <Collapse
      items={[
        { key: '1', label: 'Icon only', collapsible: 'icon', children: <p>{text}</p> },
        { key: '2', label: 'Header only', collapsible: 'header', children: <p>{text}</p> },
        { key: '3', label: 'Disabled', disabled: true, children: <p>{text}</p> },
      ]}
    />
  )
}

export default Demo5
```

## API

### Collapse

| Property             | Description                                             | Type                                                  | Default   |
| -------------------- | ------------------------------------------------------- | ----------------------------------------------------- | --------- |
| `items`              | Collapse panel items.                                   | `CollapseItem[]`                                      | `-`       |
| `activeKey`          | Controlled active panel key or keys.                    | `string \| string[]`                                  | `-`       |
| `defaultActiveKey`   | Initial active panel key or keys.                       | `string \| string[]`                                  | `-`       |
| `accordion`          | Allows only one panel to be open at a time.             | `boolean`                                             | `false`   |
| `bordered`           | Whether to render borders.                              | `boolean`                                             | `true`    |
| `ghost`              | Uses ghost styling.                                     | `boolean`                                             | `false`   |
| `collapsible`        | Controls which part of panels toggles expansion.        | `'header' \| 'icon' \| 'disabled'`                    | `-`       |
| `expandIconPosition` | Expand icon placement.                                  | `'start' \| 'end'`                                    | `'start'` |
| `onChange`           | Called with active key or keys after expansion changes. | `(activeKey: CollapseActiveKey \| undefined) => void` | `-`       |

### CollapseItem

| Property      | Description                       | Type                               | Default |
| ------------- | --------------------------------- | ---------------------------------- | ------- |
| `key`         | Unique panel key.                 | `string`                           | `-`     |
| `label`       | Panel header content.             | `JSX.Element`                      | `-`     |
| `children`    | Panel body content.               | `JSX.Element`                      | `-`     |
| `extra`       | Extra header content.             | `JSX.Element`                      | `-`     |
| `disabled`    | Disables the panel.               | `boolean`                          | `false` |
| `collapsible` | Panel-level collapsible behavior. | `'header' \| 'icon' \| 'disabled'` | `-`     |
| `class`       | Additional panel class.           | `string`                           | `-`     |
| `style`       | Panel inline style.               | `JSX.CSSProperties`                | `-`     |

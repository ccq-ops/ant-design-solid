# FloatButton

FloatButton renders floating quick actions and a back-to-top affordance.

### Basic

```tsx
import { FloatButton, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Space>
      <FloatButton icon="?" description="Help" />
      <span>The button is fixed at the lower right of the viewport.</span>
    </Space>
  )
}

export default Demo1
```

### Link

```tsx
import { FloatButton, Space } from '@ant-design-solid/core'

const Demo2 = function () {
  return <FloatButton href="https://example.com" target="_blank" icon="↗" description="Open" />
}

export default Demo2
```

### Group

```tsx
import { FloatButton, Space } from '@ant-design-solid/core'

const Demo3 = function () {
  return (
    <FloatButton.Group shape="square">
      <FloatButton type="primary" icon="+" description="New" />
      <FloatButton icon="?" description="Help" />
    </FloatButton.Group>
  )
}

export default Demo3
```

### BackTop

```tsx
import { FloatButton, Space } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
<p>Scroll down and the back-to-top button appears after the configured threshold.</p>
        <FloatButton.BackTop visibilityHeight={200} />
  )
}

export default Demo4
```

## API

### FloatButton

| Property      | Description                                            | Type                     | Default     |
| ------------- | ------------------------------------------------------ | ------------------------ | ----------- |
| `type`        | Float button visual type.                              | `'default' \| 'primary'` | `'default'` |
| `shape`       | Float button shape.                                    | `'circle' \| 'square'`   | `'circle'`  |
| `icon`        | Icon content rendered inside the button.               | `JSX.Element`            | `-`         |
| `description` | Description content rendered below or beside the icon. | `JSX.Element`            | `-`         |
| `tooltip`     | Native title text for the floating action.             | `string`                 | `-`         |
| `href`        | Renders the float button as a link when provided.      | `string`                 | `-`         |
| `target`      | Anchor target when href is provided.                   | `string`                 | `-`         |

### FloatButton.Group

| Property   | Description                                   | Type                   | Default |
| ---------- | --------------------------------------------- | ---------------------- | ------- |
| `shape`    | Shared shape applied to buttons in the group. | `'circle' \| 'square'` | `-`     |
| `children` | FloatButton items rendered in the group.      | `JSX.Element`          | `-`     |

### FloatButton.BackTop

| Property           | Description                                             | Type                                               | Default  |
| ------------------ | ------------------------------------------------------- | -------------------------------------------------- | -------- |
| `visibilityHeight` | Scroll threshold before the back-to-top button appears. | `number`                                           | `400`    |
| `target`           | Function returning the scroll container.                | `() => Window \| HTMLElement \| undefined \| null` | `window` |
| `duration`         | Reserved scroll duration configuration.                 | `number`                                           | `-`      |
| `children`         | Custom back-to-top icon content.                        | `JSX.Element`                                      | `-`      |

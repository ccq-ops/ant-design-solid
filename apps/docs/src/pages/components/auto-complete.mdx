# AutoComplete

Text input with selectable suggestions.

### Basic

```tsx
import { AutoComplete, Form } from '@ant-design-solid/core'

const options = [{ value: 'Burns Bay Road' }, { value: 'Downing Street' }, { value: 'Wall Street' }]

const Demo1 = function () {
  return <AutoComplete placeholder="Search" options={options} />
}

export default Demo1
```

### No filtering

```tsx
import { AutoComplete, Form } from '@ant-design-solid/core'

const options = [{ value: 'Burns Bay Road' }, { value: 'Downing Street' }, { value: 'Wall Street' }]

const Demo2 = function () {
  return <AutoComplete filterOption={false} placeholder="Show all on focus" options={options} />
}

export default Demo2
```

### Clearable and disabled option

```tsx
import { AutoComplete, Form } from '@ant-design-solid/core'

const options = [{ value: 'Burns Bay Road' }, { value: 'Downing Street' }, { value: 'Wall Street' }]

const Demo3 = function () {
  return (
    <AutoComplete
      allowClear
      placeholder="Pick one"
      options={[...options, { value: 'Disabled option', disabled: true }]}
    />
  )
}

export default Demo3
```

### In Form

```tsx
import { AutoComplete, Form } from '@ant-design-solid/core'

const options = [{ value: 'Burns Bay Road' }, { value: 'Downing Street' }, { value: 'Wall Street' }]

const Demo4 = function () {
  return (
    <Form>
      <Form.Item name="street" label="Street">
        <AutoComplete placeholder="Street" options={options} />
      </Form.Item>
    </Form>
  )
}

export default Demo4
```

## API

### AutoComplete

| Property                   | Description                                                            | Type                                                                       | Default |
| -------------------------- | ---------------------------------------------------------------------- | -------------------------------------------------------------------------- | ------- |
| `value`                    | Controlled input value.                                                | `string`                                                                   | `-`     |
| `defaultValue`             | Initial input value for uncontrolled usage.                            | `string`                                                                   | `-`     |
| `open`                     | Controlled popup open state.                                           | `boolean`                                                                  | `-`     |
| `defaultOpen`              | Initial popup open state.                                              | `boolean`                                                                  | `false` |
| `options`                  | Suggestion options shown in the popup.                                 | `AutoCompleteOption[]`                                                     | `-`     |
| `placeholder`              | Input placeholder text.                                                | `string`                                                                   | `-`     |
| `disabled`                 | Disables text input and selection.                                     | `boolean`                                                                  | `false` |
| `size`                     | Input size.                                                            | `'small' \| 'middle' \| 'large'`                                           | `-`     |
| `status`                   | Validation visual status.                                              | `'error' \| 'warning'`                                                     | `-`     |
| `variant`                  | Visual input variant.                                                  | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`                   | `-`     |
| `allowClear`               | Shows a clear affordance. Object form customizes the clear icon.       | `boolean \| { clearIcon?: JSX.Element }`                                   | `false` |
| `showSearch`               | Enables search configuration including filterOption and onSearch.      | `true \| AutoCompleteShowSearch`                                           | `-`     |
| `defaultActiveFirstOption` | Activates the first option by default when the popup opens.            | `boolean`                                                                  | `-`     |
| `backfill`                 | Backfills the input with the active option during keyboard navigation. | `boolean`                                                                  | `-`     |
| `notFoundContent`          | Content shown when no options match.                                   | `JSX.Element`                                                              | `-`     |
| `popupMatchSelectWidth`    | Controls popup width matching behavior.                                | `boolean \| number`                                                        | `-`     |
| `popupRender`              | Customizes popup rendering.                                            | `(originNode: JSX.Element) => JSX.Element`                                 | `-`     |
| `filterOption`             | Controls built-in filtering or supplies a custom filter.               | `boolean \| ((inputValue: string, option: AutoCompleteOption) => boolean)` | `-`     |
| `prefixCls`                | Custom CSS class prefix.                                               | `string`                                                                   | `-`     |
| `zIndex`                   | Overrides popup z-index.                                               | `number`                                                                   | `-`     |
| `getPopupContainer`        | Returns the element used to mount the popup portal.                    | `(triggerNode?: HTMLElement) => HTMLElement`                               | `-`     |
| `onChange`                 | Called when input value changes.                                       | `(value: string) => void`                                                  | `-`     |
| `onSelect`                 | Called when an option is selected.                                     | `(value: string, option: AutoCompleteOption) => void`                      | `-`     |
| `onOpenChange`             | Called when popup open state changes.                                  | `(open: boolean) => void`                                                  | `-`     |
| `onClear`                  | Called after clearing the value.                                       | `() => void`                                                               | `-`     |
| `onInputKeyDown`           | Called for input keyboard events.                                      | `(event: KeyboardEvent) => void`                                           | `-`     |
| `onPopupScroll`            | Called when the popup scrolls.                                         | `(event: UIEvent) => void`                                                 | `-`     |

### AutoCompleteOption

| Property   | Description                                             | Type          | Default |
| ---------- | ------------------------------------------------------- | ------------- | ------- |
| `label`    | Option display label. Falls back to value when omitted. | `JSX.Element` | `-`     |
| `value`    | Option value inserted into the input.                   | `string`      | `-`     |
| `disabled` | Disables selecting this option.                         | `boolean`     | `-`     |

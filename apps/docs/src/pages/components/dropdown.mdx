# Dropdown

### Click

```tsx
import { createSignal } from 'solid-js'
import { Button, Dropdown, Space } from '@ant-design-solid/core'

const items = [
  { key: 'edit', label: 'Edit' },
  { key: 'copy', label: 'Copy' },
  { key: 'disabled', label: 'Disabled', disabled: true },
  { key: 'divider', type: 'divider' },
  { key: 'delete', label: 'Delete' },
]

const Demo1 = function () {
  return (
    <Dropdown menu={{ items, onClick: (info) => console.log(info.key) }}>
      <Button>Actions</Button>
    </Dropdown>
  )
}

export default Demo1
```

### Controlled

```tsx
import { createSignal } from 'solid-js'
import { Button, Dropdown, Space } from '@ant-design-solid/core'

const items = [
  { key: 'edit', label: 'Edit' },
  { key: 'copy', label: 'Copy' },
  { key: 'disabled', label: 'Disabled', disabled: true },
  { key: 'divider', type: 'divider' },
  { key: 'delete', label: 'Delete' },
]

const Demo4 = function () {
  return (
    <Space>
      <Dropdown open={open()} onOpenChange={setOpen} menu={{ items }}>
        <Button>Controlled</Button>
      </Dropdown>
      <Button onClick={() => setOpen((next) => !next)}>{open() ? 'Close' : 'Open'}</Button>
    </Space>
  )
}

export default Demo4
```

## API

### Dropdown

| Property            | Description                                           | Type                                         | Default   |
| ------------------- | ----------------------------------------------------- | -------------------------------------------- | --------- |
| `menu`              | Menu configuration rendered in the overlay.           | `DropdownMenuProps`                          | `-`       |
| `trigger`           | Interaction that opens the dropdown.                  | `'hover' \| 'click'`                         | `'click'` |
| `placement`         | Overlay placement relative to the trigger.            | `DropdownPlacement`                          | `-`       |
| `open`              | Controlled overlay open state.                        | `boolean`                                    | `-`       |
| `defaultOpen`       | Initial overlay open state for uncontrolled usage.    | `boolean`                                    | `false`   |
| `onOpenChange`      | Called when overlay open state changes.               | `(open: boolean) => void`                    | `-`       |
| `overlayClass`      | Additional class for the overlay.                     | `string`                                     | `-`       |
| `overlayStyle`      | Inline style for the overlay.                         | `JSX.CSSProperties`                          | `-`       |
| `zIndex`            | Overrides overlay z-index.                            | `number`                                     | `-`       |
| `getPopupContainer` | Returns the element used to mount the overlay portal. | `(triggerNode?: HTMLElement) => HTMLElement` | `-`       |

### Dropdown menu

| Property  | Description                                  | Type                                    | Default |
| --------- | -------------------------------------------- | --------------------------------------- | ------- |
| `items`   | Menu items rendered in the dropdown.         | `DropdownMenuItem[]`                    | `-`     |
| `onClick` | Called when an enabled menu item is clicked. | `(info: DropdownMenuClickInfo) => void` | `-`     |

### DropdownMenuItem

| Property   | Description                         | Type                  | Default |
| ---------- | ----------------------------------- | --------------------- | ------- |
| `key`      | Stable menu item key.               | `string`              | `-`     |
| `label`    | Menu item label.                    | `JSX.Element`         | `-`     |
| `disabled` | Disables this item.                 | `boolean`             | `-`     |
| `type`     | Renders a normal item or a divider. | `'item' \| 'divider'` | `-`     |

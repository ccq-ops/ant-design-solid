# Affix

Affix pins content to the viewport after the page scrolls past an offset.

### Offset top

```tsx
import { Affix, Button, Card, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Affix offsetTop={16}>
      <Button type="primary">Affixed top</Button>
    </Affix>
  )
}

export default Demo1
```

### Offset bottom

```tsx
import { Affix, Button, Card, Space } from '@ant-design-solid/core'

const Demo2 = function () {
  return (
    <Affix offsetBottom={24}>
      <Button>Affixed bottom</Button>
    </Affix>
  )
}

export default Demo2
```

### In content

```tsx
import { Affix, Button, Card, Space } from '@ant-design-solid/core'

const Demo3 = function () {
  return (
    <Space direction="vertical" class="w-full">
      <Affix offsetTop={80}>
        <Card size="small">Sticky tools stay visible while scrolling.</Card>
      </Affix>
      <p>Scroll the page to see the affix state change.</p>
    </Space>
  )
}

export default Demo3
```

## API

| Property       | Description                                             | Type                                               | Default |
| -------------- | ------------------------------------------------------- | -------------------------------------------------- | ------- |
| `offsetTop`    | Distance from the top of the target before affixing.    | `number`                                           | `-`     |
| `offsetBottom` | Distance from the bottom of the target before affixing. | `number`                                           | `-`     |
| `target`       | Scroll container used to calculate affix state.         | `() => Window \| HTMLElement \| undefined \| null` | `-`     |
| `onChange`     | Called when affixed state changes.                      | `(affixed: boolean) => void`                       | `-`     |
| `children`     | Content to pin.                                         | `JSX.Element`                                      | `-`     |

# Descriptions

Descriptions displays multiple read-only fields in groups.

### Basic

```tsx
import { Descriptions } from '@ant-design-solid/core'

const basicItems = [
  { label: 'UserName', children: 'Zhou Maomao' },
  { label: 'Telephone', children: '1810000000' },
  { label: 'Live', children: 'Hangzhou, Zhejiang' },
]

const Demo1 = function () {
  return <Descriptions title="User Info" items={basicItems} />
}

export default Demo1
```

### Bordered

```tsx
const detailItems = [

  { label: 'Billing Mode', children: 'Prepaid' },

  { label: 'Usage Time', children: '2019-04-24 18:00:00' },
```


### Children

```tsx
import { Descriptions } from '@ant-design-solid/core'

const Demo4 = function () {
  return (
    <Descriptions title="User Info" extra={<a href="#more">More</a>}>
      <Descriptions.Item label="UserName">Zhou Maomao</Descriptions.Item>
      <Descriptions.Item label="Telephone">1810000000</Descriptions.Item>
      <Descriptions.Item label="Address" span={2}>
        Hangzhou, Zhejiang
      </Descriptions.Item>
    </Descriptions>
  )
}

export default Demo4
```

## API

### Descriptions

| Property   | Description                                                  | Type                               | Default        |
| ---------- | ------------------------------------------------------------ | ---------------------------------- | -------------- |
| `title`    | Header title content.                                        | `JSX.Element`                      | `-`            |
| `extra`    | Header extra content.                                        | `JSX.Element`                      | `-`            |
| `bordered` | Whether to render bordered table cells.                      | `boolean`                          | `false`        |
| `column`   | Number of description columns.                               | `number`                           | `3`            |
| `size`     | Descriptions size.                                           | `'default' \| 'middle' \| 'small'` | `'default'`    |
| `layout`   | Label/content layout.                                        | `'horizontal' \| 'vertical'`       | `'horizontal'` |
| `items`    | Description item data.                                       | `DescriptionsItemType[]`           | `-`            |
| `children` | Descriptions.Item children used when items are not provided. | `JSX.Element`                      | `-`            |

### Descriptions.Item

| Property    | Description                             | Type                          | Default |
| ----------- | --------------------------------------- | ----------------------------- | ------- |
| `label`     | Item label content.                     | `JSX.Element`                 | `-`     |
| `children`  | Item value content.                     | `JSX.Element`                 | `-`     |
| `span`      | Number of columns occupied by the item. | `number`                      | `1`     |
| `class`     | Additional item class.                  | `string`                      | `-`     |
| `className` | Additional item class alias.            | `string`                      | `-`     |
| `style`     | Item inline style.                      | `JSX.CSSProperties \| string` | `-`     |

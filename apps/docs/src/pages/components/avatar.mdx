# Avatar

### Sizes

```tsx
import { Avatar, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Space align="center" wrap>
      <Avatar size="small">S</Avatar>
      <Avatar>M</Avatar>
      <Avatar size="large">L</Avatar>
      <Avatar size={56}>56</Avatar>
    </Space>
  )
}

export default Demo1
```

### Shapes

```tsx
import { Avatar, Space } from '@ant-design-solid/core'

const Demo2 = function () {
  return (
    <Space align="center" wrap>
      <Avatar>U</Avatar>
      <Avatar shape="square">U</Avatar>
      <Avatar shape="square" size="large">
        AB
      </Avatar>
    </Space>
  )
}

export default Demo2
```

### Image fallback

```tsx
import { Avatar, Space } from '@ant-design-solid/core'

const Demo3 = function () {
  return (
    <Space align="center" wrap>
      <Avatar src="https://example.invalid/missing-avatar.png" alt="Jane Doe">
        JD
      </Avatar>
      <span>When the image fails to load, Avatar falls back to text.</span>
    </Space>
  )
}

export default Demo3
```

### Icon and text

```tsx
import { Avatar, Space } from '@ant-design-solid/core'

function UserIcon() {
  return <span aria-hidden="true">👤</span>
}

const Demo4 = function () {
  return (
    <Space align="center" wrap>
      <Avatar icon={<UserIcon />} />
      <Avatar>Tom</Avatar>
      <Avatar style={{ background: '#1677ff' }}>A</Avatar>
    </Space>
  )
}

export default Demo4
```

### Group overflow

```tsx
import { Avatar, Space } from '@ant-design-solid/core'

const Demo5 = function () {
  return (
    <Avatar.Group maxCount={3} size="large">
      <Avatar>A</Avatar>
      <Avatar>B</Avatar>
      <Avatar>C</Avatar>
      <Avatar>D</Avatar>
      <Avatar>E</Avatar>
    </Avatar.Group>
  )
}

export default Demo5
```

## API

### Avatar

| Property   | Description                                                   | Type                                        | Default     |
| ---------- | ------------------------------------------------------------- | ------------------------------------------- | ----------- |
| `size`     | Avatar size or custom pixel size.                             | `'small' \| 'default' \| 'large' \| number` | `'default'` |
| `shape`    | Avatar shape.                                                 | `'circle' \| 'square'`                      | `'circle'`  |
| `src`      | Image URL. Falls back to icon or children after load failure. | `string`                                    | `-`         |
| `alt`      | Alternate text for the image avatar.                          | `string`                                    | `-`         |
| `icon`     | Icon content shown when no valid image is available.          | `JSX.Element`                               | `-`         |
| `gap`      | Reserved text gap configuration.                              | `number`                                    | `-`         |
| `children` | Text or custom content shown inside the avatar.               | `JSX.Element`                               | `-`         |

### Avatar.Group

| Property   | Description                                                            | Type                                        | Default |
| ---------- | ---------------------------------------------------------------------- | ------------------------------------------- | ------- |
| `maxCount` | Maximum number of avatars to show before rendering an overflow avatar. | `number`                                    | `-`     |
| `maxStyle` | Inline style applied to the overflow avatar.                           | `JSX.CSSProperties`                         | `-`     |
| `size`     | Shared size applied to avatars in the group.                           | `'small' \| 'default' \| 'large' \| number` | `-`     |
| `shape`    | Shared shape applied to avatars in the group.                          | `'circle' \| 'square'`                      | `-`     |
| `children` | Avatar items rendered in the group.                                    | `JSX.Element`                               | `-`     |

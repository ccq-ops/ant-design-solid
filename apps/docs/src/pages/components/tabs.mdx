# Tabs

### Basic

```tsx
import { Tabs } from '@ant-design-solid/core'

const items = [
  { key: 'overview', label: 'Overview', children: <p>Overview content</p> },
  { key: 'usage', label: 'Usage', children: <p>Usage content</p> },
  {
    key: 'disabled',
    label: 'Disabled',
    disabled: true,
    children: <p>Disabled content</p>,
  },
]

const Demo1 = function () {
  return <Tabs items={items} />
}

export default Demo1
```

### Destroy inactive pane

import { Tabs } from '@ant-design-solid/core'

const items = [
  { key: 'overview', label: 'Overview', children: <p>Overview content</p> },

const Demo4 = function () {

## API

### Tabs

| Property                 | Description                                    | Type                          | Default  |
| ------------------------ | ---------------------------------------------- | ----------------------------- | -------- |
| `items`                  | Tab items rendered as tabs and panes.          | `TabsItem[]`                  | `-`      |
| `activeKey`              | Controlled active tab key.                     | `string`                      | `-`      |
| `defaultActiveKey`       | Initial active tab key for uncontrolled usage. | `string`                      | `-`      |
| `onChange`               | Called when the active tab changes.            | `(activeKey: string) => void` | `-`      |
| `type`                   | Tab visual style.                              | `'line' \| 'card'`            | `'line'` |
| `size`                   | Tab size from the theme component size scale.  | `ComponentSize`               | `-`      |
| `tabPosition`            | Tab bar position.                              | `'top' \| 'bottom'`           | `'top'`  |
| `destroyInactiveTabPane` | Unmounts inactive tab pane content.            | `boolean`                     | `false`  |

### TabsItem

| Property   | Description        | Type          | Default |
| ---------- | ------------------ | ------------- | ------- |
| `key`      | Stable tab key.    | `string`      | `-`     |
| `label`    | Tab label content. | `JSX.Element` | `-`     |
| `children` | Tab pane content.  | `JSX.Element` | `-`     |
| `disabled` | Disables this tab. | `boolean`     | `-`     |

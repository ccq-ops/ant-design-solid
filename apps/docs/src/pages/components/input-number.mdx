# InputNumber

Enter numeric values with optional controls, bounds, precision, and formatting.

Its onChange callback receives the normalized numeric value, not a DOM change event.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Form, InputNumber, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return <InputNumber defaultValue={3} />
}

export default Demo1
```

### Formatter and parser

```tsx
import { createSignal } from 'solid-js'
import { Form, InputNumber, Space } from '@ant-design-solid/core'

const Demo5 = function () {
  return (
    <InputNumber
      defaultValue={1000}
      formatter={(nextValue, info) =>
        info.userTyping ? info.input : nextValue === undefined ? '' : `$ ${nextValue}`
      }
      parser={(displayValue) => {
        const parsed = Number(displayValue.replace(/[$,\s]/g, ''))
        return Number.isNaN(parsed) ? undefined : parsed
      }}
    />
  )
}

export default Demo5
```

## API

| Property           | Description                                                                      | Type                                                                                                                    | Default      |
| ------------------ | -------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- | ------------ |
| `value`            | Controlled value. In stringMode, use a string value for high precision decimals. | `number \| string`                                                                                                      | `-`          |
| `defaultValue`     | Initial value for uncontrolled usage.                                            | `number \| string`                                                                                                      | `-`          |
| `min`              | Minimum allowed value.                                                           | `number`                                                                                                                | `-`          |
| `max`              | Maximum allowed value.                                                           | `number`                                                                                                                | `-`          |
| `step`             | Increment used by controls, keyboard, and wheel stepping.                        | `number \| string`                                                                                                      | `1`          |
| `precision`        | Number of decimal places to keep.                                                | `number`                                                                                                                | `-`          |
| `placeholder`      | Input placeholder text.                                                          | `string`                                                                                                                | `-`          |
| `disabled`         | Disables input and controls.                                                     | `boolean`                                                                                                               | `false`      |
| `readOnly`         | Makes the input read-only and disables stepping interactions.                    | `boolean`                                                                                                               | `false`      |
| `size`             | Input size from the theme component size scale.                                  | `'small' \| 'middle' \| 'large'`                                                                                        | `-`          |
| `status`           | Validation visual status.                                                        | `'error' \| 'warning'`                                                                                                  | `-`          |
| `controls`         | Shows increment/decrement controls, or customizes their icons.                   | `boolean \| { upIcon?: JSX.Element; downIcon?: JSX.Element }`                                                           | `true`       |
| `keyboard`         | Enables ArrowUp and ArrowDown stepping.                                          | `boolean`                                                                                                               | `true`       |
| `changeOnBlur`     | Commits typed input on blur when enabled, or while typing when disabled.         | `boolean`                                                                                                               | `true`       |
| `changeOnWheel`    | Allows mouse wheel stepping.                                                     | `boolean`                                                                                                               | `false`      |
| `prefix`           | Content rendered before the input.                                               | `JSX.Element`                                                                                                           | `-`          |
| `suffix`           | Content rendered after the input.                                                | `JSX.Element`                                                                                                           | `-`          |
| `variant`          | Visual variant of the input.                                                     | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`                                                                | `'outlined'` |
| `mode`             | Display mode for the input and controls.                                         | `'input' \| 'spinner'`                                                                                                  | `'input'`    |
| `stringMode`       | Returns committed values as strings for high precision scenarios.                | `boolean`                                                                                                               | `false`      |
| `decimalSeparator` | Custom decimal separator for parsing and display.                                | `string`                                                                                                                | `-`          |
| `formatter`        | Formats the value for display. The info argument indicates typing state.         | `(value: number \| string \| undefined, info: { userTyping: boolean; input: string }) => string`                        | `-`          |
| `parser`           | Parses displayed text back into a value.                                         | `(displayValue: string) => number \| string \| undefined`                                                               | `-`          |
| `classNames`       | Semantic DOM class names for root, prefix, suffix, input, and actions.           | `InputNumberSemanticClassNames`                                                                                         | `-`          |
| `styles`           | Semantic DOM inline styles for root, prefix, suffix, input, and actions.         | `InputNumberSemanticStyles`                                                                                             | `-`          |
| `rootClassName`    | Additional class name for the root element.                                      | `string`                                                                                                                | `-`          |
| `prefixCls`        | Custom component CSS class prefix.                                               | `string`                                                                                                                | `-`          |
| `onChange`         | Called with the normalized value, not a DOM change event.                        | `(value: number \| string \| undefined) => void`                                                                        | `-`          |
| `onPressEnter`     | Called when Enter is pressed.                                                    | `(event: KeyboardEvent) => void`                                                                                        | `-`          |
| `onStep`           | Called after stepping by controls, keyboard, or wheel.                           | `(value, info: { offset: number \| string; type: 'up' \| 'down'; emitter: 'handler' \| 'keydown' \| 'wheel' }) => void` | `-`          |

# List

List displays a group of related items with optional headers, actions, and metadata.

### Data source

:::preview

## API

### List

| Property     | Description                                                   | Type                                       | Default      |
| ------------ | ------------------------------------------------------------- | ------------------------------------------ | ------------ |
| `prefixCls`  | Custom component class prefix.                                | `string`                                   | `ads-list`   |
| `rootClass`  | Extra class on the root element.                              | `string`                                   | `-`          |
| `dataSource` | Array of data items rendered by renderItem.                   | `T[]`                                      | `-`          |
| `renderItem` | Render function for each data item.                           | `(item: T, index: number) => JSX.Element`  | `-`          |
| `rowKey`     | Unique item key, read from item or returned by a function.    | `keyof T \| (item: T) => string \| number` | `key`        |
| `header`     | Header content rendered above the list items.                 | `JSX.Element`                              | `-`          |
| `footer`     | Footer content rendered below the list items.                 | `JSX.Element`                              | `-`          |
| `grid`       | Grid layout config.                                           | [`ListGridType`](#list-gridtype)           | `-`          |
| `itemLayout` | List item layout.                                             | `'horizontal' \| 'vertical'`               | `horizontal` |
| `bordered`   | Whether to render the list border.                            | `boolean`                                  | `false`      |
| `split`      | Whether to render separators between items.                   | `boolean`                                  | `true`       |
| `size`       | List size.                                                    | `'default' \| 'large' \| 'small'`          | `'default'`  |
| `loading`    | Shows a loading indicator while data is being fetched.        | `boolean \| SpinProps`                     | `false`      |
| `loadMore`   | Content rendered after the list and before bottom pagination. | `JSX.Element`                              | `-`          |
| `locale`     | Locale text for empty state.                                  | `{ emptyText?: JSX.Element }`              | `-`          |
| `pagination` | Pagination config. Set to `false` to disable.                 | `boolean \| ListPaginationConfig`          | `false`      |
| `emptyText`  | Empty content alias. `locale.emptyText` has higher priority.  | `JSX.Element`                              | `'No Data'`  |
| `children`   | Manual list items used when dataSource is not provided.       | `JSX.Element`                              | `-`          |

### ListPaginationConfig

Extends `PaginationProps`.

| Property   | Description                 | Type                          | Default  |
| ---------- | --------------------------- | ----------------------------- | -------- |
| `position` | Pagination render position. | `'top' \| 'bottom' \| 'both'` | `bottom` |

### ListGridType

| Property | Description                   | Type                         | Default |
| -------- | ----------------------------- | ---------------------------- | ------- |
| `gutter` | Spacing between grid columns. | `number \| [number, number]` | `0`     |
| `column` | Default column count.         | `number`                     | `-`     |
| `xs`     | Column count below 576px.     | `number`                     | `-`     |
| `sm`     | Column count at least 576px.  | `number`                     | `-`     |
| `md`     | Column count at least 768px.  | `number`                     | `-`     |
| `lg`     | Column count at least 992px.  | `number`                     | `-`     |
| `xl`     | Column count at least 1200px. | `number`                     | `-`     |
| `xxl`    | Column count at least 1600px. | `number`                     | `-`     |
| `xxxl`   | Column count at least 1920px. | `number`                     | `-`     |

### List.Item

| Property     | Description                                     | Type                                                         | Default |
| ------------ | ----------------------------------------------- | ------------------------------------------------------------ | ------- |
| `actions`    | Action items rendered after the main content.   | `JSX.Element[]`                                              | `-`     |
| `extra`      | Extra content rendered at the end of the item.  | `JSX.Element`                                                | `-`     |
| `classNames` | Semantic class names for `actions` and `extra`. | `{ actions?: string; extra?: string }`                       | `-`     |
| `styles`     | Semantic styles for `actions` and `extra`.      | `{ actions?: JSX.CSSProperties; extra?: JSX.CSSProperties }` | `-`     |
| `children`   | Main list item content.                         | `JSX.Element`                                                | `-`     |

### List.Item.Meta

| Property      | Description                              | Type          | Default |
| ------------- | ---------------------------------------- | ------------- | ------- |
| `avatar`      | Avatar content rendered before metadata. | `JSX.Element` | `-`     |
| `title`       | Metadata title.                          | `JSX.Element` | `-`     |
| `description` | Metadata description.                    | `JSX.Element` | `-`     |
| `children`    | Additional metadata content.             | `JSX.Element` | `-`     |

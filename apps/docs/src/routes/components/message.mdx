## API

### MessageArgs

| Property       | Description                                                                        | Type                                                                                      | Default  |
| -------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- | -------- |
| `key`          | Stable key used to update or destroy a message.                                    | `string \| number`                                                                        | `-`      |
| `type`         | Message visual type.                                                               | `'info' \| 'success' \| 'error' \| 'warning' \| 'loading'`                                | `'info'` |
| `content`      | Message content.                                                                   | `JSX.Element`                                                                             | `-`      |
| `duration`     | Auto-close delay in seconds. Set to 0 to keep open. Loading messages default to 0. | `number`                                                                                  | `3`      |
| `onClose`      | Called when the message closes.                                                    | `() => void`                                                                              | `-`      |
| `icon`         | Custom icon node.                                                                  | `JSX.Element`                                                                             | `-`      |
| `class`        | Custom class for the notice wrapper.                                               | `string`                                                                                  | `-`      |
| `className`    | React-compatible alias for `class`.                                                | `string`                                                                                  | `-`      |
| `style`        | Custom style for the notice wrapper.                                               | `JSX.CSSProperties`                                                                       | `-`      |
| `classNames`   | Semantic class map, or a function receiving `{ props }`.                           | `Record<SemanticDOM, string> \| ({ props }) => Record<SemanticDOM, string>`               | `-`      |
| `styles`       | Semantic style map, or a function receiving `{ props }`.                           | `Record<SemanticDOM, CSSProperties> \| ({ props }) => Record<SemanticDOM, CSSProperties>` | `-`      |
| `onClick`      | Called when the message content is clicked.                                        | `(event: MouseEvent) => void`                                                             | `-`      |
| `pauseOnHover` | Pause auto-close timer while hovering.                                             | `boolean`                                                                                 | `true`   |

### MessageHandle

| Property | Description                                           | Type         | Default |
| -------- | ----------------------------------------------------- | ------------ | ------- |
| `call`   | The handle itself can be called to close the message. | `() => void` | `-`     |
| `close`  | Closes the message created by the returned handle.    | `() => void` | `-`     |

### MessageConfigOptions

| Property         | Description                                                     | Type                                                                                      | Default |
| ---------------- | --------------------------------------------------------------- | ----------------------------------------------------------------------------------------- | ------- |
| `top`            | Distance from top.                                              | `string \| number`                                                                        | `8`     |
| `duration`       | Global default duration in seconds.                             | `number`                                                                                  | `3`     |
| `getContainer`   | Returns the mount container.                                    | `() => HTMLElement`                                                                       | `-`     |
| `maxCount`       | Maximum visible messages; older messages are closed.            | `number`                                                                                  | `-`     |
| `prefixCls`      | Custom message root class prefix.                               | `string`                                                                                  | `-`     |
| `rtl`            | Enable RTL class.                                               | `boolean`                                                                                 | `false` |
| `pauseOnHover`   | Global pause-on-hover default.                                  | `boolean`                                                                                 | `true`  |
| `stack`          | Stack messages when the amount is over the threshold.           | `boolean \| { threshold?: number }`                                                       | `-`     |
| `transitionName` | Adds transition class names to each notice.                     | `string`                                                                                  | `-`     |
| `class`          | Global class for each notice wrapper.                           | `string`                                                                                  | `-`     |
| `className`      | React-compatible alias for `class`.                             | `string`                                                                                  | `-`     |
| `style`          | Global style for each notice wrapper.                           | `JSX.CSSProperties`                                                                       | `-`     |
| `classNames`     | Global semantic class map, or a function receiving `{ props }`. | `Record<SemanticDOM, string> \| ({ props }) => Record<SemanticDOM, string>`               | `-`     |
| `styles`         | Global semantic style map, or a function receiving `{ props }`. | `Record<SemanticDOM, CSSProperties> \| ({ props }) => Record<SemanticDOM, CSSProperties>` | `-`     |

### MessageInstance

| Property             | Description                                                                | Type                                                                | Default |
| -------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------- | ------- |
| `message.open`       | Opens a message with a full configuration object.                          | `(args: MessageArgs) => MessageHandle`                              | `-`     |
| `message.info`       | Opens an info message.                                                     | `(content, duration?, onClose?) => MessageHandle`                   | `-`     |
| `message.success`    | Opens a success message.                                                   | `(content, duration?, onClose?) => MessageHandle`                   | `-`     |
| `message.error`      | Opens an error message.                                                    | `(content, duration?, onClose?) => MessageHandle`                   | `-`     |
| `message.warning`    | Opens a warning message.                                                   | `(content, duration?, onClose?) => MessageHandle`                   | `-`     |
| `message.loading`    | Opens a loading message. It remains until closed when duration is omitted. | `(content, duration?, onClose?) => MessageHandle`                   | `-`     |
| `message.destroy`    | Destroys one keyed message or all messages.                                | `(key?: string \| number) => void`                                  | `-`     |
| `message.config`     | Configures global message defaults.                                        | `(options: MessageConfigOptions) => void`                           | `-`     |
| `message.useMessage` | Creates a local message API and context holder.                            | `(config?: MessageConfigOptions) => [MessageInstance, JSX.Element]` | `-`     |

### ConfigProvider

`ConfigProvider` supports component-level message defaults through the `message` prop:

```tsx
<ConfigProvider message={{ class: 'app-message', classNames: { title: 'app-message-title' } }}>
  <Page />
</ConfigProvider>
```

In Solid, call `message.useMessage()` inside a child component rendered under the provider so the holder is created in that provider's owner context.

### Semantic DOM

`classNames` and `styles` support these semantic slots:

| Slot          | Description                                               |
| ------------- | --------------------------------------------------------- |
| `root`        | Message root container.                                   |
| `list`        | Alias for the root/list container for antd compatibility. |
| `listContent` | Content node inside each notice.                          |
| `wrapper`     | Individual notice wrapper.                                |
| `icon`        | Icon wrapper.                                             |
| `title`       | Content text wrapper.                                     |

### Internal debug helpers

The static `message` object also exposes `_InternalPanelDoNotUseOrYouWillBeFired` and `_InternalListDoNotUseOrYouWillBeFired` for internal demos and visual debugging. They mirror antd's internal message helpers and should not be used as stable application APIs.

# Affix

Affix pins content to the viewport or a scroll container after scrolling past an offset.

### Basic

:::preview

<AffixDemo001 />

export const AffixDemo001 = (() => {
  const Demo1 = function () {
      const [top, setTop] = createSignal(100);
      const [bottom, setBottom] = createSignal(100);
      return (<>
        <Affix offsetTop={top()}>
          <Button type="primary" onClick={() => setTop(top() + 10)}>
            Affix top
          </Button>
        </Affix>
        <br />
        <Affix offsetBottom={bottom()}>
          <Button type="primary" onClick={() => setBottom(bottom() + 10)}>
            Affix bottom
          </Button>
        </Affix>
      </>);
  };

return Demo1
})()

## API

| Property       | Description                                             | Type                                               | Default        |
| -------------- | ------------------------------------------------------- | -------------------------------------------------- | -------------- |
| `offsetTop`    | Distance from the top of the target before affixing.    | `number`                                           | `0`            |
| `offsetBottom` | Distance from the bottom of the target before affixing. | `number`                                           | `-`            |
| `target`       | Scroll container used to calculate affix state.         | `() => Window \| HTMLElement \| undefined \| null` | `() => window` |
| `onChange`     | Called when affixed state changes.                      | `(affixed?: boolean) => void`                      | `-`            |
| `prefixCls`    | Custom component CSS class prefix.                      | `string`                                           | `ads-affix`    |
| `rootClass`    | Custom class applied to the fixed Affix element.        | `string`                                           | `-`            |
| `ref`          | Imperative Affix methods.                               | `AffixRef \| ((ref: AffixRef) => void)`            | `-`            |
| `children`     | Content to pin.                                         | `JSX.Element`                                      | `-`            |

```ts
interface AffixRef {
  updatePosition: () => void
}
```

Affix follows Solid naming conventions: use `class` for the wrapper class and `rootClass` for the fixed element class.

## Design Token

| Token         | Description          | Type     | Default           |
| ------------- | -------------------- | -------- | ----------------- |
| `zIndexPopup` | z-index of the Affix | `number` | `zIndexBase + 10` |

# InputNumber

Enter numeric values with optional controls, bounds, precision, and formatting.

Its onChange callback receives the normalized numeric value, not a DOM change event.
Empty committed values are reported as `null`, matching Ant Design's InputNumber behavior.

### Basic

```tsx
import { createSignal } from 'solid-js'
import { Form, InputNumber, Space } from '@ant-design-solid/core'

const Demo1 = function () {
  return <InputNumber defaultValue={3} />
}

export default Demo1
```

### Formatter and parser

```tsx
import { createSignal } from 'solid-js'
import { Form, InputNumber, Space } from '@ant-design-solid/core'

const Demo5 = function () {
  return (
    <InputNumber
      defaultValue={1000}
      formatter={(nextValue, info) =>
        info.userTyping ? info.input : nextValue === null ? '' : `$ ${nextValue}`
      }
      parser={(displayValue) => {
        const parsed = Number((displayValue ?? '').replace(/[$,\s]/g, ''))
        return Number.isNaN(parsed) ? null : parsed
      }}
    />
  )
}

export default Demo5
```

### Prefix, suffix, variants, and custom controls

```tsx
import { createSignal } from 'solid-js'
import { MinusOutlined, PlusOutlined } from '@ant-design-solid/icons'
import { Form, InputNumber, Space, useToken } from '@ant-design-solid/core'

const Demo8 = function () {
  const token = useToken()
  const mutedTextStyle = () => ({ color: token().colorTextSecondary })

  return (
    <Space wrap align="center">
      <InputNumber
        defaultValue={1280}
        prefix={<span style={mutedTextStyle()}>$</span>}
        suffix={<span style={mutedTextStyle()}>USD</span>}
        formatter={(value) => (value === null ? '' : Number(value).toLocaleString())}
        parser={(displayValue) => {
          const parsed = Number((displayValue ?? '').replace(/,/g, ''))
          return Number.isNaN(parsed) ? null : parsed
        }}
        variant="filled"
      />
      <InputNumber
        defaultValue={12}
        min={0}
        max={100}
        suffix={<span style={mutedTextStyle()}>%</span>}
        controls={{
          upIcon: <PlusOutlined width="0.85em" height="0.85em" />,
          downIcon: <MinusOutlined width="0.85em" height="0.85em" />,
        }}
        mode="spinner"
      />
      <InputNumber
        defaultValue={32}
        suffix={<span style={mutedTextStyle()}>items</span>}
        variant="underlined"
      />
    </Space>
  )
}

export default Demo8
```

### Keyboard, wheel, and step events

```tsx
import { createSignal } from 'solid-js'
import { Form, InputNumber, Space } from '@ant-design-solid/core'

const Demo10 = function () {
  const [stepInfo, setStepInfo] = createSignal('No step yet')

  return (
    <Space>
      <InputNumber
        defaultValue={2}
        step={0.5}
        changeOnWheel
        onStep={(nextValue, info) =>
          setStepInfo(`${info.emitter} ${info.type}: ${nextValue ?? 'empty'}`)
        }
      />
      <InputNumber defaultValue={2} keyboard={false} />
      <span>{stepInfo()}</span>
    </Space>
  )
}

export default Demo10
```

## API

| Property           | Description                                                                                                | Type                                                                                                                    | Default      |
| ------------------ | ---------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- | ------------ |
| `value`            | Controlled value. In stringMode, use a string value for high precision decimals.                           | `number \| string \| null`                                                                                              | `-`          |
| `defaultValue`     | Initial value for uncontrolled usage.                                                                      | `number \| string \| null`                                                                                              | `-`          |
| `min`              | Minimum allowed value.                                                                                     | `number`                                                                                                                | `-`          |
| `max`              | Maximum allowed value.                                                                                     | `number`                                                                                                                | `-`          |
| `step`             | Increment used by controls, keyboard, and wheel stepping.                                                  | `number \| string`                                                                                                      | `1`          |
| `precision`        | Number of decimal places to keep.                                                                          | `number`                                                                                                                | `-`          |
| `placeholder`      | Input placeholder text.                                                                                    | `string`                                                                                                                | `-`          |
| `disabled`         | Disables input and controls.                                                                               | `boolean`                                                                                                               | `false`      |
| `readOnly`         | Makes the input read-only and disables stepping interactions.                                              | `boolean`                                                                                                               | `false`      |
| `size`             | Input size from the theme component size scale.                                                            | `'small' \| 'middle' \| 'large'`                                                                                        | `-`          |
| `status`           | Validation visual status.                                                                                  | `'error' \| 'warning'`                                                                                                  | `-`          |
| `controls`         | Shows increment/decrement controls, or customizes their icons.                                             | `boolean \| { upIcon?: JSX.Element; downIcon?: JSX.Element }`                                                           | `true`       |
| `addonBefore`      | Deprecated compatibility addon rendered before the input number.                                           | `JSX.Element`                                                                                                           | `-`          |
| `addonAfter`       | Deprecated compatibility addon rendered after the input number.                                            | `JSX.Element`                                                                                                           | `-`          |
| `bordered`         | Deprecated compatibility prop. Use `variant` instead.                                                      | `boolean`                                                                                                               | `true`       |
| `keyboard`         | Enables ArrowUp and ArrowDown stepping.                                                                    | `boolean`                                                                                                               | `true`       |
| `changeOnBlur`     | Commits typed input on blur when enabled, or while typing when disabled.                                   | `boolean`                                                                                                               | `true`       |
| `changeOnWheel`    | Allows mouse wheel stepping.                                                                               | `boolean`                                                                                                               | `false`      |
| `prefix`           | Content rendered before the input.                                                                         | `JSX.Element`                                                                                                           | `-`          |
| `suffix`           | Content rendered after the input.                                                                          | `JSX.Element`                                                                                                           | `-`          |
| `variant`          | Visual variant of the input.                                                                               | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`                                                                | `'outlined'` |
| `mode`             | Display mode for the input and controls.                                                                   | `'input' \| 'spinner'`                                                                                                  | `'input'`    |
| `stringMode`       | Returns committed values as strings for high precision scenarios.                                          | `boolean`                                                                                                               | `false`      |
| `decimalSeparator` | Custom decimal separator for parsing and display.                                                          | `string`                                                                                                                | `-`          |
| `formatter`        | Formats the value for display. The info argument indicates typing state.                                   | `(value: number \| string \| null, info: { userTyping: boolean; input: string }) => string`                             | `-`          |
| `parser`           | Parses displayed text back into a value.                                                                   | `(displayValue: string \| undefined) => number \| string \| null`                                                       | `-`          |
| `classNames`       | Semantic DOM class names for root, prefix, suffix, input, and actions. Supports object or function form.   | `InputNumberSemanticClassNames \| ((info: { props }) => InputNumberSemanticClassNames)`                                 | `-`          |
| `styles`           | Semantic DOM inline styles for root, prefix, suffix, input, and actions. Supports object or function form. | `InputNumberSemanticStyles \| ((info: { props }) => InputNumberSemanticStyles)`                                         | `-`          |
| `rootClassName`    | Additional class name for the root element.                                                                | `string`                                                                                                                | `-`          |
| `prefixCls`        | Custom component CSS class prefix.                                                                         | `string`                                                                                                                | `-`          |
| `onInput`          | Called with the typed display text.                                                                        | `(text: string) => void`                                                                                                | `-`          |
| `onChange`         | Called with the normalized value, not a DOM change event.                                                  | `(value: number \| string \| null) => void`                                                                             | `-`          |
| `onPressEnter`     | Called when Enter is pressed.                                                                              | `(event: KeyboardEvent) => void`                                                                                        | `-`          |
| `onStep`           | Called after stepping by controls, keyboard, or wheel.                                                     | `(value, info: { offset: number \| string; type: 'up' \| 'down'; emitter: 'handler' \| 'keydown' \| 'wheel' }) => void` | `-`          |

## Ref

| Name             | Description                   | Type                                                                                  |
| ---------------- | ----------------------------- | ------------------------------------------------------------------------------------- |
| `focus(options)` | Focuses the input.            | `(options?: { preventScroll?: boolean; cursor?: 'start' \| 'end' \| 'all' }) => void` |
| `blur()`         | Removes focus from the input. | `() => void`                                                                          |
| `nativeElement`  | Root InputNumber DOM element. | `HTMLElement \| undefined`                                                            |

## Semantic DOM

| Slot      | Description                           |
| --------- | ------------------------------------- |
| `root`    | Root InputNumber element.             |
| `prefix`  | Prefix slot.                          |
| `suffix`  | Suffix slot.                          |
| `input`   | Native input element.                 |
| `actions` | Increment/decrement controls wrapper. |

| ------------------- | -------------------------------------- |
| `activeBorderColor` | Border color when focused.             |
| `activeShadow`      | Focus shadow.                          |

| `handleActiveBg`    | Handle hover/active background.        |
| `handleBg`          | Handle background.                     |

| `handleFontSize`    | Handle icon font size.                 |
| `handleHoverColor`  | Handle hover color.                    |
| `handleVisible`     | Whether handlers are visually visible. |
| `handleWidth`       | Handle column width.                   |
| `hoverBorderColor`  | Border color on hover.                 |
| `inputFontSize`     | Default input font size.               |
| `inputFontSizeLG`   | Large input font size.                 |
| `inputFontSizeSM`   | Small input font size.                 |
| `paddingBlock`      | Default vertical input padding.        |
| `paddingBlockLG`    | Large vertical input padding.          |
| `paddingBlockSM`    | Small vertical input padding.          |
| `paddingInline`     | Default horizontal input padding.      |
| `paddingInlineLG`   | Large horizontal input padding.        |
| `paddingInlineSM`   | Small horizontal input padding.        |

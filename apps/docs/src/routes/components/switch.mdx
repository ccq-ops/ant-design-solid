# Switch

Switch toggles a boolean state on or off.

### Basic

:::preview

return Demo1

## API

| Property            | Description                                                                                                     | Type                                                    | Default                                                |
| ------------------- | --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------------ |
| `checked`           | Controlled checked state.                                                                                       | `boolean`                                               | `-`                                                    |
| `value`             | Alias for `checked`, matching antd v6 value semantics.                                                          | `boolean`                                               | `-`                                                    |
| `defaultChecked`    | Initial checked state for uncontrolled usage.                                                                   | `boolean`                                               | `false`                                                |
| `defaultValue`      | Alias for `defaultChecked`, matching antd v6 value semantics.                                                   | `boolean`                                               | `false`                                                |
| `disabled`          | Disables user interaction.                                                                                      | `boolean`                                               | `false`                                                |
| `loading`           | Displays loading state and disables interaction.                                                                | `boolean`                                               | `false`                                                |
| `size`              | Switch size. `medium` is supported for antd v6 compatibility; `middle` follows the shared Solid component size. | `'small' \| 'medium' \| 'middle'`                       | ConfigProvider `componentSize` (`'middle'` by default) |
| `checkedChildren`   | Content displayed when checked.                                                                                 | `JSX.Element`                                           | `-`                                                    |
| `unCheckedChildren` | Content displayed when unchecked.                                                                               | `JSX.Element`                                           | `-`                                                    |
| `class`             | Additional root class. Solid-style replacement for React `className`.                                           | `string`                                                | `-`                                                    |
| `style`             | Additional root inline style.                                                                                   | `JSX.CSSProperties`                                     | `-`                                                    |
| `classNames`        | Semantic class names for `root`, `content`, and `indicator`.                                                    | `Partial<Record<SwitchSemanticDOM, string>>`            | `-`                                                    |
| `styles`            | Semantic inline styles for `root`, `content`, and `indicator`.                                                  | `Partial<Record<SwitchSemanticDOM, JSX.CSSProperties>>` | `-`                                                    |
| `prefixCls`         | Custom CSS class prefix.                                                                                        | `string`                                                | `-`                                                    |
| `ref`               | Exposes focus, blur, and native button access.                                                                  | `SwitchRef \| (ref: SwitchRef) => void`                 | `-`                                                    |
| `onChange`          | Called with the next checked state after user interaction.                                                      | `(checked: boolean, event: MouseEvent) => void`         | `-`                                                    |
| `onClick`           | Called when the switch is clicked with the next checked state.                                                  | `(checked: boolean, event: MouseEvent) => void`         | `-`                                                    |

## Methods

| Name      | Description                           |
| --------- | ------------------------------------- |
| `focus()` | Focuses the underlying switch button. |
| `blur()`  | Blurs the underlying switch button.   |

## Semantic DOM

Use these keys with `classNames` and `styles` to customize stable Switch parts.

| Key         | Description                         |
| ----------- | ----------------------------------- |
| `root`      | Root switch button.                 |
| `content`   | Text/content area inside the track. |
| `indicator` | Movable switch handle.              |

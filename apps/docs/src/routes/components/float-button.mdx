# FloatButton

FloatButton renders floating quick actions and a back-to-top affordance.

export const floatButtonBoxStyle = (token, options = {}) => ({
position: 'relative',
height: options.height ?? '160px',
overflow: options.overflow ?? 'hidden',
border: `1px dashed ${token.colorBorder}`,
'border-radius': '8px',
background: token.colorFillAlter,
...(options.padding ? { padding: options.padding } : {}),
})

## API

### FloatButton

| Property      | Description                                                              | Type                                                                                                      | Default                             |
| ------------- | ------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------- | ----------------------------------- |
| `type`        | Float button visual type.                                                | `'default' \| 'primary'`                                                                                  | `'default'`                         |
| `shape`       | Float button shape.                                                      | `'circle' \| 'square'`                                                                                    | `'circle'`                          |
| `icon`        | Icon content rendered inside the button.                                 | `JSX.Element`                                                                                             | default icon when no content is set |
| `content`     | Text and other content rendered in the button.                           | `JSX.Element`                                                                                             | `-`                                 |
| `description` | Deprecated. Use `content` instead.                                       | `JSX.Element`                                                                                             | `-`                                 |
| `tooltip`     | Tooltip content or Tooltip props. String values also set native `title`. | `JSX.Element \| Omit<TooltipProps, 'children'>`                                                           | `-`                                 |
| `href`        | Renders the float button as a link when provided.                        | `string`                                                                                                  | `-`                                 |
| `target`      | Anchor target when href is provided.                                     | `string`                                                                                                  | `-`                                 |
| `badge`       | Badge props attached to the button.                                      | `Omit<BadgeProps, 'status' \| 'text' \| 'title' \| 'children'>`                                           | `-`                                 |
| `disabled`    | Whether the button is disabled.                                          | `boolean`                                                                                                 | `false`                             |
| `htmlType`    | Original HTML button type.                                               | `'button' \| 'submit' \| 'reset'`                                                                         | `'button'`                          |
| `classNames`  | Semantic class names for internal nodes.                                 | `Record<'root' \| 'icon' \| 'content', string> \| (info) => Record<string, string>`                       | `-`                                 |
| `styles`      | Semantic styles for internal nodes.                                      | `Record<'root' \| 'icon' \| 'content', JSX.CSSProperties> \| (info) => Record<string, JSX.CSSProperties>` | `-`                                 |
| `class`       | Root class name.                                                         | `string`                                                                                                  | `-`                                 |
| `style`       | Root inline style.                                                       | `JSX.CSSProperties \| string`                                                                             | `-`                                 |
| `onClick`     | Click callback.                                                          | `(event: MouseEvent) => void`                                                                             | `-`                                 |

### FloatButton.Group

| Property       | Description                                       | Type                                                                                                                                                              | Default    |
| -------------- | ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| `shape`        | Shared shape applied to buttons in the group.     | `'circle' \| 'square'`                                                                                                                                            | `'circle'` |
| `trigger`      | Action that opens or closes menu mode.            | `'click' \| 'hover'`                                                                                                                                              | `-`        |
| `open`         | Controlled menu open state. Use with `trigger`.   | `boolean`                                                                                                                                                         | `-`        |
| `onOpenChange` | Callback executed when menu open state changes.   | `(open: boolean) => void`                                                                                                                                         | `-`        |
| `placement`    | Menu placement relative to the trigger button.    | `'top' \| 'left' \| 'right' \| 'bottom'`                                                                                                                          | `'top'`    |
| `closeIcon`    | Close icon used by menu mode.                     | `JSX.Element`                                                                                                                                                     | close icon |
| `children`     | FloatButton items rendered in the group.          | `JSX.Element`                                                                                                                                                     | `-`        |
| `classNames`   | Semantic class names for group and trigger nodes. | `Record<'root' \| 'list' \| 'item' \| 'itemIcon' \| 'itemContent' \| 'trigger' \| 'triggerIcon' \| 'triggerContent', string> \| (info) => Record<string, string>` | `-`        |
| `styles`       | Semantic styles for group and trigger nodes.      | `Record<string, JSX.CSSProperties> \| (info) => Record<string, JSX.CSSProperties>`                                                                                | `-`        |

Menu mode also accepts FloatButton trigger props such as `type`, `icon`, `content`, `tooltip`, `badge`, `disabled`, `htmlType`, and `onClick`.

### FloatButton.BackTop

| Property           | Description                                             | Type                                               | Default  |
| ------------------ | ------------------------------------------------------- | -------------------------------------------------- | -------- |
| `visibilityHeight` | Scroll threshold before the back-to-top button appears. | `number`                                           | `400`    |
| `target`           | Function returning the scroll container.                | `() => Window \| HTMLElement \| undefined \| null` | `window` |
| `duration`         | Time to return to top in milliseconds.                  | `number`                                           | `450`    |
| `icon`             | Custom back-to-top icon.                                | `JSX.Element`                                      | top icon |
| `children`         | Deprecated custom back-to-top icon content.             | `JSX.Element`                                      | `-`      |

BackTop also accepts FloatButton props such as `type`, `shape`, `content`, `tooltip`, `badge`, `disabled`, `classNames`, `styles`, `class`, `style`, and `onClick`.

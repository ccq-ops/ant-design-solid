## API

Common HTML attributes are supported on the root element. `href` renders `Tag` as an anchor; otherwise it renders as a `span`.

### Tag

| Property        | Description                                                                                | Type                                                                                                                                                                                                            | Default                               |
| --------------- | ------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------- |
| `children`      | Tag content.                                                                               | `JSX.Element`                                                                                                                                                                                                   | `-`                                   |
| `color`         | Preset color, status color, or custom CSS color.                                           | `'blue' \| 'purple' \| 'cyan' \| 'green' \| 'magenta' \| 'pink' \| 'red' \| 'orange' \| 'yellow' \| 'volcano' \| 'geekblue' \| 'lime' \| 'gold' \| 'success' \| 'processing' \| 'warning' \| 'error' \| string` | `-`                                   |
| `variant`       | Visual variant.                                                                            | `'filled' \| 'solid' \| 'outlined'`                                                                                                                                                                             | `'outlined'`                          |
| `icon`          | Icon before tag content.                                                                   | `JSX.Element`                                                                                                                                                                                                   | `-`                                   |
| `closable`      | Shows a close button. Object form can provide a close icon.                                | `boolean \| { closeIcon?: JSX.Element }`                                                                                                                                                                        | `false`                               |
| `closeIcon`     | Custom close icon. `false` or `null` hides it.                                             | `JSX.Element \| false \| null`                                                                                                                                                                                  | `<CloseOutlined />`                   |
| `onClose`       | Callback fired before the tag is hidden. Call `event.preventDefault()` to keep it visible. | `(event: MouseEvent) => void`                                                                                                                                                                                   | `-`                                   |
| `href`          | Render as an anchor and navigate to this URL.                                              | `string`                                                                                                                                                                                                        | `-`                                   |
| `target`        | Anchor target when `href` is set.                                                          | `string`                                                                                                                                                                                                        | `-`                                   |
| `disabled`      | Disable click, close, and anchor navigation.                                               | `boolean`                                                                                                                                                                                                       | `false`                               |
| `prefixCls`     | Custom component class prefix.                                                             | `string`                                                                                                                                                                                                        | ConfigProvider `prefixCls` + `'-tag'` |
| `rootClassName` | Extra class name on the root element.                                                      | `string`                                                                                                                                                                                                        | `-`                                   |
| `class`         | Extra class name on the root element.                                                      | `string`                                                                                                                                                                                                        | `-`                                   |
| `style`         | Inline style for the root element.                                                         | `JSX.CSSProperties \| string`                                                                                                                                                                                   | `-`                                   |
| `classNames`    | Semantic class names for stable internal slots.                                            | `Partial<Record<'root' \| 'icon' \| 'content' \| 'close', string>>`                                                                                                                                             | `-`                                   |
| `styles`        | Semantic inline styles for stable internal slots.                                          | `Partial<Record<'root' \| 'icon' \| 'content' \| 'close', JSX.CSSProperties>>`                                                                                                                                  | `-`                                   |
| `bordered`      | Deprecated borderless API. Use `variant="filled"` instead.                                 | `boolean`                                                                                                                                                                                                       | `true`                                |

### Tag.CheckableTag

`Tag.CheckableTag` is controlled. Pass `checked` and update it from `onChange`.

| Property    | Description                                       | Type                             | Default                               |
| ----------- | ------------------------------------------------- | -------------------------------- | ------------------------------------- |
| `checked`   | Checked status.                                   | `boolean`                        | `false`                               |
| `children`  | Tag content.                                      | `JSX.Element`                    | `-`                                   |
| `icon`      | Icon before tag content.                          | `JSX.Element`                    | `-`                                   |
| `disabled`  | Disable check and click interaction.              | `boolean`                        | `false`                               |
| `prefixCls` | Custom component class prefix.                    | `string`                         | ConfigProvider `prefixCls` + `'-tag'` |
| `class`     | Extra class name on the root element.             | `string`                         | `-`                                   |
| `style`     | Inline style for the root element.                | `JSX.CSSProperties \| string`    | `-`                                   |
| `onChange`  | Callback fired with the next checked state.       | `(checked: boolean) => void`     | `-`                                   |
| `onClick`   | Callback fired after a non-disabled click.        | `(event: MouseEvent) => void`    | `-`                                   |
| `onKeyDown` | Callback fired before built-in keyboard handling. | `(event: KeyboardEvent) => void` | `-`                                   |

### Tag.CheckableTagGroup

`Tag.CheckableTagGroup` supports uncontrolled usage with `defaultValue` and controlled usage with `value`.

| Property        | Description                                                                                         | Type                                                                                                                                        | Default                               |
| --------------- | --------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------- |
| `options`       | Option list. Primitive options use the value as the label. Object options support item class/style. | `Array<{ value: string \| number; label: JSX.Element; className?: string; class?: string; style?: JSX.CSSProperties } \| string \| number>` | `[]`                                  |
| `value`         | Controlled selected value or values.                                                                | `string \| number \| Array<string \| number> \| null`                                                                                       | `-`                                   |
| `defaultValue`  | Initial selected value or values.                                                                   | `string \| number \| Array<string \| number> \| null`                                                                                       | `-`                                   |
| `multiple`      | Enable multiple selection.                                                                          | `boolean`                                                                                                                                   | `false`                               |
| `disabled`      | Disable all options.                                                                                | `boolean`                                                                                                                                   | `false`                               |
| `prefixCls`     | Custom component class prefix.                                                                      | `string`                                                                                                                                    | ConfigProvider `prefixCls` + `'-tag'` |
| `rootClassName` | Extra class name on the group root.                                                                 | `string`                                                                                                                                    | `-`                                   |
| `class`         | Extra class name on the group root.                                                                 | `string`                                                                                                                                    | `-`                                   |
| `style`         | Inline style for the group root.                                                                    | `JSX.CSSProperties \| string`                                                                                                               | `-`                                   |
| `role`          | ARIA role for the group root.                                                                       | `string`                                                                                                                                    | `'group'`                             |
| `onChange`      | Callback fired with the next value.                                                                 | `(value: string \| number \| Array<string \| number> \| null) => void`                                                                      | `-`                                   |
| `classNames`    | Semantic class names for stable internal slots.                                                     | `Partial<Record<'root' \| 'item', string>>`                                                                                                 | `-`                                   |
| `styles`        | Semantic inline styles for stable internal slots.                                                   | `Partial<Record<'root' \| 'item', JSX.CSSProperties>>`                                                                                      | `-`                                   |

## Semantic DOM

### Tag

| Slot      | Description                                  |
| --------- | -------------------------------------------- |
| `root`    | Root `span` or `a` element.                  |
| `icon`    | Icon wrapper rendered before the content.    |
| `content` | Content wrapper rendered when `icon` exists. |
| `close`   | Close button element.                        |

### Tag.CheckableTagGroup

| Slot   | Description                          |
| ------ | ------------------------------------ |
| `root` | Group root element.                  |
| `item` | Class or style applied to each item. |

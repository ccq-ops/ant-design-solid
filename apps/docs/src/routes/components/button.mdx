## API

Button accepts native button attributes. When `href` is provided, it renders as an anchor-like button and anchor attributes such as `target`, `rel`, `download`, and `referrerPolicy` are passed to the link.

Solid code uses `class` instead of React `className`. The antd `rootClassName` API is exposed as `rootClass`.

| Property          | Description                                                                 | Type                                                                                                      | Default             |
| ----------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- | ------------------- |
| `autoInsertSpace` | Automatically inserts a space between two Chinese characters.               | `boolean`                                                                                                 | `true`              |
| `block`           | Makes the button fit the width of its parent.                               | `boolean`                                                                                                 | `false`             |
| `classNames`      | Semantic class names for internal slots. Supports object or function.       | `Record<'root' \| 'icon' \| 'content', string> \| (info) => Record<string, string>`                       | `-`                 |
| `color`           | Sets the button color. Preset colors follow the antd color names.           | `'default' \| 'primary' \| 'danger' \| PresetColor`                                                       | `derived from type` |
| `danger`          | Applies danger styling. Explicit color takes priority when both are set.    | `boolean`                                                                                                 | `false`             |
| `disabled`        | Disables user interaction.                                                  | `boolean`                                                                                                 | `false`             |
| `ghost`           | Makes the button background transparent.                                    | `boolean`                                                                                                 | `false`             |
| `href`            | Renders the button as an anchor link when provided.                         | `string`                                                                                                  | `-`                 |
| `htmlType`        | Sets the native button type when href is not provided.                      | `'button' \| 'submit' \| 'reset'`                                                                         | `'button'`          |
| `icon`            | Renders an icon next to the button content.                                 | `JSX.Element`                                                                                             | `-`                 |
| `iconPlacement`   | Controls whether the icon appears before or after the content.              | `'start' \| 'end'`                                                                                        | `'start'`           |
| `iconPosition`    | Legacy name for iconPlacement. Prefer iconPlacement for new code.           | `'start' \| 'end'`                                                                                        | `'start'`           |
| `loading`         | Shows a loading icon and prevents clicks. Supports delay and custom icon.   | `boolean \| { delay?: number; icon?: JSX.Element }`                                                       | `false`             |
| `onClick`         | Called when the button is clicked and not disabled or loading.              | `(event: MouseEvent) => void`                                                                             | `-`                 |
| `prefixCls`       | Overrides the component class prefix.                                       | `string`                                                                                                  | `ads-btn`           |
| `rootClass`       | Extra class name on the root element. Solid equivalent of `rootClassName`.  | `string`                                                                                                  | `-`                 |
| `shape`           | Sets the button shape.                                                      | `'default' \| 'circle' \| 'round'`                                                                        | `'default'`         |
| `size`            | Sets the button size. `middle` is accepted for compatibility.               | `'small' \| 'medium' \| 'middle' \| 'large'`                                                              | `'medium'`          |
| `styles`          | Semantic inline styles for internal slots. Supports object or function.     | `Record<'root' \| 'icon' \| 'content', JSX.CSSProperties> \| (info) => Record<string, JSX.CSSProperties>` | `-`                 |
| `target`          | Same as target attribute of `a`, works when `href` is specified.            | `string`                                                                                                  | `-`                 |
| `type`            | Syntactic sugar for common button styles.                                   | `'default' \| 'primary' \| 'dashed' \| 'text' \| 'link'`                                                  | `'default'`         |
| `variant`         | Sets the visual variant. Explicit variant takes priority over type mapping. | `'outlined' \| 'dashed' \| 'solid' \| 'filled' \| 'text' \| 'link'`                                       | `derived from type` |

### Semantic DOM

| Slot      | Description                    |
| --------- | ------------------------------ |
| `root`    | Root button or anchor element. |
| `icon`    | Icon wrapper.                  |
| `content` | Text/content wrapper.          |

### PresetColor

`PresetColor` is one of `blue | purple | cyan | green | magenta | pink | red | orange | yellow | volcano | geekblue | lime | gold`.

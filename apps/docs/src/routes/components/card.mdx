## API

| Property              | Description                                             | Type                                                                                                   | Default      |
| --------------------- | ------------------------------------------------------- | ------------------------------------------------------------------------------------------------------ | ------------ |
| `actions`             | Footer action items.                                    | `JSX.Element[]`                                                                                        | `-`          |
| `activeTabKey`        | Current tab key.                                        | `string`                                                                                               | `-`          |
| `bordered`            | Whether to render a card border. Prefer `variant`.      | `boolean`                                                                                              | `true`       |
| `classNames`          | Customize class for each semantic structure.            | `Record<CardSemanticSlot, string> \| (info) => Record<CardSemanticSlot, string>`                       | `-`          |
| `cover`               | Cover content rendered before the card body.            | `JSX.Element`                                                                                          | `-`          |
| `defaultActiveTabKey` | Initial tab key when `activeTabKey` is not set.         | `string`                                                                                               | First tab    |
| `extra`               | Extra content rendered on the right side of the header. | `JSX.Element`                                                                                          | `-`          |
| `hoverable`           | Enables hover elevation styling.                        | `boolean`                                                                                              | `false`      |
| `loading`             | Shows a skeleton placeholder in the body.               | `boolean`                                                                                              | `false`      |
| `onTabChange`         | Callback when tab is switched.                          | `(key: string) => void`                                                                                | `-`          |
| `prefixCls`           | Custom CSS class prefix.                                | `string`                                                                                               | `ads-card`   |
| `rootClass`           | Extra class on the root element.                        | `string`                                                                                               | `-`          |
| `size`                | Card size.                                              | `'medium' \| 'small' \| 'default'`                                                                     | `'medium'`   |
| `styles`              | Customize inline style for each semantic structure.     | `Record<CardSemanticSlot, JSX.CSSProperties> \| (info) => Record<CardSemanticSlot, JSX.CSSProperties>` | `-`          |
| `tabBarExtraContent`  | Extra content in tab bar.                               | `JSX.Element \| { left?: JSX.Element; right?: JSX.Element }`                                           | `-`          |
| `tabList`             | List of tab headers.                                    | `CardTabListType[]`                                                                                    | `-`          |
| `tabProps`            | Props passed to the inner Tabs component.               | `Partial<TabsProps>`                                                                                   | `-`          |
| `title`               | Card title rendered in the header.                      | `JSX.Element`                                                                                          | `-`          |
| `type`                | Card style type.                                        | `'inner'`                                                                                              | `-`          |
| `variant`             | Card variant.                                           | `'outlined' \| 'borderless'`                                                                           | `'outlined'` |

### Semantic DOM

`Card` supports semantic slots: `root`, `header`, `body`, `extra`, `title`, `actions`, `cover`.

### Card.Grid

| Property    | Description                          | Type      | Default |
| ----------- | ------------------------------------ | --------- | ------- |
| `hoverable` | Lift up when hovering the grid item. | `boolean` | `true`  |

`Card.Grid` also supports native `div` attributes such as `class`, `style`, and `data-*`.

### Card.Meta

| Property      | Description                                  | Type                                                                                                           | Default    |
| ------------- | -------------------------------------------- | -------------------------------------------------------------------------------------------------------------- | ---------- |
| `avatar`      | Avatar or icon.                              | `JSX.Element`                                                                                                  | `-`        |
| `classNames`  | Customize class for each semantic structure. | `Record<CardMetaSemanticSlot, string> \| (info) => Record<CardMetaSemanticSlot, string>`                       | `-`        |
| `description` | Description content.                         | `JSX.Element`                                                                                                  | `-`        |
| `prefixCls`   | Custom CSS class prefix.                     | `string`                                                                                                       | `ads-card` |
| `styles`      | Customize inline style for each structure.   | `Record<CardMetaSemanticSlot, JSX.CSSProperties> \| (info) => Record<CardMetaSemanticSlot, JSX.CSSProperties>` | `-`        |
| `title`       | Title content.                               | `JSX.Element`                                                                                                  | `-`        |

`Card.Meta` supports semantic slots: `root`, `section`, `avatar`, `title`, `description`.

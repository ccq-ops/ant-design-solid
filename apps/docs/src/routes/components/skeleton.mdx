### Loaded content

:::preview

<SkeletonDemo007 />

export const SkeletonDemo007 = (() => {
  const Demo7 = function () {
      const [loaded, setLoaded] = createSignal(false);
      return (<Space direction="vertical" class="w-full">
        <button type="button" onClick={() => setLoaded((value) => !value)}>
          Toggle loaded
        </button>
        <Skeleton loading={!loaded()} active>
          <Card title="Loaded">Content is ready.</Card>
        </Skeleton>
      </Space>);
  };

return Demo7
})()

## API

### Skeleton

| Property        | Description                                         | Type                                | Default |
| --------------- | --------------------------------------------------- | ----------------------------------- | ------- |
| `active`        | Shows animated loading effect.                      | `boolean`                           | `false` |
| `avatar`        | Shows avatar placeholder or configures it.          | `boolean \| SkeletonAvatarProps`    | `false` |
| `children`      | Content rendered when loading is false.             | `JSX.Element`                       | `-`     |
| `class`         | Root class name.                                    | `string`                            | `-`     |
| `classNames`    | Semantic DOM class names.                           | `SkeletonSemanticClassNames`        | `-`     |
| `loading`       | Controls whether skeleton or children are rendered. | `boolean`                           | `true`  |
| `paragraph`     | Shows paragraph placeholders or configures them.    | `boolean \| SkeletonParagraphProps` | `true`  |
| `prefixCls`     | Custom component class prefix.                      | `string`                            | `-`     |
| `rootClassName` | Extra class name on the root element.               | `string`                            | `-`     |
| `round`         | Uses rounded placeholder corners.                   | `boolean`                           | `false` |
| `style`         | Root inline style.                                  | `JSX.CSSProperties \| string`       | `-`     |
| `styles`        | Semantic DOM inline styles.                         | `SkeletonSemanticStyles`            | `-`     |
| `title`         | Shows title placeholder or configures it.           | `boolean \| SkeletonTitleProps`     | `true`  |

### Semantic DOM

`Skeleton` supports these semantic slots for `classNames` and `styles`:

| Slot        | Description                  |
| ----------- | ---------------------------- |
| `root`      | Root skeleton wrapper.       |
| `header`    | Avatar wrapper.              |
| `section`   | Title and paragraph wrapper. |
| `avatar`    | Avatar placeholder.          |
| `title`     | Title placeholder.           |
| `paragraph` | Paragraph list wrapper.      |

Element components (`Skeleton.Button`, `Skeleton.Avatar`, `Skeleton.Input`, `Skeleton.Image`, `Skeleton.Node`) support `root` and `content` semantic slots.

### Skeleton avatar

| Property | Description               | Type                                                    | Default    |
| -------- | ------------------------- | ------------------------------------------------------- | ---------- |
| `size`   | Avatar placeholder size.  | `'small' \| 'medium' \| 'default' \| 'large' \| number` | `-`        |
| `shape`  | Avatar placeholder shape. | `'circle' \| 'square'`                                  | `'circle'` |

### Skeleton title

| Property | Description              | Type               | Default |
| -------- | ------------------------ | ------------------ | ------- |
| `width`  | Title placeholder width. | `number \| string` | `-`     |

### Skeleton paragraph

| Property | Description                                                            | Type                               | Default |
| -------- | ---------------------------------------------------------------------- | ---------------------------------- | ------- |
| `rows`   | Number of paragraph rows.                                              | `number`                           | `-`     |
| `width`  | Paragraph row width or widths. A single width applies to the last row. | `SkeletonWidth \| SkeletonWidth[]` | `-`     |

### Skeleton.Button

| Property        | Description                                  | Type                                                        | Default  |
| --------------- | -------------------------------------------- | ----------------------------------------------------------- | -------- |
| `active`        | Shows animated loading effect.               | `boolean`                                                   | `false`  |
| `block`         | Makes the placeholder fill its parent width. | `boolean`                                                   | `false`  |
| `class`         | Root class name.                             | `string`                                                    | `-`      |
| `classNames`    | Semantic DOM class names.                    | `{ root?: string; content?: string }`                       | `-`      |
| `prefixCls`     | Custom component class prefix.               | `string`                                                    | `-`      |
| `rootClassName` | Extra class name on the root element.        | `string`                                                    | `-`      |
| `shape`         | Button placeholder shape.                    | `'circle' \| 'round' \| 'square' \| 'default'`              | `-`      |
| `size`          | Button placeholder size.                     | `'small' \| 'medium' \| 'default' \| 'large' \| number`     | `medium` |
| `style`         | Content inline style.                        | `JSX.CSSProperties \| string`                               | `-`      |
| `styles`        | Semantic DOM inline styles.                  | `{ root?: JSX.CSSProperties; content?: JSX.CSSProperties }` | `-`      |

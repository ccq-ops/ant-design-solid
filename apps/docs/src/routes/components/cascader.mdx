# Cascader

Select from a nested set of related options with one menu column per level.

### Basic

:::preview

### Load Data

:::preview

<CascaderDemo008 />

export const CascaderDemo008 = (() => {
  const Demo8 = function () {
      const [options, setOptions] = createSignal([
          { value: 'zhejiang', label: 'Zhejiang', isLeaf: false },
          { value: 'jiangsu', label: 'Jiangsu', isLeaf: false },
      ]);
      const loadData = async (selectedOptions) => {
          const target = selectedOptions.at(-1);
          await new Promise((resolve) => setTimeout(resolve, 500));
          setOptions((current) => current.map((option) => option.value === target?.value
              ? {
                  ...option,
                  children: [
                      { label: `${option.label} Dynamic 1`, value: `${option.value}-1` },
                      { label: `${option.label} Dynamic 2`, value: `${option.value}-2` },
                  ],
              }
              : option));
      };
      return <Cascader options={options()} loadData={loadData} placeholder="Please select"/>;
  };

return Demo8
})()

```tsx
import { createSignal } from 'solid-js'
import { Cascader } from '@ant-design-solid/core'

const Demo8 = function () {
  const [options, setOptions] = createSignal([
    { value: 'zhejiang', label: 'Zhejiang', isLeaf: false },
    { value: 'jiangsu', label: 'Jiangsu', isLeaf: false },
  ])

  const loadData = async (selectedOptions) => {
    const target = selectedOptions.at(-1)
    await new Promise((resolve) => setTimeout(resolve, 500))
    setOptions((current) =>
      current.map((option) =>
        option.value === target?.value
          ? {
              ...option,
              children: [
                { label: `${option.label} Dynamic 1`, value: `${option.value}-1` },
                { label: `${option.label} Dynamic 2`, value: `${option.value}-2` },
              ],
            }
          : option,
      ),
    )
  }

  return <Cascader options={options()} loadData={loadData} placeholder="Please select" />
}

export default Demo8
```

## API

### Cascader

| Property               | Description                                                                          | Type                                                                                                                        | Default                                                    |
| ---------------------- | ------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| `ref`                  | Imperative focus and blur handle.                                                    | `CascaderRef \| { current?: CascaderRef } \| ((ref: CascaderRef) => void)`                                                  | `-`                                                        |
| `options`              | Nested option tree rendered by the cascader. Supports `fieldNames` mapped data.      | `CascaderOptionInput[]`                                                                                                     | `-`                                                        |
| `value`                | Controlled selected path values. Arrays of arrays are used by `multiple`.            | `OptionValue[] \| OptionValue[][]`                                                                                          | `-`                                                        |
| `defaultValue`         | Initial selected path values for uncontrolled usage.                                 | `OptionValue[] \| OptionValue[][]`                                                                                          | `-`                                                        |
| `open`                 | Controlled popup open state.                                                         | `boolean`                                                                                                                   | `-`                                                        |
| `defaultOpen`          | Initial popup open state.                                                            | `boolean`                                                                                                                   | `false`                                                    |
| `placeholder`          | Placeholder shown when no path is selected.                                          | `JSX.Element`                                                                                                               | `-`                                                        |
| `disabled`             | Disables opening and selection.                                                      | `boolean`                                                                                                                   | `false`                                                    |
| `allowClear`           | Shows a clear button when a value is selected.                                       | `boolean \| { clearIcon?: JSX.Element }`                                                                                    | `true`                                                     |
| `changeOnSelect`       | Allows selecting intermediate parent options.                                        | `boolean`                                                                                                                   | `false`                                                    |
| `expandIcon`           | Custom expand icon for parent menu items.                                            | `JSX.Element`                                                                                                               | `-`                                                        |
| `expandTrigger`        | Interaction used to expand child columns.                                            | `'click' \| 'hover'`                                                                                                        | `'click'`                                                  |
| `displayRender`        | Customizes selected path display.                                                    | `(labels: JSX.Element[], selectedOptions: CascaderOption[]) => JSX.Element`                                                 | `labels.join(' / ')`                                       |
| `fieldNames`           | Maps custom option field names.                                                      | `{ label?: string; value?: string; children?: string }`                                                                     | `{ label: 'label', value: 'value', children: 'children' }` |
| `getPopupContainer`    | Returns the element used to mount the popup portal.                                  | `(triggerNode?: HTMLElement) => HTMLElement \| ShadowRoot`                                                                  | `document.body`                                            |
| `showSearch`           | Enables path search or configures filtering, sorting, rendering, and search control. | `boolean \| CascaderShowSearch`                                                                                             | `false`                                                    |
| `searchValue`          | Controlled search text.                                                              | `string`                                                                                                                    | `-`                                                        |
| `onSearch`             | Called when search input changes.                                                    | `(search: string) => void`                                                                                                  | `-`                                                        |
| `loadData`             | Loads child options lazily for options with `isLeaf` set to false.                   | `(selectedOptions: CascaderOption[]) => void \| Promise<void>`                                                              | `-`                                                        |
| `loadingIcon`          | Custom loading indicator for lazy options.                                           | `JSX.Element`                                                                                                               | `-`                                                        |
| `notFoundContent`      | Content rendered when no option or search result is available.                       | `JSX.Element`                                                                                                               | `'Not Found'`                                              |
| `optionRender`         | Customizes dropdown option rendering.                                                | `(option: CascaderOption) => JSX.Element`                                                                                   | `-`                                                        |
| `popupRender`          | Wraps or replaces popup content.                                                     | `(originNode: JSX.Element) => JSX.Element`                                                                                  | `-`                                                        |
| `placement`            | Popup placement.                                                                     | `'bottomLeft' \| 'bottomRight' \| 'topLeft' \| 'topRight'`                                                                  | `'bottomLeft'`                                             |
| `multiple`             | Enables multiple path selection.                                                     | `boolean`                                                                                                                   | `false`                                                    |

### Cascader.Panel

`Cascader.Panel` renders the menu columns directly without selector chrome. It accepts the data and selection props used by the popup panel, including `options`, `fieldNames`, `value`, `defaultValue`, `multiple`, `changeOnSelect`, `expandTrigger`, `loadData`, `loadingIcon`, `expandIcon`, `notFoundContent`, `optionRender`, `classNames`, `styles`, and `onChange`.

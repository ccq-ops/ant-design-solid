## API

TimePicker values are `dayjs` objects. String values are accepted for compatibility, but callbacks emit `dayjs` values and formatted strings.

| Property              | Description                                                                    | Type                                                               | Default                   |
| --------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------ | ------------------------- |
| `value`               | Controlled time value.                                                         | `Dayjs \| string \| null`                                          | `-`                       |
| `defaultValue`        | Initial time value for uncontrolled usage.                                     | `Dayjs \| string`                                                  | `-`                       |
| `format`              | Time display and emitted string format.                                        | `string`                                                           | `'HH:mm:ss'`              |
| `placeholder`         | Placeholder text when no time is selected.                                     | `string`                                                           | `'Select time'`           |
| `disabled`            | Disables the picker.                                                           | `boolean`                                                          | `false`                   |
| `allowClear`          | Shows or customizes the clear button.                                          | `boolean \| { clearIcon?: JSX.Element }`                           | `true`                    |
| `open`                | Controlled popup open state.                                                   | `boolean`                                                          | `-`                       |
| `defaultOpen`         | Initial popup open state.                                                      | `boolean`                                                          | `false`                   |
| `hourStep`            | Hour option interval.                                                          | `number`                                                           | `1`                       |
| `minuteStep`          | Minute option interval.                                                        | `number`                                                           | `1`                       |
| `secondStep`          | Second option interval.                                                        | `number`                                                           | `1`                       |
| `hideDisabledOptions` | Hides disabled time options.                                                   | `boolean`                                                          | `false`                   |
| `changeOnScroll`      | Selects the nearest option when a column scrolls.                              | `boolean`                                                          | `false`                   |
| `disabledTime`        | Returns disabled hour, minute, or second values.                               | `(now: Dayjs) => DisabledTimeConfig`                               | `-`                       |
| `disabledHours`       | Returns disabled hour values.                                                  | `() => number[]`                                                   | `-`                       |
| `disabledMinutes`     | Returns disabled minute values for selected hour.                              | `(selectedHour: number) => number[]`                               | `-`                       |
| `disabledSeconds`     | Returns disabled second values.                                                | `(selectedHour: number, selectedMinute: number) => number[]`       | `-`                       |
| `placement`           | Popup placement.                                                               | `'bottomLeft' \| 'bottomRight' \| 'topLeft' \| 'topRight'`         | `'bottomLeft'`            |
| `inputReadOnly`       | Sets the internal input mirror to read-only.                                   | `boolean`                                                          | `false`                   |
| `prefix`              | Custom prefix content.                                                         | `JSX.Element`                                                      | `-`                       |
| `suffixIcon`          | Custom suffix icon.                                                            | `JSX.Element`                                                      | `<ClockCircleOutlined />` |
| `cellRender`          | Custom rendering function for time cells.                                      | `(current: number, info: TimePickerCellRenderInfo) => JSX.Element` | `-`                       |
| `renderExtraFooter`   | Renders extra footer content.                                                  | `() => JSX.Element`                                                | `-`                       |
| `needConfirm`         | Compatibility prop; single picker selections are confirmed with OK by default. | `boolean`                                                          | `true`                    |
| `showNow`             | Shows a Now button in the panel.                                               | `boolean`                                                          | `true`                    |
| `use12Hours`          | Uses a 12-hour display format by default.                                      | `boolean`                                                          | `false`                   |
| `size`                | Picker size class.                                                             | `'large' \| 'medium' \| 'small'`                                   | `-`                       |
| `status`              | Validation status class.                                                       | `'error' \| 'warning' \| 'success' \| 'validating'`                | `-`                       |
| `variant`             | Visual variant class.                                                          | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`           | `-`                       |
| `onChange`            | Called when selection changes.                                                 | `(time: Dayjs \| null, timeString: string) => void`                | `-`                       |
| `onOpenChange`        | Called when popup open state changes.                                          | `(open: boolean) => void`                                          | `-`                       |
| `classNames`          | Semantic class names.                                                          | `Partial<Record<TimePickerSemanticSlot, string>>`                  | `-`                       |
| `styles`              | Semantic inline styles.                                                        | `Partial<Record<TimePickerSemanticSlot, JSX.CSSProperties>>`       | `-`                       |
| `popupClassName`      | Legacy popup class name.                                                       | `string`                                                           | `-`                       |
| `dropdownClassName`   | Legacy popup class name alias.                                                 | `string`                                                           | `-`                       |
| `popupStyle`          | Legacy popup inline style.                                                     | `JSX.CSSProperties`                                                | `-`                       |
| `prefixCls`           | Custom CSS class prefix.                                                       | `string`                                                           | `-`                       |
| `zIndex`              | Custom popup z-index.                                                          | `number`                                                           | `-`                       |
| `getPopupContainer`   | Returns popup portal container.                                                | `(triggerNode?: HTMLElement) => HTMLElement`                       | `-`                       |

### DisabledTime

```ts
interface DisabledTimeConfig {
  disabledHours?: () => number[]
  disabledMinutes?: (selectedHour: number) => number[]
  disabledSeconds?: (selectedHour: number, selectedMinute: number) => number[]
  disabledMilliseconds?: (
    selectedHour: number,
    selectedMinute: number,
    selectedSecond: number,
  ) => number[]
}

type DisabledTime = (now: Dayjs) => DisabledTimeConfig
```

### CellRender Info

| Property     | Description                          | Type                                           |
| ------------ | ------------------------------------ | ---------------------------------------------- |
| `originNode` | Default cell node.                   | `JSX.Element`                                  |
| `today`      | Current dayjs value.                 | `Dayjs`                                        |
| `range`      | Range side when used in RangePicker. | `'start' \| 'end'`                             |
| `subType`    | Time column kind.                    | `'hour' \| 'minute' \| 'second' \| 'meridiem'` |

### Semantic DOM

| Slot       | Description                     |
| ---------- | ------------------------------- |
| `root`     | Root wrapper.                   |
| `selector` | Clickable selector surface.     |
| `input`    | Hidden input mirror.            |
| `clear`    | Clear button.                   |
| `suffix`   | Suffix icon wrapper.            |
| `popup`    | Popup container.                |
| `panel`    | Time columns panel.             |
| `column`   | Hour, minute, or second column. |
| `cell`     | Time option cell.               |
| `footer`   | Popup footer.                   |

### Methods

| Name      | Description   |
| --------- | ------------- |
| `focus()` | Focus picker. |
| `blur()`  | Blur picker.  |

## RangePicker API

`TimePicker.RangePicker` supports the common TimePicker props above and the following range-specific props.

| Property       | Description                       | Type                                                         | Default                      |
| -------------- | --------------------------------- | ------------------------------------------------------------ | ---------------------------- |
| `value`        | Controlled time range.            | `[Dayjs \| null, Dayjs \| null] \| null`                     | `-`                          |
| `defaultValue` | Initial time range.               | `[Dayjs \| null, Dayjs \| null] \| null`                     | `-`                          |
| `placeholder`  | Range placeholders.               | `[string, string]`                                           | `['Start time', 'End time']` |
| `disabledTime` | Returns disabled values per side. | `(now: Dayjs, type: 'start' \| 'end') => DisabledTimeConfig` | `-`                          |
| `order`        | Orders start and end time values. | `boolean`                                                    | `true`                       |
| `onChange`     | Called when either side changes.  | `(times, timeStrings: [string, string]) => void`             | `-`                          |

# Tour

Tour guides users through features step by step.

### Basic

:::preview

## API

### Tour

| Property                | Description                                                   | Type                                                                                 | Default      |
| ----------------------- | ------------------------------------------------------------- | ------------------------------------------------------------------------------------ | ------------ |
| `open`                  | Controlled tour open state.                                   | `boolean`                                                                            | `-`          |
| `defaultOpen`           | Initial tour open state.                                      | `boolean`                                                                            | `false`      |
| `current`               | Controlled current step index.                                | `number`                                                                             | `-`          |
| `defaultCurrent`        | Initial current step index.                                   | `number`                                                                             | `0`          |
| `steps`                 | Tour step definitions.                                        | `TourStep[]`                                                                         | `-`          |
| `placement`             | Default step placement.                                       | `TourPlacement`                                                                      | `'bottom'`   |
| `mask`                  | Whether to show the mask, or custom mask style and color.     | `boolean \| { style?: JSX.CSSProperties; color?: string }`                           | `true`       |
| `arrow`                 | Whether to show the arrow, or point it at target center.      | `boolean \| { pointAtCenter: boolean }`                                              | `true`       |
| `closeIcon`             | Custom close icon.                                            | `JSX.Element`                                                                        | `-`          |
| `indicatorsRender`      | Custom step indicator renderer.                               | `(current: number, total: number) => JSX.Element`                                    | `-`          |
| `actionsRender`         | Custom action area renderer.                                  | `(originNode: JSX.Element, info: { current: number; total: number }) => JSX.Element` | `-`          |
| `disabledInteraction`   | Disables interaction with the highlighted area.               | `boolean`                                                                            | `false`      |
| `keyboard`              | Enables Escape key closing.                                   | `boolean`                                                                            | `true`       |
| `scrollIntoViewOptions` | Scroll target into view before positioning.                   | `boolean \| ScrollIntoViewOptions`                                                   | `true`       |
| `type`                  | Visual type of the guide card.                                | `'default' \| 'primary'`                                                             | `'default'`  |
| `zIndex`                | Custom overlay z-index.                                       | `number`                                                                             | `-`          |
| `getPopupContainer`     | Returns portal container.                                     | `(triggerNode?: HTMLElement) => HTMLElement`                                         | `-`          |
| `gap`                   | Gap around target mask and popup.                             | `number \| { offset?: number \| [number, number]; radius?: number }`                 | `8`          |
| `nextText`              | Default next button text.                                     | `JSX.Element`                                                                        | `'Next'`     |
| `prevText`              | Default previous button text.                                 | `JSX.Element`                                                                        | `'Previous'` |
| `finishText`            | Default finish button text.                                   | `JSX.Element`                                                                        | `'Finish'`   |
| `onClose`               | Called when the tour closes.                                  | `(current: number) => void`                                                          | `-`          |
| `onChange`              | Called when the current step changes.                         | `(current: number) => void`                                                          | `-`          |
| `onFinish`              | Called when the finish button is clicked.                     | `() => void`                                                                         | `-`          |
| `class`                 | Custom class for the guide card.                              | `string`                                                                             | `-`          |
| `classList`             | Solid classList for the guide card.                           | `Record<string, boolean \| undefined>`                                               | `-`          |
| `style`                 | Custom style for the guide card.                              | `JSX.CSSProperties`                                                                  | `-`          |
| `classNames`            | Semantic class names, or a function receiving merged props.   | `TourSemanticClassNames \| (info: { props: TourProps }) => TourSemanticClassNames`   | `-`          |
| `styles`                | Semantic inline styles, or a function receiving merged props. | `TourSemanticStyles \| (info: { props: TourProps }) => TourSemanticStyles`           | `-`          |

### TourStep

| Property                | Description                                    | Type                                                                                 | Default |
| ----------------------- | ---------------------------------------------- | ------------------------------------------------------------------------------------ | ------- |
| `title`                 | Step title.                                    | `JSX.Element`                                                                        | `-`     |
| `description`           | Step description.                              | `JSX.Element`                                                                        | `-`     |
| `target`                | Step target element or resolver.               | `HTMLElement \| null \| (() => HTMLElement \| null \| undefined)`                    | `-`     |
| `placement`             | Step placement override.                       | `TourPlacement`                                                                      | `-`     |
| `mask`                  | Step mask override.                            | `boolean \| { style?: JSX.CSSProperties; color?: string }`                           | `-`     |
| `arrow`                 | Step arrow override.                           | `boolean \| { pointAtCenter: boolean }`                                              | `-`     |
| `closeIcon`             | Step close icon override.                      | `JSX.Element`                                                                        | `-`     |
| `cover`                 | Step cover content, such as an image or video. | `JSX.Element`                                                                        | `-`     |
| `type`                  | Step visual type override.                     | `'default' \| 'primary'`                                                             | `-`     |
| `scrollIntoViewOptions` | Step scroll behavior override.                 | `boolean \| ScrollIntoViewOptions`                                                   | `-`     |
| `indicatorsRender`      | Step indicator renderer override.              | `(current: number, total: number) => JSX.Element`                                    | `-`     |
| `actionsRender`         | Step action area renderer override.            | `(originNode: JSX.Element, info: { current: number; total: number }) => JSX.Element` | `-`     |
| `onClose`               | Called before the tour closes from this step.  | `() => void`                                                                         | `-`     |
| `nextButtonProps`       | Props merged into the next or finish button.   | `ButtonProps`                                                                        | `-`     |
| `prevButtonProps`       | Props merged into the previous button.         | `ButtonProps`                                                                        | `-`     |
| `class`                 | Custom class for this step card.               | `string`                                                                             | `-`     |
| `style`                 | Custom style for this step card.               | `JSX.CSSProperties`                                                                  | `-`     |
| `classNames`            | Semantic class names for this step.            | `TourSemanticClassNames`                                                             | `-`     |
| `styles`                | Semantic inline styles for this step.          | `TourSemanticStyles`                                                                 | `-`     |

### Semantic DOM

| Key           | Description                      |
| ------------- | -------------------------------- |
| `root`        | Guide card root.                 |
| `mask`        | Full-screen mask or target mask. |
| `section`     | Inner content section.           |
| `cover`       | Cover content wrapper.           |
| `close`       | Close button.                    |
| `header`      | Title and description wrapper.   |
| `title`       | Title node.                      |
| `description` | Description node.                |
| `footer`      | Footer wrapper.                  |
| `actions`     | Default action buttons wrapper.  |
| `indicator`   | Current indicator node.          |
| `indicators`  | Indicator wrapper.               |

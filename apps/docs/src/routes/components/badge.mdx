# Badge

Small numerical value or status descriptor for UI elements.

## When To Use

Badge normally appears in proximity to notifications or user avatars with eye-catching appeal,
typically displaying unread messages count.

### Basic

## API

### Badge

| Property        | Description                                                                    | Type                                                                                                | Default  |
| --------------- | ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------- | -------- |
| `class`         | Root class name. Solid replacement for React `className`.                      | `string`                                                                                            | `-`      |
| `classNames`    | Customize class for each semantic structure. Supports object or function.      | `Partial<Record<'root' \| 'indicator', string>> \| (info: { props: BadgeProps }) => ...`            | `-`      |
| `color`         | Customize Badge dot, count, or status dot color. Supports preset color names.  | `string`                                                                                            | `-`      |
| `count`         | Number or custom content to show in badge.                                     | `JSX.Element`                                                                                       | `-`      |
| `dot`           | Whether to display a red dot instead of `count`.                               | `boolean`                                                                                           | `false`  |
| `offset`        | Set offset of the badge indicator.                                             | `[number \| string, number \| string]`                                                              | `-`      |
| `overflowCount` | Max count to show.                                                             | `number`                                                                                            | `99`     |
| `showZero`      | Whether to show badge when `count` is zero.                                    | `boolean`                                                                                           | `false`  |
| `size`          | If `count` is set, `size` sets the size of badge.                              | `'medium' \| 'small'`                                                                               | `medium` |
| `status`        | Set Badge as a status dot.                                                     | `'success' \| 'processing' \| 'default' \| 'error' \| 'warning'`                                    | `-`      |
| `styles`        | Customize inline style for each semantic structure. Supports object/function.  | `Partial<Record<'root' \| 'indicator', JSX.CSSProperties>> \| (info: { props: BadgeProps }) => ...` | `-`      |
| `text`          | If `status` or `color` is set, `text` sets the display text of the status dot. | `JSX.Element`                                                                                       | `-`      |
| `title`         | Text to show when hovering over the badge indicator.                           | `string`                                                                                            | `count`  |

### Badge.Ribbon

| Property     | Description                                                                   | Type                                                                                                              | Default |
| ------------ | ----------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- | ------- |
| `class`      | Ribbon indicator class name. Solid replacement for React `className`.         | `string`                                                                                                          | `-`     |
| `classNames` | Customize class for each semantic structure. Supports object or function.     | `Partial<Record<'root' \| 'content' \| 'indicator', string>> \| (info: { props: RibbonProps }) => ...`            | `-`     |
| `color`      | Customize Ribbon color. Supports preset color names.                          | `string`                                                                                                          | `-`     |
| `placement`  | The placement of the Ribbon.                                                  | `'start' \| 'end'`                                                                                                | `end`   |
| `styles`     | Customize inline style for each semantic structure. Supports object/function. | `Partial<Record<'root' \| 'content' \| 'indicator', JSX.CSSProperties>> \| (info: { props: RibbonProps }) => ...` | `-`     |
| `text`       | Content inside the Ribbon.                                                    | `JSX.Element`                                                                                                     | `-`     |

## Semantic DOM

### Badge

| Slot        | Description                |
| ----------- | -------------------------- |
| `root`      | Root wrapper element.      |
| `indicator` | Count, dot, or status dot. |

### Badge.Ribbon

| Slot        | Description             |
| ----------- | ----------------------- |
| `root`      | Ribbon wrapper element. |
| `indicator` | Ribbon element.         |
| `content`   | Ribbon text content.    |

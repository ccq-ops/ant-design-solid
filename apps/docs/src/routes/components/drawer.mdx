# Drawer

Drawer renders a slide-out panel for contextual tasks without leaving the page. It follows the antd v6 Drawer API shape while using Solid conventions such as `class` and `rootClass`.

### Basic

## API

| Property          | Description                                                                                    | Type                                                                                                                                  | Default                                        |
| ----------------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- |
| `afterOpenChange` | Callback after the drawer open state changes.                                                  | `(open: boolean) => void`                                                                                                             | `-`                                            |
| `class`           | Solid class for the drawer panel wrapper.                                                      | `string`                                                                                                                              | `-`                                            |
| `classList`       | Solid classList for conditional drawer panel wrapper classes.                                  | `Record<string, boolean \| undefined>`                                                                                                | `-`                                            |
| `classNames`      | Semantic class names. Supports object or function.                                             | `Partial<Record<DrawerSemanticDOM, string>> \| (info: { props }) => Partial<Record<DrawerSemanticDOM, string>>`                       | `-`                                            |
| `closable`        | Whether to show a close button. Object form configures disabled state and start/end placement. | `boolean \| { closeIcon?: JSX.Element; disabled?: boolean; placement?: 'start' \| 'end' }`                                            | `true`                                         |
| `closeIcon`       | Custom close icon content.                                                                     | `JSX.Element`                                                                                                                         | `<CloseOutlined />`                            |
| `defaultSize`     | Default drawer size used when `size` is not provided.                                          | `number`                                                                                                                              | `378`                                          |
| `destroyOnHidden` | Unmounts children after the drawer is hidden.                                                  | `boolean`                                                                                                                             | `false`                                        |
| `drawerRender`    | Custom render wrapper for the drawer panel node.                                               | `(node: JSX.Element) => JSX.Element`                                                                                                  | `-`                                            |
| `extra`           | Extra header action content.                                                                   | `JSX.Element`                                                                                                                         | `-`                                            |
| `focusable`       | Focus trap and return-focus configuration.                                                     | `{ trap?: boolean; focusTriggerAfterClose?: boolean }`                                                                                | `{ trap: true, focusTriggerAfterClose: true }` |
| `footer`          | Footer content.                                                                                | `JSX.Element`                                                                                                                         | `-`                                            |
| `forceRender`     | Pre-renders Drawer content even when closed.                                                   | `boolean`                                                                                                                             | `false`                                        |
| `getContainer`    | Portal mount container. Pass false to render inline.                                           | `HTMLElement \| () => HTMLElement \| string \| false`                                                                                 | `document.body`                                |
| `keyboard`        | Allows closing with Escape.                                                                    | `boolean`                                                                                                                             | `true`                                         |
| `loading`         | Shows a skeleton loading state in the body instead of children.                                | `boolean`                                                                                                                             | `false`                                        |
| `mask`            | Mask effect. Object form configures enabled, blur, and closable behavior.                      | `boolean \| { enabled?: boolean; blur?: boolean; closable?: boolean }`                                                                | `true`                                         |
| `maxSize`         | Maximum width or height while resizing.                                                        | `number`                                                                                                                              | `-`                                            |
| `open`            | Controls whether the drawer is visible.                                                        | `boolean`                                                                                                                             | `false`                                        |
| `placement`       | Side of the viewport the drawer slides from.                                                   | `'left' \| 'right' \| 'top' \| 'bottom'`                                                                                              | `'right'`                                      |
| `push`            | Nested drawers push behavior.                                                                  | `boolean \| { distance: number \| string }`                                                                                           | `{ distance: 180 }`                            |
| `resizable`       | Enables drag resizing and optional resize lifecycle callbacks.                                 | `boolean \| { onResizeStart?: () => void; onResize?: (size: number) => void; onResizeEnd?: () => void }`                              | `-`                                            |
| `rootClass`       | Solid class for the root element containing mask and drawer panel.                             | `string`                                                                                                                              | `-`                                            |
| `rootStyle`       | Inline style for the root element containing mask and drawer panel.                            | `JSX.CSSProperties`                                                                                                                   | `-`                                            |
| `size`            | Preset or custom drawer size.                                                                  | `'default' \| 'large' \| number \| string`                                                                                            | `'default'`                                    |
| `style`           | Inline style for the drawer panel wrapper.                                                     | `JSX.CSSProperties`                                                                                                                   | `-`                                            |
| `styles`          | Semantic inline styles. Supports object or function.                                           | `Partial<Record<DrawerSemanticDOM, JSX.CSSProperties>> \| (info: { props }) => Partial<Record<DrawerSemanticDOM, JSX.CSSProperties>>` | `-`                                            |
| `title`           | Drawer title content.                                                                          | `JSX.Element`                                                                                                                         | `-`                                            |
| `zIndex`          | Drawer z-index.                                                                                | `number`                                                                                                                              | `1000`                                         |
| `onClose`         | Called when the drawer requests close from the close button, mask, or keyboard.                | `(event?: MouseEvent \| KeyboardEvent) => void`                                                                                       | `-`                                            |
| `aria-label`      | Accessible label for the dialog.                                                               | `string`                                                                                                                              | `-`                                            |
| `aria-labelledby` | ID of an external element that labels the dialog.                                              | `string`                                                                                                                              | `-`                                            |
| `children`        | Drawer body content.                                                                           | `JSX.Element`                                                                                                                         | `-`                                            |

## Semantic DOM

Use these keys with `classNames` and `styles` to customize stable semantic sections of the drawer.

| Property  | Description                                                                 |
| --------- | --------------------------------------------------------------------------- |
| `root`    | Root element containing fixed positioning, z-index, mask, and drawer panel. |
| `mask`    | Background mask element.                                                    |
| `wrapper` | Positioned drawer panel wrapper.                                            |
| `section` | Drawer body container with flex layout, background, and overflow.           |
| `header`  | Header area that contains close, title, and extra content.                  |
| `title`   | Title text element.                                                         |
| `extra`   | Extra action content element.                                               |
| `body`    | Scrollable body content area.                                               |
| `footer`  | Footer action area.                                                         |
| `dragger` | Resize drag handle.                                                         |
| `close`   | Close button element.                                                       |

## ClosableConfig

| Property    | Description                                                                 | Type               | Default   |
| ----------- | --------------------------------------------------------------------------- | ------------------ | --------- |
| `closeIcon` | Custom close icon content. Prefer top-level `closeIcon` for antd v6 parity. | `JSX.Element`      | `-`       |
| `disabled`  | Disables the close button.                                                  | `boolean`          | `false`   |
| `placement` | Places the close button before or after the title area.                     | `'start' \| 'end'` | `'start'` |

## MaskConfig

| Property   | Description                               | Type      | Default |
| ---------- | ----------------------------------------- | --------- | ------- |
| `enabled`  | Whether to render the mask.               | `boolean` | `true`  |
| `blur`     | Adds backdrop blur to the mask.           | `boolean` | `false` |
| `closable` | Whether clicking the mask requests close. | `boolean` | `true`  |

## FocusableConfig

| Property                 | Description                                                  | Type      | Default |
| ------------------------ | ------------------------------------------------------------ | --------- | ------- |
| `trap`                   | Traps Tab focus inside the drawer.                           | `boolean` | `true`  |
| `focusTriggerAfterClose` | Returns focus to the previously focused trigger after close. | `boolean` | `true`  |

## ResizableConfig

| Property        | Description                                  | Type                     | Default |
| --------------- | -------------------------------------------- | ------------------------ | ------- |
| `onResizeStart` | Called when dragging starts.                 | `() => void`             | `-`     |
| `onResize`      | Called while dragging with the current size. | `(size: number) => void` | `-`     |
| `onResizeEnd`   | Called when dragging ends.                   | `() => void`             | `-`     |

## Design Token

| Token                 | Description                | Type     | Default |
| --------------------- | -------------------------- | -------- | ------- |
| `zIndexPopup`         | Drawer z-index token.      | `number` | `1000`  |
| `footerPaddingBlock`  | Footer vertical padding.   | `number` | `8`     |

## ConfigProvider

`ConfigProvider` accepts a `drawer` config for shared Drawer defaults.

| Property            | Description                                        | Type                              | Default |
| ------------------- | -------------------------------------------------- | --------------------------------- | ------- |
| `drawer.class`      | Default Solid class for the drawer panel wrapper.  | `string`                          | `-`     |
| `drawer.rootClass`  | Default Solid class for the root element.          | `string`                          | `-`     |
| `drawer.style`      | Default inline style for the drawer panel wrapper. | `JSX.CSSProperties`               | `-`     |
| `drawer.rootStyle`  | Default inline style for the root element.         | `JSX.CSSProperties`               | `-`     |
| `drawer.classNames` | Default semantic class names.                      | `DrawerSemanticClassNames`        | `-`     |
| `drawer.styles`     | Default semantic inline styles.                    | `DrawerSemanticStyles`            | `-`     |
| `drawer.closable`   | Default close button config.                       | `boolean \| DrawerClosableConfig` | `-`     |
| `drawer.mask`       | Default mask config.                               | `boolean \| DrawerMaskConfig`     | `-`     |
| `drawer.focusable`  | Default focus management config.                   | `DrawerFocusableConfig`           | `-`     |

## API

| Property       | Description                                                                 | Type                                                                                            | Default         |
| -------------- | --------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- | --------------- |
| `value`        | Content encoded into the QR code.                                           | `string \| string[]`                                                                            | `-`             |
| `type`         | Render element type.                                                        | `'canvas' \| 'svg'`                                                                             | `'svg'`         |
| `size`         | QR code size in pixels.                                                     | `number`                                                                                        | `160`           |
| `color`        | Module color.                                                               | `string`                                                                                        | `'#000'`        |
| `bgColor`      | Background color.                                                           | `string`                                                                                        | `'transparent'` |
| `bordered`     | Whether to render a border.                                                 | `boolean`                                                                                       | `true`          |
| `icon`         | Image URL rendered in the center.                                           | `string`                                                                                        | `-`             |
| `iconSize`     | Center icon size.                                                           | `number \| { width: number; height: number }`                                                   | `40`            |
| `errorLevel`   | QR error correction level.                                                  | `'L' \| 'M' \| 'Q' \| 'H'`                                                                      | `'M'`           |
| `boostLevel`   | Increase the error correction level when it does not increase QR code size. | `boolean`                                                                                       | `true`          |
| `marginSize`   | Quiet zone size in QR modules.                                              | `number`                                                                                        | `0`             |
| `status`       | QR code status overlay.                                                     | `'active' \| 'expired' \| 'loading' \| 'scanned'`                                               | `'active'`      |
| `statusRender` | Custom status overlay renderer.                                             | `(info: { status: QRCodeStatus; locale: QRCodeLocale; onRefresh?: () => void }) => JSX.Element` | `-`             |
| `onRefresh`    | Refresh callback passed to `statusRender`.                                  | `() => void`                                                                                    | `-`             |
| `locale`       | Status text locale.                                                         | `Partial<{ expired: string; loading: string; scanned: string; refresh: string }>`               | Built-in text   |
| `classes`      | Semantic class names using Solid naming.                                    | `Partial<Record<'root' \| 'image' \| 'cover', string>>`                                         | `-`             |
| `classNames`   | Alias of `classes` for Ant Design API parity.                               | `Partial<Record<'root' \| 'image' \| 'cover', string>>`                                         | `-`             |
| `styles`       | Semantic styles.                                                            | `Partial<Record<'root' \| 'image' \| 'cover', JSX.CSSProperties>>`                              | `-`             |
| `prefixCls`    | Custom CSS class prefix.                                                    | `string`                                                                                        | `-`             |

## API

### ColorPicker

| Property               | Description                                                     | Type                                                                                    | Default          |
| ---------------------- | --------------------------------------------------------------- | --------------------------------------------------------------------------------------- | ---------------- |
| `mode`                 | Color editing mode.                                             | `'single' \| 'gradient' \| Array<'single' \| 'gradient'>`                               | `'single'`       |
| `value`                | Controlled color value.                                         | `ColorPickerValue`                                                                      | `-`              |
| `defaultValue`         | Initial color value for uncontrolled usage.                     | `ColorPickerValue`                                                                      | `-`              |
| `children`             | Custom trigger content.                                         | `JSX.Element`                                                                           | `-`              |
| `open`                 | Controlled popup open state.                                    | `boolean`                                                                               | `-`              |
| `defaultOpen`          | Initial popup open state.                                       | `boolean`                                                                               | `false`          |
| `disabled`             | Disable the picker.                                             | `boolean`                                                                               | `false`          |
| `placement`            | Popup placement.                                                | `TooltipPlacement`                                                                      | `'bottomLeft'`   |
| `trigger`              | Interaction that opens the popup.                               | `'click' \| 'hover'`                                                                    | `'click'`        |
| `format`               | Controlled color input format.                                  | `'hex' \| 'rgb' \| 'hsb'`                                                               | `-`              |
| `defaultFormat`        | Initial color input format.                                     | `'hex' \| 'rgb' \| 'hsb'`                                                               | `'hex'`          |
| `disabledFormat`       | Disable format switching.                                       | `boolean`                                                                               | `false`          |
| `disabledAlpha`        | Disable alpha editing.                                          | `boolean`                                                                               | `false`          |
| `allowClear`           | Show a clear button.                                            | `boolean`                                                                               | `false`          |
| `arrow`                | Show and configure popup arrow.                                 | `boolean \| { pointAtCenter?: boolean }`                                                | `true`           |
| `showText`             | Show selected color text or custom text renderer.               | `boolean \| ((color: Color \| undefined) => JSX.Element)`                               | `-`              |
| `presets`              | Preset color groups.                                            | `ColorPickerPreset[]`                                                                   | `-`              |
| `panelRender`          | Custom panel renderer.                                          | `(panel: JSX.Element, extra: ColorPickerPanelRenderExtra) => JSX.Element`               | `-`              |
| `class`                | Trigger class name.                                             | `string`                                                                                | `-`              |
| `rootClass`            | Additional trigger root class.                                  | `string`                                                                                | `-`              |
| `classNames`           | Semantic class names.                                           | `ColorPickerSemanticClassNames \| ((info: { props }) => ColorPickerSemanticClassNames)` | `-`              |
| `styles`               | Semantic inline styles.                                         | `ColorPickerSemanticStyles \| ((info: { props }) => ColorPickerSemanticStyles)`         | `-`              |
| `popupClass`           | Additional popup class.                                         | `string`                                                                                | `-`              |
| `popupStyle`           | Popup inline style.                                             | `JSX.CSSProperties`                                                                     | `-`              |
| `zIndex`               | Custom popup z-index.                                           | `number`                                                                                | `-`              |
| `getPopupContainer`    | Return popup portal container.                                  | `(triggerNode?: HTMLElement) => HTMLElement \| ShadowRoot`                              | `ConfigProvider` |
| `autoAdjustOverflow`   | Automatically adjust placement when popup overflows.            | `boolean \| { adjustX?: boolean; adjustY?: boolean }`                                   | `true`           |
| `destroyOnHidden`      | Destroy popup content when hidden.                              | `boolean`                                                                               | `false`          |
| `destroyTooltipOnHide` | Deprecated alias for `destroyOnHidden`.                         | `boolean \| { keepParent?: boolean }`                                                   | `false`          |
| `onOpenChange`         | Called when popup open state changes.                           | `(open: boolean) => void`                                                               | `-`              |
| `onFormatChange`       | Called when uncontrolled format changes.                        | `(format?: 'hex' \| 'rgb' \| 'hsb') => void`                                            | `-`              |
| `onChange`             | Called when the color changes with Color object and CSS string. | `(value: Color \| undefined, css: string) => void`                                      | `-`              |
| `onChangeComplete`     | Called after a committed color change.                          | `(value: Color \| undefined) => void`                                                   | `-`              |
| `onClear`              | Called after the color is cleared.                              | `() => void`                                                                            | `-`              |

### ColorPickerPreset

| Property      | Description                          | Type                 | Default |
| ------------- | ------------------------------------ | -------------------- | ------- |
| `label`       | Preset group label.                  | `JSX.Element`        | `-`     |
| `colors`      | Preset single colors or gradients.   | `ColorPickerValue[]` | `-`     |
| `defaultOpen` | Whether the group is initially open. | `boolean`            | `true`  |
| `key`         | Stable group key for open state.     | `string \| number`   | `index` |

### ColorPickerValue

| Type           | Shape                                                                                                                   |
| -------------- | ----------------------------------------------------------------------------------------------------------------------- |
| Single color   | `string \| { r: number; g: number; b: number; a?: number } \| { h: number; s: number; b: number; a?: number } \| Color` |
| Gradient color | `Array<{ color: SingleColorPickerValue; percent: number }>`                                                             |
| Empty          | `null \| undefined`                                                                                                     |

### Color

| Method          | Description                                  | Return                      |
| --------------- | -------------------------------------------- | --------------------------- |
| `toHex()`       | Return hex without `#`.                      | `string`                    |
| `toHexString()` | Return hex with `#`.                         | `string`                    |
| `toRgb()`       | Return normalized RGB channels.              | `{ r; g; b; a }`            |
| `toRgbString()` | Return CSS rgb or rgba string.               | `string`                    |
| `toHsb()`       | Return normalized HSB channels.              | `{ h; s; b; a }`            |
| `toHsbString()` | Return hsb or hsba string.                   | `string`                    |
| `toCssString()` | Return rgb/rgba or linear-gradient string.   | `string`                    |
| `isGradient()`  | Whether the value represents a gradient.     | `boolean`                   |
| `getColors()`   | Return gradient stops or current color.      | `ParsedGradientColorStop[]` |
| `equals(color)` | Compare normalized single or gradient value. | `boolean`                   |

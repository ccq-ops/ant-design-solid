# Avatar

Avatars can be used to represent people or objects. They support images, icons, characters, and grouped overflow.

### Autoset Font Size

:::preview

<AvatarDemo004 />

export const AvatarDemo004 = (() => {
  const users = ['U', 'Lucy', 'Tom', 'Edward'];
  const Demo = function () {
      const [index, setIndex] = createSignal(0);
      return (<Space align="center">
        <Avatar style={{ background: '#f56a00' }} size="large" gap={4}>
          {users[index()]}
        </Avatar>
        <Button onClick={() => setIndex((index() + 1) % users.length)}>Change</Button>
      </Space>);
  };

return Demo
})()

---

```tsx
import { createSignal } from 'solid-js'
import { Avatar, Button, Space } from '@ant-design-solid/core'

const users = ['U', 'Lucy', 'Tom', 'Edward']

const Demo = function () {
  const [index, setIndex] = createSignal(0)

        {users[index()]}
      </Avatar>
      <Button onClick={() => setIndex((index() + 1) % users.length)}>Change</Button>
    </Space>
  )
}

export default Demo
```

## API

### Avatar

| Property      | Description                                                           | Type                                                                                                                                                        | Default    |
| ------------- | --------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| `alt`         | Alternate text describing the image.                                  | `string`                                                                                                                                                    | `-`        |
| `gap`         | Letter type unit distance between left and right sides.               | `number`                                                                                                                                                    | `4`        |
| `icon`        | Custom icon for an icon avatar.                                       | `JSX.Element`                                                                                                                                               | `-`        |
| `shape`       | Avatar shape.                                                         | `'circle' \| 'square'`                                                                                                                                      | `'circle'` |
| `size`        | Avatar size. `default` is kept as a compatibility alias for `medium`. | `number \| 'large' \| 'medium' \| 'default' \| 'small' \| { xs?: number; sm?: number; md?: number; lg?: number; xl?: number; xxl?: number; xxxl?: number }` | `'medium'` |
| `src`         | Image URL or image element.                                           | `string \| JSX.Element`                                                                                                                                     | `-`        |
| `srcSet`      | Image source set for different screen resolutions.                    | `string`                                                                                                                                                    | `-`        |
| `draggable`   | Whether the picture is draggable.                                     | `boolean \| 'true' \| 'false'`                                                                                                                              | `true`     |
| `crossOrigin` | CORS setting for the image.                                           | `'' \| 'anonymous' \| 'use-credentials'`                                                                                                                    | `-`        |
| `onError`     | Handler when image loading fails. Return `false` to prevent fallback. | `() => boolean`                                                                                                                                             | `-`        |
| `class`       | Custom root class in Solid style.                                     | `string`                                                                                                                                                    | `-`        |
| `rootClass`   | Extra root class.                                                     | `string`                                                                                                                                                    | `-`        |
| `prefixCls`   | Custom component class prefix.                                        | `string`                                                                                                                                                    | `-`        |

`icon` and `children` can be used as image load failure fallback, with priority `icon` > `children`.

### Avatar.Group

| Property              | Description                                         | Type                                                                    | Default    |
| --------------------- | --------------------------------------------------- | ----------------------------------------------------------------------- | ---------- |
| `max`                 | Maximum display configuration.                      | `{ count?: number; style?: JSX.CSSProperties; popover?: PopoverProps }` | `-`        |
| `maxCount`            | Deprecated. Use `max={{ count: number }}`.          | `number`                                                                | `-`        |
| `maxPopoverPlacement` | Deprecated. Use `max={{ popover: { placement } }}`. | `'top' \| 'bottom'`                                                     | `'top'`    |
| `maxPopoverTrigger`   | Deprecated. Use `max={{ popover: { trigger } }}`.   | `'hover' \| 'focus' \| 'click' \| 'contextMenu' \| Array`               | `'hover'`  |
| `maxStyle`            | Deprecated. Use `max={{ style: CSSProperties }}`.   | `JSX.CSSProperties`                                                     | `-`        |
| `size`                | Shared avatar size.                                 | Same as Avatar `size`                                                   | `'medium'` |
| `shape`               | Shared avatar shape.                                | `'circle' \| 'square'`                                                  | `'circle'` |
| `class`               | Custom root class in Solid style.                   | `string`                                                                | `-`        |
| `rootClass`           | Extra root class.                                   | `string`                                                                | `-`        |
| `prefixCls`           | Custom component class prefix.                      | `string`                                                                | `-`        |

## Design Token

| Token              | Description                                 | Default   |
| ------------------ | ------------------------------------------- | --------- |
| `containerSize`    | Size of Avatar.                             | `32`      |
| `containerSizeLG`  | Size of large Avatar.                       | `40`      |
| `containerSizeSM`  | Size of small Avatar.                       | `24`      |
| `textFontSize`     | Font size of Avatar text.                   | `14`      |
| `textFontSizeLG`   | Font size of large Avatar text.             | `14`      |
| `textFontSizeSM`   | Font size of small Avatar text.             | `14`      |
| `iconFontSize`     | Font size of Avatar icon.                   | `18`      |
| `iconFontSizeLG`   | Font size of large Avatar icon.             | `24`      |
| `iconFontSizeSM`   | Font size of small Avatar icon.             | `14`      |
| `groupSpace`       | Spacing between avatars in a group popover. | `4`       |
| `groupOverlapping` | Overlapping of avatars in a group.          | `-8`      |
| `groupBorderColor` | Border color of avatars in a group.         | `#ffffff` |

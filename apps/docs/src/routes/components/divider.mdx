## API

| Property            | Description                                                      | Type                                                                                                 | Default        |
| ------------------- | ---------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- | -------------- |
| `children`          | Text or content rendered inside a horizontal divider.            | `JSX.Element`                                                                                        | `-`            |
| `class`             | Root class name.                                                 | `string`                                                                                             | `-`            |
| `classNames`        | Semantic class names.                                            | `{ root?: string; rail?: string; content?: string } \| (info) => …`                                  | `-`            |
| `dashed`            | Uses a dashed divider line. `variant` is preferred for new code. | `boolean`                                                                                            | `false`        |
| `orientation`       | Divider direction.                                               | `'horizontal' \| 'vertical'`                                                                         | `'horizontal'` |
| `orientationMargin` | Margin for text placed at the start or end.                      | `string \| number`                                                                                   | `-`            |
| `plain`             | Uses plain text styling for divider content.                     | `boolean`                                                                                            | `false`        |
| `prefixCls`         | Custom component class prefix.                                   | `string`                                                                                             | `-`            |
| `rootClass`         | Extra root class name.                                           | `string`                                                                                             | `-`            |
| `size`              | Horizontal divider spacing size.                                 | `'small' \| 'medium' \| 'middle' \| 'large'`                                                         | `-`            |
| `styles`            | Semantic styles.                                                 | `{ root?: JSX.CSSProperties; rail?: JSX.CSSProperties; content?: JSX.CSSProperties } \| (info) => …` | `-`            |
| `titlePlacement`    | Text placement for horizontal dividers with children.            | `'left' \| 'right' \| 'center' \| 'start' \| 'end'`                                                  | `'center'`     |
| `variant`           | Divider line style.                                              | `'dashed' \| 'dotted' \| 'solid'`                                                                    | `'solid'`      |
| `vertical`          | Shortcut for `orientation="vertical"`.                           | `boolean`                                                                                            | `false`        |

## Design Token

| Token                  | Description                            | Default          |
| ---------------------- | -------------------------------------- | ---------------- |
| `textPaddingInline`    | Horizontal padding of text.            | `'1em'`          |
| `orientationMargin`    | Distance between text and edge.        | `0.05`           |
| `verticalMarginInline` | Horizontal margin of vertical Divider. | `token.marginXS` |

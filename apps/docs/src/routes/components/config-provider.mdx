## API

| Property                | Description                                                                                                               | Type                                                       | Default      |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- | ------------ |
| `prefixCls`             | Global CSS class prefix for components.                                                                                   | `string`                                                   | `'ads'`      |
| `iconPrefixCls`         | Global CSS class prefix for icons.                                                                                        | `string`                                                   | `'adsicon'`  |
| `componentSize`         | Default component size. `middle` is kept for compatibility; `medium` follows antd v6.                                     | `'small' \| 'medium' \| 'middle' \| 'large'`               | `'middle'`   |
| `componentDisabled`     | Set disabled state for supported child components.                                                                        | `boolean`                                                  | `false`      |
| `direction`             | Text and layout direction.                                                                                                | `'ltr' \| 'rtl'`                                           | `'ltr'`      |
| `variant`               | Default variant for supported data-entry components.                                                                      | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`   | `'outlined'` |
| `theme`                 | Theme token configuration. Supports `token`, `components`, `algorithm`, `inherit`, `hashed`, `cssVar`, and `zeroRuntime`. | `ThemeConfig`                                              | `-`          |
| `locale`                | Global locale object.                                                                                                     | `Locale`                                                   | `-`          |
| `renderEmpty`           | Custom empty-state renderer.                                                                                              | `(componentName?: string) => JSX.Element`                  | `-`          |
| `getPopupContainer`     | Global popup portal container resolver.                                                                                   | `(triggerNode?: HTMLElement) => HTMLElement \| ShadowRoot` | `-`          |
| `getTargetContainer`    | Global target container resolver.                                                                                         | `() => HTMLElement \| Window \| ShadowRoot`                | `-`          |
| `virtual`               | Enables virtual scrolling for supported components.                                                                       | `boolean`                                                  | `true`       |
| `popupMatchSelectWidth` | Default popup width matching behavior for supported select-like components.                                               | `boolean \| number`                                        | `-`          |
| `popupOverflow`         | Popup overflow strategy.                                                                                                  | `'viewport' \| 'scroll'`                                   | `'viewport'` |
| `csp`                   | Content Security Policy config.                                                                                           | `{ nonce?: string }`                                       | `-`          |
| `warning`               | Warning behavior config.                                                                                                  | `{ strict?: boolean }`                                     | `-`          |
| `wave`                  | Wave effect config.                                                                                                       | `{ disabled?: boolean; triggerType?: TriggerType }`        | `-`          |
| `children`              | Application or subtree receiving the configuration.                                                                       | `JSX.Element`                                              | `-`          |

## Component Config

Component config props follow antd v6 names, with Solid conventions:

- `class` replaces React `className`.
- Deprecated compatibility props from antd are not included.
- Config is inherited by nested providers and shallow-merged by component key.

Supported component config keys include `affix`, `alert`, `anchor`, `app`, `avatar`, `badge`, `borderBeam`, `breadcrumb`, `button`, `calendar`, `card`, `cardMeta`, `carousel`, `cascader`, `checkbox`, `collapse`, `colorPicker`, `datePicker`, `rangePicker`, `descriptions`, `divider`, `drawer`, `dropdown`, `empty`, `flex`, `floatButton`, `floatButtonGroup`, `form`, `image`, `input`, `inputPassword`, `inputSearch`, `otp`, `inputNumber`, `textArea`, `layout`, `list`, `masonry`, `mentions`, `menu`, `message`, `modal`, `notification`, `pagination`, `popconfirm`, `popover`, `progress`, `qrcode`, `radio`, `rate`, `result`, `ribbon`, `segmented`, `select`, `skeleton`, `slider`, `space`, `spin`, `splitter`, `statistic`, `steps`, `switch`, `table`, `tabs`, `tag`, `timeline`, `timePicker`, `tooltip`, `tour`, `transfer`, `tree`, `treeSelect`, `typography`, `upload`, and `watermark`.

## Static Methods

| Method                         | Description                                                                       | Type                                                                                                                                           |
| ------------------------------ | --------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `ConfigProvider.config`        | Sets global static defaults for prefix, icon prefix, theme, and holder rendering. | `(config: { prefixCls?: string; iconPrefixCls?: string; theme?: ThemeConfig; holderRender?: (children: JSX.Element) => JSX.Element }) => void` |
| `ConfigProvider.useConfig`     | Reads current ConfigProvider context.                                             | `() => ConfigContextValue`                                                                                                                     |
| `ConfigProvider.ConfigContext` | Raw Solid context for advanced integrations.                                      | `Context<ConfigContextValue>`                                                                                                                  |

# Radio

Radio lets users choose one option from a mutually exclusive set.

## API

### Radio / Radio.Button

| Property         | Description                                              | Type                                                                                                      | Default |
| ---------------- | -------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- | ------- |
| `checked`        | Controlled checked state.                                | `boolean`                                                                                                 | `-`     |
| `defaultChecked` | Initial checked state for uncontrolled usage.            | `boolean`                                                                                                 | `false` |
| `disabled`       | Disables the radio.                                      | `boolean`                                                                                                 | `false` |
| `value`          | Radio option value.                                      | `string \| number \| boolean`                                                                             | `-`     |
| `prefixCls`      | Custom CSS class prefix.                                 | `string`                                                                                                  | `-`     |
| `children`       | Radio label content.                                     | `JSX.Element`                                                                                             | `-`     |
| `classNames`     | Semantic class names for the wrapper and native input.   | `Partial<Record<'wrapper' \| 'input', string>> \| ((info: { props }) => Partial<Record<...>>)`            | `-`     |
| `styles`         | Semantic inline styles for the wrapper and native input. | `Partial<Record<'wrapper' \| 'input', JSX.CSSProperties>> \| ((info: { props }) => Partial<Record<...>>)` | `-`     |
| `ref`            | Imperative ref for focus and blur methods.               | `{ current?: RadioRef } \| ((ref: RadioRef) => void)`                                                     | `-`     |
| `onChange`       | Called when the native radio change event fires.         | `JSX.EventHandler<HTMLInputElement, Event>`                                                               | `-`     |

### Radio.Group

| Property       | Description                                                                                              | Type                                                                                           | Default        |
| -------------- | -------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- | -------------- |
| `value`        | Controlled selected value.                                                                               | `string \| number \| boolean`                                                                  | `-`            |
| `defaultValue` | Initial selected value for uncontrolled usage.                                                           | `string \| number \| boolean`                                                                  | `-`            |
| `options`      | Options rendered as radios.                                                                              | `OptionInput[]`                                                                                | `-`            |
| `disabled`     | Disables all radios in the group.                                                                        | `boolean`                                                                                      | `false`        |
| `optionType`   | Renders options as standard radios or button-style radios.                                               | `'default' \| 'button'`                                                                        | `'default'`    |
| `buttonStyle`  | Style variant for button-style radios.                                                                   | `'outline' \| 'solid'`                                                                         | `'outline'`    |
| `block`        | Makes button-style radios fill the group width.                                                          | `boolean`                                                                                      | `false`        |
| `name`         | Sets the `name` attribute for all inner radio inputs. A generated name is used when omitted.             | `string`                                                                                       | generated      |
| `orientation`  | Group layout direction. Takes priority over `vertical`.                                                  | `'horizontal' \| 'vertical'`                                                                   | `'horizontal'` |
| `vertical`     | Shortcut for vertical layout when `orientation` is not provided.                                         | `boolean`                                                                                      | `false`        |
| `size`         | Size for button-style radios.                                                                            | `'small' \| 'medium' \| 'large'`                                                               | `-`            |
| `prefixCls`    | Custom radio CSS class prefix.                                                                           | `string`                                                                                       | `-`            |
| `children`     | Custom `Radio` or `Radio.Button` children.                                                               | `JSX.Element`                                                                                  | `-`            |
| `classNames`   | Semantic class names for the group wrapper.                                                              | `Partial<Record<'wrapper', string>> \| ((info: { props }) => Partial<Record<...>>)`            | `-`            |
| `styles`       | Semantic inline styles for the group wrapper.                                                            | `Partial<Record<'wrapper', JSX.CSSProperties>> \| ((info: { props }) => Partial<Record<...>>)` | `-`            |
| `onChange`     | Called with the next selected value. This keeps the Solid-style value callback instead of an antd event. | `(checkedValue: OptionValue) => void`                                                          | `-`            |

### OptionInput

| Property                      | Description                                    | Type                                        | Default |
| ----------------------------- | ---------------------------------------------- | ------------------------------------------- | ------- |
| `string \| number \| boolean` | Primitive option used as both label and value. | `OptionInput`                               | `-`     |
| `label`                       | Option label content for object options.       | `JSX.Element`                               | `-`     |
| `value`                       | Option value.                                  | `string \| number \| boolean`               | `-`     |
| `disabled`                    | Disables this option.                          | `boolean`                                   | `false` |
| `style`                       | Inline style for this option wrapper.          | `JSX.CSSProperties`                         | `-`     |
| `class`                       | Additional Solid-style class for this option.  | `string`                                    | `-`     |
| `title`                       | Title attribute for this option wrapper.       | `string`                                    | `-`     |
| `id`                          | Id attribute for this option input.            | `string`                                    | `-`     |
| `required`                    | Required attribute for this option input.      | `boolean`                                   | `false` |
| `onChange`                    | Called when this option changes.               | `JSX.EventHandler<HTMLInputElement, Event>` | `-`     |

### RadioRef

| Method          | Description                              |
| --------------- | ---------------------------------------- |
| `focus()`       | Focuses the native radio input.          |
| `blur()`        | Blurs the native radio input.            |
| `nativeElement` | Native radio input element when mounted. |

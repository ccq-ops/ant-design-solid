## API

Common props such as `id`, `name`, `title`, `autofocus`, keyboard handlers, and focus/blur
handlers follow Solid DOM prop names and are passed to the native input or group wrapper where
appropriate.

### Radio / Radio.Button

| Property         | Description                                                                                                                | Type                                                                                                                        | Default |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- | ------- |
| `checked`        | Controlled checked state.                                                                                                  | `boolean`                                                                                                                   | `false` |
| `class`          | Additional Solid-style class for the radio wrapper.                                                                        | `string`                                                                                                                    | `-`     |
| `classNames`     | Semantic class names. antd v6 keys are `root`, `icon`, and `label`; legacy `wrapper` and `input` keys are also accepted.   | `Partial<Record<SemanticDOM, string>> \| ((info: { props }) => Partial<Record<SemanticDOM, string>>)`                       | `-`     |
| `defaultChecked` | Initial checked state for uncontrolled usage.                                                                              | `boolean`                                                                                                                   | `false` |
| `disabled`       | Disable radio.                                                                                                             | `boolean`                                                                                                                   | `false` |
| `rootClass`      | Additional class for the radio root wrapper. Solid-style replacement for antd React `rootClassName`.                       | `string`                                                                                                                    | `-`     |
| `skipGroup`      | Prevents a radio inside `Radio.Group` from reading group value, name, and disabled state.                                  | `boolean`                                                                                                                   | `false` |
| `styles`         | Semantic inline styles. antd v6 keys are `root`, `icon`, and `label`; legacy `wrapper` and `input` keys are also accepted. | `Partial<Record<SemanticDOM, JSX.CSSProperties>> \| ((info: { props }) => Partial<Record<SemanticDOM, JSX.CSSProperties>>)` | `-`     |
| `value`          | Value used for comparison when inside a group.                                                                             | `string \| number \| boolean`                                                                                               | `-`     |
| `onChange`       | Called when the native radio change event fires.                                                                           | `JSX.EventHandler<HTMLInputElement, Event>`                                                                                 | `-`     |

### Radio.Group

Radio group can wrap a group of `Radio`.

| Property       | Description                                                                                                          | Type                                                                                                                                        | Default        |
| -------------- | -------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- | -------------- |
| `block`        | Option to fit radio group width to its parent width.                                                                 | `boolean`                                                                                                                                   | `false`        |
| `buttonStyle`  | The style type of radio button.                                                                                      | `'outline' \| 'solid'`                                                                                                                      | `'outline'`    |
| `class`        | Additional Solid-style class for the group wrapper.                                                                  | `string`                                                                                                                                    | `-`            |
| `classNames`   | Semantic class names for the group root. `wrapper` is accepted as a legacy alias.                                    | `Partial<Record<'root' \| 'wrapper', string>> \| ((info: { props }) => Partial<Record<'root' \| 'wrapper', string>>)`                       | `-`            |
| `defaultValue` | Default selected value.                                                                                              | `string \| number \| boolean`                                                                                                               | `-`            |
| `disabled`     | Disable all radio buttons.                                                                                           | `boolean`                                                                                                                                   | `false`        |
| `name`         | The `name` property of all `input[type="radio"]` children. If not set, it will fallback to a generated name.         | `string`                                                                                                                                    | generated      |
| `options`      | Set children optional.                                                                                               | `Array<string \| number \| boolean \| Option>`                                                                                              | `-`            |
| `optionType`   | Set radio option type.                                                                                               | `'default' \| 'button'`                                                                                                                     | `'default'`    |
| `orientation`  | Orientation. Takes priority over `vertical`.                                                                         | `'horizontal' \| 'vertical'`                                                                                                                | `'horizontal'` |
| `rootClass`    | Additional class for the group root wrapper. Solid-style replacement for antd React `rootClassName`.                 | `string`                                                                                                                                    | `-`            |
| `size`         | The size of radio button style.                                                                                      | `'large' \| 'medium' \| 'small'`                                                                                                            | `-`            |
| `styles`       | Semantic inline styles for the group root. `wrapper` is accepted as a legacy alias.                                  | `Partial<Record<'root' \| 'wrapper', JSX.CSSProperties>> \| ((info: { props }) => Partial<Record<'root' \| 'wrapper', JSX.CSSProperties>>)` | `-`            |

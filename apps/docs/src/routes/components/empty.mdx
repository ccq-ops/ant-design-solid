## API

| Property        | Description                                            | Type                                                                                                             | Default                         |
| --------------- | ------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------- | ------------------------------- |
| `class`         | Extra class name for the root element.                 | `string`                                                                                                         | `-`                             |
| `rootClassName` | Extra root class name.                                 | `string`                                                                                                         | `-`                             |
| `prefixCls`     | Custom component class prefix.                         | `string`                                                                                                         | `ads-empty`                     |
| `image`         | Custom empty image content or image URL.               | `JSX.Element`                                                                                                    | `Empty.PRESENTED_IMAGE_DEFAULT` |
| `description`   | Description text or content displayed below the image. | `JSX.Element`                                                                                                    | `'No Data'`                     |
| `children`      | Footer content, usually actions.                       | `JSX.Element`                                                                                                    | `-`                             |
| `classNames`    | Semantic class names.                                  | `Partial<Record<SemanticDOM, string>> \| ((info) => Partial<Record<SemanticDOM, string>>)`                       | `-`                             |
| `styles`        | Semantic styles.                                       | `Partial<Record<SemanticDOM, JSX.CSSProperties>> \| ((info) => Partial<Record<SemanticDOM, JSX.CSSProperties>>)` | `-`                             |

### Static image

| Property                        | Description          |
| ------------------------------- | -------------------- |
| `Empty.PRESENTED_IMAGE_DEFAULT` | Default empty image. |
| `Empty.PRESENTED_IMAGE_SIMPLE`  | Simple empty image.  |

### Semantic DOM

| Key           | Description          |
| ------------- | -------------------- |
| `root`        | Root container.      |
| `image`       | Image wrapper.       |
| `description` | Description content. |
| `footer`      | Footer content.      |

### ConfigProvider

| Property           | Description                           | Type                |
| ------------------ | ------------------------------------- | ------------------- |
| `empty.class`      | Extra class name for all Empty roots. | `string`            |
| `empty.style`      | Root style for all Empty components.  | `JSX.CSSProperties` |
| `empty.image`      | Default image for Empty.              | `JSX.Element`       |
| `empty.classNames` | Semantic class names for Empty.       | Same as `Empty`     |
| `empty.styles`     | Semantic styles for Empty.            | Same as `Empty`     |

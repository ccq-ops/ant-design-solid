## API

### Alert

| Property      | Description                                                                | Type                                          | Default                              | Global Config      |
| ------------- | -------------------------------------------------------------------------- | --------------------------------------------- | ------------------------------------ | ------------------ |
| `action`      | The action of Alert.                                                       | `JSX.Element`                                 | `-`                                  | No                 |
| `afterClose`  | Deprecated. Called when Alert closes; use `closable.afterClose` instead.   | `() => void`                                  | `-`                                  | No                 |
| `banner`      | Whether to show as banner.                                                 | `boolean`                                     | `false`                              | No                 |
| `variant`     | Variant of Alert style.                                                    | `'outlined' \| 'filled'`                      | `'outlined'`                         | Yes                |
| `classNames`  | Customize class for each semantic structure. Supports object or function.  | `AlertSemanticClassNames`                     | `-`                                  | Yes                |
| `styles`      | Customize style for each semantic structure. Supports object or function.  | `AlertSemanticStyles`                         | `-`                                  | Yes                |
| `closable`    | Shows close button or configures close behavior.                           | `boolean \| AlertClosableConfig`              | `false`                              | Yes                |
| `description` | Additional content of Alert.                                               | `JSX.Element`                                 | `-`                                  | No                 |
| `icon`        | Custom icon, effective when `showIcon` is true.                            | `JSX.Element`                                 | `-`                                  | No                 |
| `message`     | Deprecated. Content of Alert; use `title` instead.                         | `JSX.Element`                                 | `-`                                  | No                 |
| `onClose`     | Deprecated. Callback when Alert is closed; use `closable.onClose` instead. | `(event: MouseEvent) => void`                 | `-`                                  | No                 |
| `closeIcon`   | Deprecated. Custom close icon; use `closable.closeIcon` instead.           | `JSX.Element \| boolean \| null`              | `-`                                  | Global config only |
| `closeText`   | Deprecated. Close text to show; use `closable.closeIcon` instead.          | `JSX.Element`                                 | `-`                                  | No                 |
| `showIcon`    | Whether to show icon.                                                      | `boolean`                                     | `false`, `true` in banner mode       | No                 |
| `title`       | Content of Alert.                                                          | `JSX.Element`                                 | `-`                                  | No                 |
| `type`        | Type of Alert styles.                                                      | `'success' \| 'info' \| 'warning' \| 'error'` | `'info'`, `'warning'` in banner mode | No                 |
| `prefixCls`   | Custom component class prefix.                                             | `string`                                      | `-`                                  | No                 |
| `rootClass`   | Custom root class. Solid replacement for React `rootClassName`.            | `string`                                      | `-`                                  | No                 |

### Alert closable config

| Property     | Description                               | Type                          | Default |
| ------------ | ----------------------------------------- | ----------------------------- | ------- |
| `closeIcon`  | Custom close icon.                        | `JSX.Element`                 | `-`     |
| `afterClose` | Called after this Alert closes.           | `() => void`                  | `-`     |
| `onClose`    | Called when this close button is clicked. | `(event: MouseEvent) => void` | `-`     |

### Alert.ErrorBoundary

| Property      | Description                                          | Type          | Default       |
| ------------- | ---------------------------------------------------- | ------------- | ------------- |
| `description` | Custom error description to show.                    | `JSX.Element` | Error stack   |
| `message`     | Deprecated. Custom error title; use `title` instead. | `JSX.Element` | Error message |
| `title`       | Custom error title to show.                          | `JSX.Element` | Error message |

### Semantic DOM

| Name          | Description                                                                           |
| ------------- | ------------------------------------------------------------------------------------- |
| `root`        | Root element with border, background, padding, border-radius, and positioning styles. |
| `icon`        | Icon element for status icon.                                                         |
| `section`     | Content element controlling title and description layout.                             |
| `title`       | Title element.                                                                        |
| `description` | Description element.                                                                  |
| `actions`     | Action container.                                                                     |
| `close`       | Close button element.                                                                 |

### ConfigProvider alert

| Property      | Description                 | Type                             |
| ------------- | --------------------------- | -------------------------------- |
| `variant`     | Default Alert variant.      | `'outlined' \| 'filled'`         |
| `closable`    | Default closable config.    | `boolean \| AlertClosableConfig` |
| `closeIcon`   | Global close icon fallback. | `JSX.Element`                    |
| `successIcon` | Global success icon.        | `JSX.Element`                    |
| `infoIcon`    | Global info icon.           | `JSX.Element`                    |
| `warningIcon` | Global warning icon.        | `JSX.Element`                    |
| `errorIcon`   | Global error icon.          | `JSX.Element`                    |
| `classNames`  | Default semantic classes.   | `AlertSemanticClassNames`        |
| `styles`      | Default semantic styles.    | `AlertSemanticStyles`            |

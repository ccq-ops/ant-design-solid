# Popover

Popover displays rich floating content around a trigger element.

### Basic

:::preview

<PopoverDemo001 />

export const PopoverDemo001 = (() => {
  const Demo1 = function () {
      return (<Popover title="Title" content="Helpful content">
        <Button>Hover me</Button>
      </Popover>);
  };

  return Demo1
})()

### Controlled

:::preview

<PopoverDemo005 />

export const PopoverDemo005 = (() => {
  const Demo5 = function () {
      const [open, setOpen] = createSignal(false);
      return (<Space>
        <Popover open={open()} onOpenChange={setOpen} content="Controlled content">
          <Button>Controlled</Button>
        </Popover>
        <Button onClick={() => setOpen((next) => !next)}>{open() ? 'Close' : 'Open'}</Button>
      </Space>);
  };

  return Demo5
})()

### Fresh And Destroy

:::preview

<PopoverDemo009 />

export const PopoverDemo009 = (() => {
  const Demo9 = function () {
      const [count, setCount] = createSignal(0);
      return (<Space>
        <Popover content={() => `Count: ${count()}`} fresh destroyOnHidden>
          <Button>Fresh content</Button>
        </Popover>
        <Button onClick={() => setCount((value) => value + 1)}>Update</Button>
      </Space>);
  };

  return Demo9
})()

## API

| Property               | Description                                                                           | Type                                                                                                                         | Default         |
| ---------------------- | ------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | --------------- |
| `title`                | Popover title content.                                                                | `JSX.Element \| () => JSX.Element`                                                                                           | `-`             |
| `content`              | Popover body content.                                                                 | `JSX.Element \| () => JSX.Element`                                                                                           | `-`             |
| `color`                | Background color of the overlay.                                                      | `string`                                                                                                                     | `-`             |
| `placement`            | Overlay placement relative to the trigger.                                            | `PopoverPlacement`                                                                                                           | `'top'`         |
| `builtinPlacements`    | Custom placement align config map.                                                    | `Partial<Record<PopoverPlacement, TooltipAlignConfig>>`                                                                      | `-`             |
| `trigger`              | Interaction that opens the overlay. Arrays enable multiple triggers.                  | `'hover' \| 'click' \| 'focus' \| 'contextMenu' \| Array<string>`                                                            | `'hover'`       |
| `open`                 | Controlled overlay open state.                                                        | `boolean`                                                                                                                    | `-`             |
| `defaultOpen`          | Initial overlay open state for uncontrolled usage.                                    | `boolean`                                                                                                                    | `false`         |
| `onOpenChange`         | Called when overlay open state changes.                                               | `(open: boolean) => void`                                                                                                    | `-`             |
| `afterOpenChange`      | Called after the open state has changed.                                              | `(open: boolean) => void`                                                                                                    | `-`             |
| `mouseEnterDelay`      | Delay in seconds before opening on hover.                                             | `number`                                                                                                                     | `0.1`           |
| `mouseLeaveDelay`      | Delay in seconds before closing after hover leaves.                                   | `number`                                                                                                                     | `0.1`           |
| `align`                | Fine-grained popup alignment config.                                                  | `TooltipAlignConfig`                                                                                                         | `-`             |
| `arrow`                | Shows the arrow or points it at the trigger center.                                   | `boolean \| { pointAtCenter?: boolean }`                                                                                     | `true`          |
| `autoAdjustOverflow`   | Whether to adjust placement automatically when off screen.                            | `boolean \| TooltipOverflowConfig`                                                                                           | `true`          |
| `fresh`                | Keep closed overlay content updating instead of using the cached content.             | `boolean`                                                                                                                    | `false`         |
| `destroyOnHidden`      | Destroy overlay DOM when hidden.                                                      | `boolean`                                                                                                                    | `false`         |
| `destroyTooltipOnHide` | Deprecated alias for `destroyOnHidden`.                                               | `boolean \| { keepParent?: boolean }`                                                                                        | `false`         |
| `rootClass`            | Additional class for the overlay root.                                                | `string`                                                                                                                     | `-`             |
| `openClass`            | Additional class applied to the trigger while open.                                   | `string`                                                                                                                     | `-`             |
| `overlayClass`         | Additional class for the overlay root. Prefer `classNames.root` for new code.         | `string`                                                                                                                     | `-`             |
| `overlayStyle`         | Inline style for the overlay root. Prefer `styles.root` for new code.                 | `JSX.CSSProperties`                                                                                                          | `-`             |
| `overlayInnerStyle`    | Inline style for the overlay inner container. Prefer `styles.container` for new code. | `JSX.CSSProperties`                                                                                                          | `-`             |
| `classNames`           | Customize semantic classes.                                                           | `Partial<Record<'root' \| 'container' \| 'arrow' \| 'title' \| 'content', string>> \| ((info: { props }) => ...)`            | `-`             |
| `styles`               | Customize semantic styles.                                                            | `Partial<Record<'root' \| 'container' \| 'arrow' \| 'title' \| 'content', JSX.CSSProperties>> \| ((info: { props }) => ...)` | `-`             |
| `zIndex`               | Overrides overlay z-index.                                                            | `number`                                                                                                                     | `-`             |
| `getPopupContainer`    | Returns the element used to mount the overlay portal.                                 | `(triggerNode?: HTMLElement) => HTMLElement`                                                                                 | `document.body` |
| `ref`                  | Receives methods and elements from the underlying tooltip layer.                      | `PopoverRef \| ((ref: PopoverRef) => void)`                                                                                  | `-`             |

## Semantic DOM

| Slot        | Description              |
| ----------- | ------------------------ |
| `root`      | Overlay root element.    |
| `container` | Overlay inner container. |
| `arrow`     | Arrow element.           |
| `title`     | Title element.           |
| `content`   | Content element.         |

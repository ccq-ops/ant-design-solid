# Typography

Typography provides article, title, text, link, and paragraph primitives for readable content.

## API

### Typography

| Property     | Description                                                                  | Type                                                                                                                  | Default     |
| ------------ | ---------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- | ----------- |
| `component`  | Root element to render.                                                      | `'article' \| 'section' \| 'div' \| 'span' \| 'p' \| 'a' \| 'h1'...'h5'`                                              | `'article'` |
| `class`      | Root class.                                                                  | `string`                                                                                                              | `-`         |
| `rootClass`  | Additional root class, matching antd root class semantics with Solid naming. | `string`                                                                                                              | `-`         |
| `classNames` | Semantic class slots.                                                        | `{ root?: string; actions?: string; action?: string; textarea?: string }`                                             | `-`         |
| `styles`     | Semantic style slots.                                                        | `{ root?: JSX.CSSProperties; actions?: JSX.CSSProperties; action?: JSX.CSSProperties; textarea?: JSX.CSSProperties }` | `-`         |
| `children`   | Typography content.                                                          | `JSX.Element`                                                                                                         | `-`         |

### Shared block props

Applied to `Typography.Text`, `Typography.Link`, `Typography.Title`, and `Typography.Paragraph` unless noted.

| Property     | Description                                              | Type                                                | Default                |
| ------------ | -------------------------------------------------------- | --------------------------------------------------- | ---------------------- |
| `actions`    | Placement for editable/copyable actions.                 | `{ placement?: 'start' \| 'end' }`                  | `{ placement: 'end' }` |
| `type`       | Semantic text color style.                               | `'secondary' \| 'success' \| 'warning' \| 'danger'` | `-`                    |
| `disabled`   | Disabled visual and action state.                        | `boolean`                                           | `false`                |
| `copyable`   | Show copy action.                                        | `boolean \| CopyConfig`                             | `false`                |
| `editable`   | Show edit action or controlled editing surface.          | `boolean \| EditConfig`                             | `false`                |
| `ellipsis`   | Enable truncation.                                       | `boolean \| EllipsisConfig`                         | `false`                |
| `code`       | Render content as code.                                  | `boolean`                                           | `false`                |
| `mark`       | Render content as marked text.                           | `boolean`                                           | `false`                |
| `underline`  | Render content as underlined text.                       | `boolean`                                           | `false`                |
| `delete`     | Render content as deleted text.                          | `boolean`                                           | `false`                |
| `strong`     | Render content as strong text. Not available on `Title`. | `boolean`                                           | `false`                |
| `keyboard`   | Render content as keyboard input.                        | `boolean`                                           | `false`                |
| `italic`     | Render content as italic text.                           | `boolean`                                           | `false`                |
| `title`      | Native title attribute.                                  | `string`                                            | `-`                    |
| `classNames` | Semantic class slots.                                    | `TypographySemanticClassNames`                      | `-`                    |
| `styles`     | Semantic style slots.                                    | `TypographySemanticStyles`                          | `-`                    |

### CopyConfig

| Property   | Description                                              | Type                                          | Default             |
| ---------- | -------------------------------------------------------- | --------------------------------------------- | ------------------- |
| `text`     | Text to copy. If omitted, string children are copied.    | `string \| (() => string \| Promise<string>)` | `-`                 |
| `onCopy`   | Called after copy resolves.                              | `(event?: MouseEvent) => void`                | `-`                 |
| `icon`     | Custom copy icon.                                        | `JSX.Element`                                 | `-`                 |
| `tooltips` | Tooltip content, or `false` to disable tooltip wrapping. | `JSX.Element \| false`                        | `'Copy' / 'Copied'` |
| `format`   | Clipboard format.                                        | `'text/plain' \| 'text/html'`                 | `'text/plain'`      |
| `tabIndex` | Copy action tab index.                                   | `number`                                      | `-`                 |

### EditConfig

| Property      | Description                                              | Type                                                | Default         |
| ------------- | -------------------------------------------------------- | --------------------------------------------------- | --------------- |
| `text`        | Text loaded into the textarea.                           | `string`                                            | string children |
| `editing`     | Controlled editing state.                                | `boolean`                                           | `-`             |
| `icon`        | Custom edit icon.                                        | `JSX.Element`                                       | `-`             |
| `tooltip`     | Tooltip content, or `false` to disable tooltip wrapping. | `JSX.Element \| false`                              | `'Edit'`        |
| `onStart`     | Called when editing starts.                              | `() => void`                                        | `-`             |
| `onChange`    | Called when textarea input changes.                      | `(value: string) => void`                           | `-`             |
| `onCancel`    | Called when Escape cancels editing.                      | `() => void`                                        | `-`             |
| `onEnd`       | Called when Enter or the finish action ends editing.     | `() => void`                                        | `-`             |
| `maxLength`   | Textarea max length.                                     | `number`                                            | `-`             |
| `autoSize`    | Textarea auto-size constraints.                          | `boolean \| { minRows?: number; maxRows?: number }` | `false`         |
| `triggerType` | Edit triggers.                                           | `('icon' \| 'text')[]`                              | `['icon']`      |
| `enterIcon`   | Finish action icon.                                      | `JSX.Element`                                       | `EnterOutlined` |
| `tabIndex`    | Edit action tab index.                                   | `number`                                            | `-`             |

### EllipsisConfig

| Property          | Description                                                 | Type                                                       | Default               |
| ----------------- | ----------------------------------------------------------- | ---------------------------------------------------------- | --------------------- |
| `rows`            | Maximum visible rows. Supported by `Paragraph` and `Title`. | `number`                                                   | `1`                   |
| `expandable`      | Show expand control. `collapsible` also shows collapse.     | `boolean \| 'collapsible'`                                 | `false`               |
| `suffix`          | Suffix rendered after content.                              | `string`                                                   | `-`                   |
| `symbol`          | Expand/collapse symbol.                                     | `JSX.Element \| ((expanded: boolean) => JSX.Element)`      | `'more' / 'collapse'` |
| `defaultExpanded` | Initial expanded state.                                     | `boolean`                                                  | `false`               |
| `expanded`        | Controlled expanded state.                                  | `boolean`                                                  | `-`                   |
| `onExpand`        | Called when expand/collapse is clicked.                     | `(event: MouseEvent, info: { expanded: boolean }) => void` | `-`                   |
| `onEllipsis`      | Called when ellipsis mode is active.                        | `(ellipsis: boolean) => void`                              | `-`                   |
| `tooltip`         | Reserved for tooltip content parity.                        | `JSX.Element \| false`                                     | `-`                   |

### Typography.Title

| Property | Description                              | Type                    | Default |
| -------- | ---------------------------------------- | ----------------------- | ------- |
| `level`  | Heading level rendered as h1 through h5. | `1 \| 2 \| 3 \| 4 \| 5` | `1`     |

### Typography.Link

`Typography.Link` supports shared block props, anchor attributes, and boolean-only `ellipsis`. When `target="_blank"` and `rel` is omitted, `rel="noopener noreferrer"` is added.

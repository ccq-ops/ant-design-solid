# Space

Space adds consistent gaps between inline or stacked elements.

### Basic

:::preview

        <Button>Two</Button>

      </Space>);
  };

  return Demo1
})()

## API

| Property      | Description                                                            | Type                                                                                         | Default        |
| ------------- | ---------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- | -------------- |
| `size`        | Gap size between items. Tuple values set horizontal and vertical gaps. | `Size \| [Size, Size]`                                                                       | `'small'`      |
| `orientation` | Stack items horizontally or vertically.                                | `'horizontal' \| 'vertical'`                                                                 | `'horizontal'` |
| `vertical`    | Shorthand for vertical orientation. `orientation` takes priority.      | `boolean`                                                                                    | `false`        |
| `direction`   | Deprecated alias of `orientation`.                                     | `'horizontal' \| 'vertical'`                                                                 | `'horizontal'` |
| `align`       | Cross-axis alignment of items.                                         | `'start' \| 'end' \| 'center' \| 'baseline'`                                                 | `-`            |
| `wrap`        | Allows horizontal items to wrap onto multiple lines.                   | `boolean`                                                                                    | `false`        |
| `separator`   | Separator rendered between adjacent items.                             | `JSX.Element`                                                                                | `-`            |
| `split`       | Deprecated alias of `separator`.                                       | `JSX.Element`                                                                                | `-`            |
| `classNames`  | Customize class for semantic slots. Use Solid's `class` naming.        | `Record<SemanticDOM, string> \| (info) => Record<SemanticDOM, string>`                       | `-`            |
| `styles`      | Customize inline style for semantic slots.                             | `Record<SemanticDOM, JSX.CSSProperties> \| (info) => Record<SemanticDOM, JSX.CSSProperties>` | `-`            |

### Size

```ts
type Size = 'small' | 'middle' | 'large' | number
```

### Space.Compact

Use `Space.Compact` when adjacent controls should connect visually.

| Property      | Description                                                       | Type                                         | Default        |
| ------------- | ----------------------------------------------------------------- | -------------------------------------------- | -------------- |
| `block`       | Fit width to the parent.                                          | `boolean`                                    | `false`        |
| `orientation` | Set compact layout direction.                                     | `'horizontal' \| 'vertical'`                 | `'horizontal'` |
| `vertical`    | Shorthand for vertical orientation. `orientation` takes priority. | `boolean`                                    | `false`        |
| `direction`   | Deprecated alias of `orientation`.                                | `'horizontal' \| 'vertical'`                 | `'horizontal'` |
| `size`        | Set compact size class.                                           | `'large' \| 'medium' \| 'middle' \| 'small'` | `'medium'`     |

### Space.Addon

Used to create custom cells in compact layouts.

| Property   | Description    | Type          | Default |
| ---------- | -------------- | ------------- | ------- |
| `children` | Custom content | `JSX.Element` | `-`     |

### Semantic DOM

| Slot    | Description                         |
| ------- | ----------------------------------- |
| `item`  | Wrapper around each rendered child. |
| `split` | Wrapper around each separator node. |

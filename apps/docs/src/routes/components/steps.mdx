# Steps

Steps display progress through an ordered task and can also act as clickable navigation between stages.

### Basic

```tsx
import { Steps } from '@ant-design-solid/core'

const Demo1 = function () {
  return (
    <Steps
      current={1}
      items={[
        { title: 'Finished', content: 'This step is complete.' },
        { title: 'In Progress', content: 'This step is running.' },
        { title: 'Waiting', content: 'This step has not started.' },
      ]}
    />
  )
}

export default Demo1
```

### Clickable

```tsx
import { createSignal } from 'solid-js'
import { Space, Steps } from '@ant-design-solid/core'

const Demo5 = function () {
  const [current, setCurrent] = createSignal(0)

  return (
    <Space direction="vertical" class="w-full">
      <Steps
        current={current()}
        onChange={setCurrent}
        items={[
          { title: 'Start', content: 'Create a draft.' },
          { title: 'Review', content: 'Check the submission.' },
          { title: 'Disabled', disabled: true, content: 'This step cannot be selected.' },
          { title: 'Done', content: 'Finish the task.' },
        ]}
      />
      <span>Current step: {current() + 1}</span>
    </Space>
  )
}

export default Demo5
```

## API

### Steps

| Property         | Description                                                                                                                        | Type                                                                      | Default             |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- | ------------------- |
| `items`          | Step items rendered in order.                                                                                                      | `StepItem[]`                                                              | `[]`                |
| `current`        | Current step index, counting from 0.                                                                                               | `number`                                                                  | `0`                 |
| `initial`        | Initial number offset for generated step numbers.                                                                                  | `number`                                                                  | `0`                 |
| `status`         | Status for the current step.                                                                                                       | `'wait' \| 'process' \| 'finish' \| 'error'`                              | `'process'`         |
| `orientation`    | Step bar orientation.                                                                                                              | `'horizontal' \| 'vertical'`                                              | `'horizontal'`      |
| `direction`      | Deprecated alias of `orientation`.                                                                                                 | `'horizontal' \| 'vertical'`                                              | `'horizontal'`      |
| `titlePlacement` | Place title and content horizontally or vertically. Dot and inline steps force vertical title placement in horizontal orientation. | `'horizontal' \| 'vertical'`                                              | `'horizontal'`      |
| `labelPlacement` | Deprecated alias of `titlePlacement`.                                                                                              | `'horizontal' \| 'vertical'`                                              | `'horizontal'`      |
| `size`           | Step size. `default` is kept as a deprecated alias for `medium`.                                                                   | `'medium' \| 'small' \| 'default'`                                        | `'medium'`          |
| `type`           | Step style type.                                                                                                                   | `'default' \| 'dot' \| 'inline' \| 'navigation' \| 'panel'`               | `'default'`         |
| `variant`        | Style variant.                                                                                                                     | `'filled' \| 'outlined'`                                                  | `'filled'`          |
| `percent`        | Progress percentage for the current `process` step in basic Steps. Values are clamped to the 0-100 range.                          | `number`                                                                  | `-`                 |
| `progressDot`    | Deprecated dot style switch or custom dot renderer. Prefer `type="dot"`.                                                           | `boolean \| StepsProgressDotRender`                                       | `false`             |
| `responsive`     | Switch to vertical orientation below 532px.                                                                                        | `boolean`                                                                 | `true`              |
| `ellipsis`       | Apply ellipsis styling for long step text.                                                                                         | `boolean`                                                                 | `false`             |
| `offset`         | Inline type item offset value exposed as `--<prefixCls>-items-offset`.                                                             | `number`                                                                  | `0`                 |
| `iconRender`     | Custom renderer for generated or item icons.                                                                                       | `StepsIconRender`                                                         | `-`                 |
| `onChange`       | Called when a non-disabled step is clicked. When present, default Steps become clickable.                                          | `(current: number) => void`                                               | `-`                 |
| `className`      | Root class name.                                                                                                                   | `string`                                                                  | `-`                 |
| `rootClassName`  | Additional root class name.                                                                                                        | `string`                                                                  | `-`                 |
| `classNames`     | Semantic DOM class names, or a function receiving merged component props.                                                          | `StepsSemanticClassNames \| (info: { props }) => StepsSemanticClassNames` | `-`                 |
| `styles`         | Semantic DOM inline styles, or a function receiving merged component props.                                                        | `StepsSemanticStyles \| (info: { props }) => StepsSemanticStyles`         | `-`                 |
| `prefixCls`      | Custom CSS class prefix.                                                                                                           | `string`                                                                  | From ConfigProvider |
| `rootComponent`  | Root element override for internal consumers.                                                                                      | `'div' \| 'ol'`                                                           | `'div'`             |
| `itemComponent`  | Item element override for internal consumers.                                                                                      | `'div' \| 'li'`                                                           | `'li'`              |

### StepItem

| Property      | Description                                                                   | Type                                              | Default                         |
| ------------- | ----------------------------------------------------------------------------- | ------------------------------------------------- | ------------------------------- |
| `title`       | Step title.                                                                   | `JSX.Element`                                     | `-`                             |
| `subTitle`    | Step subtitle.                                                                | `JSX.Element`                                     | `-`                             |
| `content`     | Step content.                                                                 | `JSX.Element`                                     | `-`                             |
| `description` | Deprecated alias of `content`. `content` wins when both are provided.         | `JSX.Element`                                     | `-`                             |
| `status`      | Per-step status override.                                                     | `'wait' \| 'process' \| 'finish' \| 'error'`      | Derived from `current`          |
| `icon`        | Custom step icon.                                                             | `JSX.Element`                                     | Generated number or status icon |
| `disabled`    | Disables click.                                                               | `boolean`                                         | `false`                         |
| `onClick`     | Called when the item container is clicked.                                    | `JSX.EventHandler<HTMLElement, MouseEvent>`       | `-`                             |
| `class`       | Item class name.                                                              | `string`                                          | `-`                             |
| `className`   | Item class name alias.                                                        | `string`                                          | `-`                             |
| `style`       | Item inline style.                                                            | `JSX.CSSProperties`                               | `-`                             |
| `classNames`  | Semantic DOM class names for this item. `root` and `list` are not accepted.   | `Omit<StepsSemanticClassNames, 'root' \| 'list'>` | `-`                             |
| `styles`      | Semantic DOM inline styles for this item. `root` and `list` are not accepted. | `Omit<StepsSemanticStyles, 'root' \| 'list'>`     | `-`                             |

### Semantic DOM

| Key            | Description                                                        |
| -------------- | ------------------------------------------------------------------ |
| `root`         | Root Steps element.                                                |
| `list`         | Ordered list wrapper when the root is not already an ordered list. |
| `item`         | Single step item.                                                  |
| `itemWrapper`  | Wrapper around each step's visual content.                         |
| `itemSection`  | Section containing header and content.                             |
| `itemHeader`   | Header containing icon, title, and subtitle.                       |
| `itemIcon`     | Icon container.                                                    |
| `itemTitle`    | Title node.                                                        |
| `itemSubtitle` | Subtitle node.                                                     |
| `itemContent`  | Content node.                                                      |
| `itemRail`     | Connector rail between steps.                                      |

### Related types

| Type                     | Definition                                                                                                                                                   |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `StepsStatus`            | `'wait' \| 'process' \| 'finish' \| 'error'`                                                                                                                 |
| `StepsDirection`         | `'horizontal' \| 'vertical'`                                                                                                                                 |
| `StepsSize`              | `'medium' \| 'small' \| 'default'`                                                                                                                           |
| `StepsType`              | `'default' \| 'navigation' \| 'dot' \| 'inline' \| 'panel'`                                                                                                  |
| `StepsVariant`           | `'filled' \| 'outlined'`                                                                                                                                     |
| `StepsIconRender`        | `(originNode: JSX.Element, info: { index: number; active: boolean; item: StepItem }) => JSX.Element`                                                         |
| `StepsProgressDotRender` | `(iconDot: JSX.Element, info: { index: number; status: StepsStatus; title?: JSX.Element; description?: JSX.Element; content?: JSX.Element }) => JSX.Element` |

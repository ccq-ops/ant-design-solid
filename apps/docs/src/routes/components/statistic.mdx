# Statistic

Statistic displays a formatted numerical value with optional title and adornments.

### Basic

:::preview

  const Demo1 = function () {
      return <Statistic title="Active Users" value={112893}/>;
  };

return Demo1
})()

import { Card, Space, Statistic } from '@ant-design-solid/core'

### Timer

:::preview

<StatisticDemo007 />

export const StatisticDemo007 = (() => {
  const Demo7 = function () {
      const [deadline, setDeadline] = createSignal(Date.now() + 1000 * 60 * 60 * 24);
      return (<Space wrap>
        <Statistic.Timer title="Countdown" type="countdown" value={deadline()} format="D [days] HH:mm:ss" onFinish={() => setDeadline(Date.now() + 1000 * 60 * 60 * 24)}/>
        <Statistic.Timer title="Running" type="countup" value={Date.now() - 1000 * 95}/>
      </Space>);
  };

return Demo7
})()

---

```tsx
import { createSignal } from 'solid-js'
import { Space, Statistic } from '@ant-design-solid/core'

const Demo7 = function () {
  const [deadline, setDeadline] = createSignal(Date.now() + 1000 * 60 * 60 * 24)

  return (
    <Space wrap>
      <Statistic.Timer
        title="Countdown"
        type="countdown"
        value={deadline()}
        format="D [days] HH:mm:ss"
        onFinish={() => setDeadline(Date.now() + 1000 * 60 * 60 * 24)}
      />
      <Statistic.Timer title="Running" type="countup" value={Date.now() - 1000 * 95} />
    </Space>
  )
}

export default Demo7
```

## API

### Statistic

| Property           | Description                                                                            | Type                                                                                                | Default    |
| ------------------ | -------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- | ---------- |
| `title`            | Statistic title.                                                                       | `JSX.Element`                                                                                       | `-`        |
| `value`            | Value content. Numeric values are formatted with separators by default.                | `string \| number \| null`                                                                          | `-`        |
| `precision`        | Decimal precision for numeric values.                                                  | `number`                                                                                            | `-`        |
| `prefix`           | Content rendered before the value.                                                     | `JSX.Element`                                                                                       | `-`        |
| `suffix`           | Content rendered after the value.                                                      | `JSX.Element`                                                                                       | `-`        |
| `formatter`        | Custom value display logic. Pass `false` to render the raw value string.               | `false \| 'number' \| 'countdown' \| ((value, config) => JSX.Element)`                              | `'number'` |
| `decimalSeparator` | Decimal separator for formatted numeric values.                                        | `string`                                                                                            | `'.'`      |
| `groupSeparator`   | Group separator for formatted numeric values.                                          | `string`                                                                                            | `','`      |
| `loading`          | Shows a loading placeholder instead of the value.                                      | `boolean`                                                                                           | `false`    |
| `classNames`       | Semantic DOM class names, or a function receiving merged props.                        | `StatisticSemanticClassNames \| ((info: { props: StatisticProps }) => StatisticSemanticClassNames)` | `-`        |
| `styles`           | Semantic DOM inline styles, or a function receiving merged props.                      | `StatisticSemanticStyles \| ((info: { props: StatisticProps }) => StatisticSemanticStyles)`         | `-`        |
| `valueStyle`       | Deprecated. Inline style applied to the content section. Use `styles.content` instead. | `JSX.CSSProperties \| string`                                                                       | `-`        |

### Statistic.Timer

| Property   | Description                                                                    | Type                       | Default    |
| ---------- | ------------------------------------------------------------------------------ | -------------------------- | ---------- |
| `type`     | Timer direction.                                                               | `'countdown' \| 'countup'` | `-`        |
| `format`   | Timer format. Supports `Y`, `M`, `D`, `H`, `m`, `s`, `S` and escaped `[text]`. | `string`                   | `HH:mm:ss` |
| `value`    | Target time for countdown, or start time for countup. Timestamp in ms.         | `string \| number \| null` | `-`        |
| `prefix`   | Content rendered before the timer value.                                       | `JSX.Element`              | `-`        |
| `suffix`   | Content rendered after the timer value.                                        | `JSX.Element`              | `-`        |
| `title`    | Timer title.                                                                   | `JSX.Element`              | `-`        |
| `onFinish` | Triggered when countdown reaches zero. Only called for `type="countdown"`.     | `() => void`               | `-`        |
| `onChange` | Triggered when the timer value changes. Receives remaining or elapsed ms.      | `(value: number) => void`  | `-`        |

### Statistic.Countdown

`Statistic.Countdown` is deprecated. Use `Statistic.Timer type="countdown"` for new code.

| Property   | Description                                                                        | Type                       | Default    |
| ---------- | ---------------------------------------------------------------------------------- | -------------------------- | ---------- |
| `format`   | Countdown format. Supports `Y`, `M`, `D`, `H`, `m`, `s`, `S` and escaped `[text]`. | `string`                   | `HH:mm:ss` |
| `value`    | Target countdown time. Timestamp in ms.                                            | `string \| number \| null` | `-`        |
| `prefix`   | Content rendered before the countdown value.                                       | `JSX.Element`              | `-`        |
| `suffix`   | Content rendered after the countdown value.                                        | `JSX.Element`              | `-`        |
| `title`    | Countdown title.                                                                   | `JSX.Element`              | `-`        |
| `onFinish` | Triggered when countdown reaches zero.                                             | `() => void`               | `-`        |
| `onChange` | Triggered when the countdown value changes. Receives remaining ms.                 | `(value: number) => void`  | `-`        |

### Semantic DOM

`Statistic` supports `root`, `header`, `title`, `content`, `value`, `prefix`, and `suffix` semantic slots.

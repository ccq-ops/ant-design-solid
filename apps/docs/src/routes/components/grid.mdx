# Grid

Grid uses rows and columns to build layouts on a 24-column system.

## API

### Row

| Property    | Description                                                                                                       | Type                                                                                                                    | Default |
| ----------- | ----------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- | ------- |
| `align`     | Vertical alignment of columns inside the row. Supports responsive values.                                         | `'top' \| 'middle' \| 'bottom' \| 'stretch' \| Record<Breakpoint, RowAlign>`                                            | `-`     |
| `gutter`    | Horizontal gutter, or horizontal and vertical gutter pair. Supports numbers, CSS strings, and responsive objects. | `number \| string \| Record<Breakpoint, number \| string> \| [Gutter, Gutter]`                                          | `0`     |
| `justify`   | Horizontal distribution of columns inside the row. Supports responsive values.                                    | `'start' \| 'end' \| 'center' \| 'space-around' \| 'space-between' \| 'space-evenly' \| Record<Breakpoint, RowJustify>` | `-`     |
| `prefixCls` | Custom class prefix for the row.                                                                                  | `string`                                                                                                                | `-`     |
| `wrap`      | Allows columns to wrap when there is not enough horizontal space.                                                 | `boolean`                                                                                                               | `true`  |

### Col

| Property                                          | Description                                                                               | Type                                   | Default |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------- | -------------------------------------- | ------- |
| `flex`                                            | Flex layout value. Numbers map to proportional flex, CSS lengths map to fixed flex basis. | `number \| 'none' \| 'auto' \| string` | `-`     |
| `offset`                                          | Number of grid units to move the column to the right with margin.                         | `number \| string`                     | `-`     |
| `order`                                           | CSS flex order for the column.                                                            | `number \| string`                     | `-`     |
| `prefixCls`                                       | Custom class prefix for the column.                                                       | `string`                               | `-`     |
| `pull`                                            | Moves the column left by the specified grid units.                                        | `number \| string`                     | `-`     |
| `push`                                            | Moves the column right by the specified grid units.                                       | `number \| string`                     | `-`     |
| `span`                                            | Column width as a number of units in the 24-column grid. `0` hides the column.            | `number \| string`                     | `-`     |
| `xs` / `sm` / `md` / `lg` / `xl` / `xxl` / `xxxl` | Responsive column configuration.                                                          | `number \| string \| ColSize`          | `-`     |

### ColSize

| Property | Description            | Type                                   |
| -------- | ---------------------- | -------------------------------------- |
| `flex`   | Responsive flex value. | `number \| 'none' \| 'auto' \| string` |
| `offset` | Responsive offset.     | `number \| string`                     |
| `order`  | Responsive order.      | `number \| string`                     |
| `pull`   | Responsive pull.       | `number \| string`                     |
| `push`   | Responsive push.       | `number \| string`                     |
| `span`   | Responsive span.       | `number \| string`                     |

### Grid

| Property          | Description                                             | Type                                                   |
| ----------------- | ------------------------------------------------------- | ------------------------------------------------------ |
| `useBreakpoint()` | Returns a Solid accessor for the active breakpoint map. | `() => Accessor<Partial<Record<Breakpoint, boolean>>>` |

### Breakpoint

`Breakpoint` is one of `xs`, `sm`, `md`, `lg`, `xl`, `xxl`, or `xxxl`.

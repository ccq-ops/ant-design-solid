# Descriptions

Descriptions displays multiple read-only fields in groups.

## API

### Descriptions

| Property     | Description                                                  | Type                                                                                                                                             | Default        |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------ | -------------- |
| `title`      | Header title content.                                        | `JSX.Element`                                                                                                                                    | `-`            |
| `extra`      | Header extra content.                                        | `JSX.Element`                                                                                                                                    | `-`            |
| `bordered`   | Whether to render bordered table cells.                      | `boolean`                                                                                                                                        | `false`        |
| `colon`      | Whether to show the colon after labels in horizontal layout. | `boolean`                                                                                                                                        | `true`         |
| `column`     | Number of description columns.                               | `number \| Partial<Record<Breakpoint, number>>`                                                                                                  | `3`            |
| `size`       | Descriptions size.                                           | `'large' \| 'medium' \| 'small'`                                                                                                                 | `'large'`      |
| `layout`     | Label/content layout.                                        | `'horizontal' \| 'vertical'`                                                                                                                     | `'horizontal'` |
| `items`      | Description item data.                                       | `DescriptionsItemType[]`                                                                                                                         | `-`            |
| `children`   | Descriptions.Item children used when items are not provided. | `JSX.Element`                                                                                                                                    | `-`            |
| `classNames` | Semantic class names.                                        | `{ root?: string; header?: string; title?: string; extra?: string; label?: string; content?: string }`                                           | `-`            |
| `styles`     | Semantic inline styles.                                      | `{ root?: CSSProperties; header?: CSSProperties; title?: CSSProperties; extra?: CSSProperties; label?: CSSProperties; content?: CSSProperties }` | `-`            |

### Descriptions.Item

| Property     | Description                                                            | Type                                                        | Default |
| ------------ | ---------------------------------------------------------------------- | ----------------------------------------------------------- | ------- |
| `key`        | Stable item key used by item data.                                     | `string \| number`                                          | `-`     |
| `label`      | Item label content.                                                    | `JSX.Element`                                               | `-`     |
| `children`   | Item value content.                                                    | `JSX.Element`                                               | `-`     |
| `span`       | Number of columns occupied by the item, or `filled` for remaining row. | `number \| 'filled' \| Partial<Record<Breakpoint, number>>` | `1`     |
| `class`      | Additional item class.                                                 | `string`                                                    | `-`     |
| `style`      | Item cell inline style.                                                | `JSX.CSSProperties \| string`                               | `-`     |
| `classNames` | Semantic label/content class names.                                    | `{ label?: string; content?: string }`                      | `-`     |
| `styles`     | Semantic label/content inline styles.                                  | `{ label?: CSSProperties; content?: CSSProperties }`        | `-`     |

## API

Icon components accept standard Solid SVG props and the common icon props below.

### Common Icon

| Property       | Description                                                           | Type                                             | Default |
| -------------- | --------------------------------------------------------------------- | ------------------------------------------------ | ------- |
| `class`        | The class name of Icon.                                               | `string`                                         | `-`     |
| `rotate`       | Rotates the icon by degrees.                                          | `number`                                         | `-`     |
| `spin`         | Rotates the icon continuously.                                        | `boolean`                                        | `false` |
| `style`        | The style properties of icon, like `font-size` and `color`.           | `JSX.CSSProperties \| string`                    | `-`     |
| `twoToneColor` | Only supports two-tone icons. Sets primary or primary/secondary fill. | `string \| [primary: string, secondary: string]` | `-`     |

### Custom Icon

| Property    | Description                               | Type                                             | Default |
| ----------- | ----------------------------------------- | ------------------------------------------------ | ------- |
| `component` | The component used for the root SVG node. | `Component<JSX.SvgSVGAttributes<SVGSVGElement>>` | `-`     |
| `rotate`    | Rotates the icon by degrees.              | `number`                                         | `-`     |
| `spin`      | Rotates the icon continuously.            | `boolean`                                        | `false` |
| `style`     | The style properties of icon.             | `JSX.CSSProperties \| string`                    | `-`     |

### Custom Font Icon

`createFromIconfontCN` creates a component that uses SVG sprites from iconfont.cn.

| Property    | Description                                                          | Type                 | Default |
| ----------- | -------------------------------------------------------------------- | -------------------- | ------- |
| `scriptUrl` | The URL generated by an iconfont.cn project. Supports multiple URLs. | `string \| string[]` | `-`     |

Decorative icons are rendered with `aria-hidden="true"` by default. Provide `aria-label` or
`aria-labelledby` when an icon carries semantic meaning.

# Mentions

Mentions lets users select people or topics while typing in a textarea.

### Basic

:::preview

<MentionsDemo001 />

  const peopleOptions = [

  ];

      return <Mentions placeholder="Mention someone with @" options={peopleOptions}/>;
  };

return Demo1

## API

### Mentions

| Property            | Description                                                     | Type                                                                       | Default      |
| ------------------- | --------------------------------------------------------------- | -------------------------------------------------------------------------- | ------------ |
| `value`             | Controlled textarea value.                                      | `string`                                                                   | `-`          |
| `defaultValue`      | Initial textarea value for uncontrolled usage.                  | `string`                                                                   | `-`          |
| `open`              | Controlled suggestions popup open state.                        | `boolean`                                                                  | `-`          |
| `defaultOpen`       | Initial suggestions popup open state.                           | `boolean`                                                                  | `false`      |
| `options`           | Suggestion options.                                             | `MentionsOption[]`                                                         | `-`          |
| `children`          | Legacy option children. Prefer `options` for data-driven usage. | `JSX.Element`                                                              | `-`          |
| `prefix`            | Mention trigger prefix or prefixes.                             | `string \| string[]`                                                       | `'@'`        |
| `split`             | Text inserted after a selected mention.                         | `string`                                                                   | `' '`        |
| `placeholder`       | Textarea placeholder text.                                      | `string`                                                                   | `-`          |
| `disabled`          | Disables editing and suggestions.                               | `boolean`                                                                  | `false`      |
| `size`              | Textarea size.                                                  | `'small' \| 'middle' \| 'large'`                                           | `-`          |
| `status`            | Validation status.                                              | `'error' \| 'warning'`                                                     | `-`          |
| `variant`           | Visual variant.                                                 | `'outlined' \| 'borderless' \| 'filled' \| 'underlined'`                   | `'outlined'` |
| `allowClear`        | Shows a clear affordance when the value is not empty.           | `boolean \| { clearIcon?: JSX.Element; disabled?: boolean }`               | `false`      |
| `loading`           | Shows a loading option and suppresses normal options.           | `boolean`                                                                  | `false`      |
| `notFoundContent`   | Content shown when no option matches.                           | `JSX.Element`                                                              | `-`          |
| `placement`         | Popup placement.                                                | `'top' \| 'bottom'`                                                        | `'bottom'`   |
| `rootClass`         | Extra class on the root element.                                | `string`                                                                   | `-`          |
| `popupClass`        | Extra class on the popup element.                               | `string`                                                                   | `-`          |
| `showCount`         | Shows textarea character count.                                 | `boolean \| { formatter?: (info: CountFormatterInfo) => JSX.Element }`     | `false`      |
| `count`             | Character count configuration.                                  | `CountConfig`                                                              | `-`          |
| `autoSize`          | Auto size rows configuration.                                   | `boolean \| { minRows?: number; maxRows?: number }`                        | `false`      |
| `classNames`        | Semantic class names.                                           | `MentionsSemanticClassNames`                                               | `-`          |
| `styles`            | Semantic styles.                                                | `MentionsSemanticStyles`                                                   | `-`          |
| `filterOption`      | Controls filtering or supplies a custom option filter.          | `boolean \| ((inputValue: string, option: MentionsOption) => boolean)`     | `-`          |
| `validateSearch`    | Controls whether text after a prefix should open suggestions.   | `(text: string, props: MentionsProps) => boolean`                          | `-`          |
| `prefixCls`         | Custom CSS class prefix.                                        | `string`                                                                   | `-`          |
| `zIndex`            | Overrides popup z-index.                                        | `number`                                                                   | `-`          |
| `getPopupContainer` | Returns the element used to mount the popup portal.             | `(triggerNode?: HTMLElement) => HTMLElement`                               | `-`          |
| `ref`               | Exposes `focus`, `blur`, `textarea`, and `nativeElement`.       | `MentionsRef \| { current?: MentionsRef } \| ((ref: MentionsRef) => void)` | `-`          |
| `onChange`          | Called when textarea value changes.                             | `(value: string) => void`                                                  | `-`          |
| `onSelect`          | Called when an option is selected.                              | `(option: MentionsOption, prefix: string) => void`                         | `-`          |
| `onSearch`          | Called while searching after a prefix.                          | `(text: string, prefix: string) => void`                                   | `-`          |
| `onOpenChange`      | Called when popup open state changes.                           | `(open: boolean) => void`                                                  | `-`          |
| `onClear`           | Called when the clear button is clicked.                        | `() => void`                                                               | `-`          |
| `onPressEnter`      | Called on Enter key down.                                       | `(event: KeyboardEvent) => void`                                           | `-`          |
| `onResize`          | Called when textarea size is observed to change.                | `(size: { width: number; height: number }) => void`                        | `-`          |
| `onPopupScroll`     | Called when popup scrolls.                                      | `(event: UIEvent) => void`                                                 | `-`          |

### Static methods

| Method                 | Description                        | Type                                                                                                               |
| ---------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `Mentions.getMentions` | Parses mention entities from text. | `(value: string, config?: { prefix?: string \| string[]; split?: string }) => { prefix: string; value: string }[]` |

### MentionsOption

| Property   | Description                                             | Type                | Default |
| ---------- | ------------------------------------------------------- | ------------------- | ------- |
| `label`    | Option display label. Falls back to value when omitted. | `JSX.Element`       | `-`     |
| `value`    | Mention value inserted after the prefix.                | `string`            | `-`     |
| `disabled` | Disables selecting this option.                         | `boolean`           | `-`     |
| `class`    | Option class.                                           | `string`            | `-`     |
| `style`    | Option style.                                           | `JSX.CSSProperties` | `-`     |
| `key`      | Option key.                                             | `string`            | `-`     |
| `title`    | Option title attribute.                                 | `string`            | `-`     |

### Mentions.Option

| Property   | Description                     | Type          | Default |
| ---------- | ------------------------------- | ------------- | ------- |
| `children` | Option display content.         | `JSX.Element` | `-`     |
| `value`    | Mention value after the prefix. | `string`      | `-`     |
| `disabled` | Disables selecting this option. | `boolean`     | `false` |

# DatePicker

Select dates, date ranges, and date-time values. Like antd, DatePicker values are `dayjs` objects rather than strings.

## API

DatePicker and RangePicker values use `dayjs`. The `dateString` callback argument is only the formatted string representation of those dayjs values.

## ShowTimeOptions

Pass these fields through `showTime` to configure the time panel. The same disabled callbacks are also available from `disabledTime`.

| Property              | Description                                                                | Type                      | Default |
| --------------------- | -------------------------------------------------------------------------- | ------------------------- | ------- |
| `defaultOpenValue`    | Default time shown before a date is selected. RangePicker accepts a tuple. | `Dayjs \| [Dayjs, Dayjs]` | `-`     |
| `format`              | Time display format.                                                       | `string`                  | `-`     |
| `showHour`            | Shows the hour column.                                                     | `boolean`                 | `true`  |
| `showMinute`          | Shows the minute column.                                                   | `boolean`                 | `true`  |
| `showSecond`          | Shows the second column.                                                   | `boolean`                 | `true`  |
| `use12Hours`          | Uses 12-hour meridiem display.                                             | `boolean`                 | `-`     |
| `hideDisabledOptions` | Hides disabled time options instead of rendering them disabled.            | `boolean`                 | `-`     |
